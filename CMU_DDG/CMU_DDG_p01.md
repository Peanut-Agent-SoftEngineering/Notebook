# CMU_DDG_p01

> 主题：CMU 离散微分几何课程导论 —— 教学定位与运行机制

## 一、课程定位与核心理念

### 1. 这是什么课？

- **不是高级课程**，而是**第一堂真正的几何学课程**，专注于曲线和曲面。
- 目标不是教你证明定理或实现算法，而是 **重新训练你思考几何的方式**——如何**表示**、**谈论**、**处理**形状。

### 2. 教学路线的颠覆

- **传统微分几何**：拓扑空间 → 图册 → 流形，抽象且令人挣扎。
- **本课程**：直觉优先策略，始终把几何视为**曲线/曲面在 2D/3D 空间中的浸入（immersion）**。
  - 更贴近自然思考方式
  - 更贴近计算实际

### 3. 数学与计算的桥梁

- **核心隐喻**：**网格（mesh）** 扮演**抽象拓扑空间**的角色，**顶点位置**定义网格的**浸入**（即如何坐落在空间中）。
- **双向增益**：
  - 离散视角 → 帮你快速理清光滑几何中的困难概念
  - 光滑视角 → 彻底改变你思考算法和计算的方式

### 4. 教学风格

- **直观（Visual）**：所有内容尽量绘制成图，建立快速直觉。
- **动手（Hands-on）**：课堂讨论的所有概念最终都转化为**在你电脑上运行的真实代码**。

## 二、运行机制

### 1. 讲座与互动模式

- **讲座**：预先录制并发布在 YouTube 上，可暂停、倒带、倍速。
- **同步答疑**：定期通过 **Zoom** 提供实时问答；遇到问题暂停视频，去 Zoom 提问，得到答案后继续。
- **多元答疑通道**：
  - **Discord**：即时沟通
  - **课程网页评论**：针对帖子提问
  - **TA 远程办公**：助教 Zoom 答疑

### 2. 评分体系

**无考试，纯作业制**。分数由三部分构成：

| 类型 | 内容 |
| :--- | :--- |
| 书面练习 | 笔算/推导 |
| 编码作业 | 编程实现 |
| 阅读反馈 | 课程笔记或论文的总结 + 1-2 个不理解的问题或想深入的方向 |

- **官方文本**：课程网页链接的课程笔记。
- **进阶推荐**：提供优质教科书书单。

### 3. 迟到政策

- **5 天迟到额度**：整个学期可任意使用（生病、面试等），核心理念是**保持学习乐趣**。
- **趣味记录方式：与柏拉图立体对应**——提交时根据迟到天数绘制对应立体（提供数字图像可直接插入）。

| 迟到天数 | 柏拉图立体 |
| :---: | :--- |
| 1 | 四面体 |
| 2 | 立方体 |
| 3 | 八面体 |
| 4 | 十二面体 |
| 5 | 二十面体 |

### 4. 协作与学术诚信

- **鼓励跨领域协作**：背景涵盖数学、CS、工程、物理、艺术、设计、建筑——互相学习。
- **最终作品必须原创**：协作后**关闭窗口 → 删除聊天记录 → 完全用自己的话独立完成**。
- 完整作弊政策见课程网页。

### 5. 编码作业框架

| 特性 | JavaScript 版 | C++ 版 |
| :--- | :--- | :--- |
| 适用人群 | 编码新手 | 有经验的程序员 |
| 运行环境 | 浏览器直接运行 | 现代网格处理库 |
| 安装要求 | 无（打开网页即可） | 需配置环境 |
| 可视化 | 内置简易 UI | 复杂可视化工具 |
| 优势 | 零门槛 | 便于原型/算法研究 |

> 课程团队已完成大部分繁重工作——提供**代码骨架**和**完整可视化系统**，你只需填写**核心几何算法**。

### 6. 先修要求

- 线性代数
- 向量微积分（三维微积分）
- 编程背景（语言不重要）
