# CMU_DDG_p14

> 主题：离散曲线——从光滑参数化曲线到分段线性曲线的完整翻译；离散正则性、离散曲率/挠率、离散基本定理、曲率流（曲线缩短流、弹性流、桥本流）与排斥能量

## 一、从光滑到离散：基本对照

### 1. 思想转变

| 设定 | 曲线对象 |
| :--- | :--- |
| 光滑 | 连续映射 $\gamma:[0,L]\to\mathbb{R}^n$ |
| 离散 | 顶点序列 $\gamma_0,\gamma_1,\dots,\gamma_n$，相邻直线段连接 |

看似简单的"折线"，竟能完整捕捉微分几何的核心结构。

### 2. 光滑曲线的关键回顾

- **参数化**：曲线是从实数区间到空间的光滑映射；同一形状有无穷多种参数化。
- **Frenet 框架**：把局部几何量（切向量、法向量、副法向量、曲率、挠率）组织在一起的局部坐标系。
- **嵌入 vs 浸没**：嵌入保拓扑、不自交；浸没允许全局自交但**处处导数良好**。
- **平面曲线两个整数**：
  - **转数（Turning Number）**：沿曲线一周切线累积旋转的圈数。
  - **绕数（Winding Number）**：曲线绕给定点的圈数。
- **基本定理**：弧长参数化下，曲率（平面）或曲率 + 挠率（空间）唯一确定曲线（差一个刚体运动）。前进速度 + 转弯速度 = 几何形状。

## 二、离散曲线的正式定义

### 1. 基础描述

- 顶点序列：$\gamma_0,\gamma_1,\dots,\gamma_L$，相邻顶点连成线段。
- 解释为 $[0,L]$ 到平面的连续（分段线性）映射。
- 参数位置 $s_i$ 标记第 $i$ 个顶点。

**示例**（两段折线）：
$$\gamma(s)=\begin{cases}(s,0)&s\in[0,1]\\(1,s-1)&s\in[1,2]\end{cases}$$

### 2. 离散外微积分语言

- **抽象单纯 1-复形**：边和顶点的有序集合（包含定向）。
- **离散 $\mathbb{R}^n$ 值零形式 $\gamma$**：为每个顶点赋予一个空间位置。
- **从离散到连续——Whitney 基**（即帽子函数）：
  - 在曲线情形就是分段线性插值，把顶点位置插成连续线段。
  - 这是把"微分形式语言"和"离散计算"打通的关键工具。

## 三、离散微分、切线与法线

### 1. 离散微分（关键！）

定义在边 $e_i=(\gamma_i,\gamma_{i+1})$ 上的离散外导数：
$$d\gamma(e_i)=\gamma_{i+1}-\gamma_i$$

**直观**：离散微分就是"从当前顶点指向下一顶点的边向量"——未归一化。沿边对 $d\gamma$ 求和直接得到这条边本身（自然性）。

### 2. 单位切线

每条边上的归一化边向量：
$$T_i=\frac{\gamma_{i+1}-\gamma_i}{\|\gamma_{i+1}-\gamma_i\|}$$

### 3. 离散法线

$$N_i=R_{90}(T_i)$$

将每条边的单位切线逆时针旋转 90° 得到。

### 4. 微分 vs 切线

| 量 | 性质 |
| :--- | :--- |
| 微分 $d\gamma$ | **映射**——输入任意标量/向量，输出空间向量；可接受任意缩放 |
| 单位切线 $T_i$ | **特定向量**——曲线自身的几何属性，长度恒为 1 |

不能简单说"微分=未归一化的切线"，前者是函数，后者是值。

### 5. 顶点处的难题

折线在顶点处不可微，光滑情形下"顶点处的切线/法线/曲率"无定义。**为这些量在顶点处寻找合适的离散模拟，是整个离散微分几何的核心活动**——下文的多种离散曲率定义、Frenet 替代品都源于此。

## 四、离散正则性的两种定义

### 1. 朴素定义：所有边长 > 0

$$\|\gamma_{i+1}-\gamma_i\|>0\quad\forall i$$

### 2. 朴素定义的缺陷：零角退化

即使所有边长 > 0，仍可能出现**相邻边夹角为 0**（边沿同一方向折返、部分重叠）。这种退化导致：

- 局部路径"来回走过同一段"。
- 在保持边长 > 0 的连续形变中，曲线可以**穿过自身**而改变绕数/转数。
- 失去光滑情形下"正则曲线不能任意改变拓扑"的关键约束（Whitney–Graustein 一类性质）。

### 3. 更好的定义：离散浸没（局部单射）

> **离散正则曲线 = 离散浸没**：映射在定义域每个点附近都是单射（存在足够小的邻域使限制为单射）。

**优势**：

- 完全不依赖"导数"概念，只看映射本身。
- 自动同时排除零边长和零角——零角 = 路径折返 = 局部非单射。
- 允许全局自交（如 8 字交点），仍属于正则——因为交点附近曲线仍是简单线段。

| 曲线类型 | 全局单射 | 局部单射 |
| :--- | :---: | :---: |
| 8 字形（自交） | ❌ | ✅ |
| 零角折返 | ❌ | ❌ |

### 4. 与光滑情形的对比

| 设定 | 正则性条件 |
| :--- | :--- |
| 光滑 | $\gamma'(t)\neq 0$（局部微分同胚） |
| 离散（朴素） | 所有边长 > 0（**不充分**） |
| 离散（推荐） | **局部单射** |

> **方法论启示**：离散微分几何的核心范式——光滑里多个等价定义，到离散后会"分裂"成不等价的多个定义，必须选**保留最多结构**的那个。

## 五、离散曲率：三种推导路径

光滑曲率是切线方向变化率、长度变分、密切圆半径倒数等多种等价定义。离散后这些定义不再等价，但在网格细化下都收敛到同一光滑曲率值。

### 1. 路径 A：转角法（外角）

光滑：$\kappa=d\theta/ds$。离散：相邻边之间的**外角** $\theta_i$ 直接作为顶点曲率。

直观：在折线顶点转了多少角度，就是该点的离散曲率度量。

### 2. 路径 B：长度变分（梯度法）

光滑：$\frac{dL}{dt}=\int\kappa\,ds$，长度梯度沿法向偏移即曲率法线。
离散：把所有边长求和的梯度作为顶点处的离散曲率：
$$\nabla L_{\text{total}}\;\text{（顶点位置）}\;\sim\;\kappa_i \mathbf{N}_i$$

直观：把顶点向外推一点，总长度变得多快，曲率就有多大。

### 3. 路径 C：密切圆（三点圆法）

取连续三个顶点 $\gamma_{i-1},\gamma_i,\gamma_{i+1}$ 定一个圆，半径倒数即曲率。

### 4. "无免费午餐"

各定义只满足理想性质（符号一致性、局部性、与长度的线性关系等）的子集；**不存在单一定义能同时满足所有性质**。需根据应用场景选择合适定义。所有合理定义在网格细化下收敛到同一光滑值，这是合法性保证。

## 六、离散平面曲线基本定理

> **正则离散平面曲线（差一个刚体运动）由其边长序列 $\{l_i\}$ 与外角序列 $\{\phi_i\}$ 唯一确定。**

### 1. 与光滑版本的对应

| 设定 | 内蕴量 |
| :--- | :--- |
| 光滑（弧长参数化） | $\kappa(s)$ |
| 离散 | $\{l_i\}$ + $\{\phi_i\}$ |

外角求和扮演了曲率积分的角色。

### 2. 重建过程（两步"积分"）

**步骤 1**：方向（外角累加）
$$\theta_{i+1}=\theta_0+\sum_{j=1}^{i}\phi_j$$

**步骤 2**：位置（边向量累加）
$$X_{i+1}=X_i+l_i\cdot(\cos\theta_i,\sin\theta_i)$$

两个自由度 $\theta_0,X_0$ 对应**旋转**与**平移**——刚体运动。

### 3. 离散 Whitney–Graustein 定理

- **平滑版本**：两条平面闭曲线正则同伦 ⇔ 转数相同（充要条件）。
- **离散版本**：
  - 若采用 **"局部单射"** 作为离散正则性 → 定理**精确成立**。
  - 若采用 **"边长非零"** 作为离散正则性 → 定理**不成立**（存在反例：边长保持非零但绕数可改变）。

> **方法论启示**：判断离散定义"好坏"，看它能让多少光滑结构和定理在离散世界中保留下来。

## 七、离散空间曲线

### 1. 定义

顶点序列 $\gamma_i\in\mathbb{R}^3$，按序号或累积弧长参数化；结构与平面情形相同，仅维度增加。

### 2. 离散曲率（顶点处）

任何**两条相邻边总位于同一平面**——把该平面内相邻边的外角作为离散曲率。

### 3. 离散挠率（边上）

挠率衡量曲线"扭出当前平面"的速度。

- 取**三条连续边** $e_1=\gamma_1\gamma_2$, $e_2=\gamma_2\gamma_3$, $e_3=\gamma_3\gamma_4$。
- 前两条定义平面 $P_1$，后两条定义平面 $P_2$。
- **离散挠率角** = 这两个平面**法线的夹角**：
$$\tau=\angle(\mathbf{n}_{P_1},\mathbf{n}_{P_2})$$

记忆点：**曲率在顶点（平面内转角）；挠率在边（连续平面之间的扭角）。**

### 4. 离散空间曲线基本定理

> 离散空间曲线（差刚体运动）由 **边长 + 离散曲率 + 离散挠率** 唯一确定。

### 5. 约化坐标（Reduced Coordinates）

不直接给出顶点坐标，而用三元链描述曲线：边长（≈$ds$）、曲率（顶点处）、挠率（边处）。

类比光滑情形的 $\kappa(s),\tau(s)$，便于分析与变形，对应离散版的 Frenet–Serret。

## 八、从内蕴量重建空间曲线

### 1. 算法

```
输入: 边长 {l_i}, 曲率 {κ_i}, 挠率 {τ_i}
      初始点 P_0, 初始切线 T_0, 初始法线 N_0（确定 B_0）

for each i:
    1. 沿 T_i 方向移动 l_i → 得到新顶点 P_{i+1}
    2. 在当前平面内将切线绕 B_i 旋转 κ_i → 得到 T_{i+1}
    3. 将平面绕 T_i 旋转 τ_i → 更新 B_{i+1}, N_{i+1}
```

本质：**走一步，转平面，扭空间，再走一步**。

### 2. 旋转的数学表示

绕单位轴 $\mathbf{u}$ 旋转角 $\theta$ 的旋转矩阵：

**李代数 / 矩阵指数**：
$$R=\exp(\theta[\mathbf{u}]_\times)$$
其中 $[\mathbf{u}]_\times$ 是 $\mathbf{u}$ 的叉积矩阵。该公式建立 $SO(3)$ 与其李代数的桥梁。

**Rodrigues 公式（实现用）**：
$$R(\mathbf{u},\theta)=I+\sin\theta\,[\mathbf{u}]_\times+(1-\cos\theta)\,[\mathbf{u}]_\times^2$$

其中
$$[\mathbf{u}]_\times=\begin{pmatrix}0&-u_z&u_y\\ u_z&0&-u_x\\ -u_y&u_x&0\end{pmatrix}$$

### 3. 离散 vs 光滑：实践巨大差异

- **光滑**：从 $\kappa(s),\tau(s)$ 解析积分恢复曲线极其困难（只有圆、螺旋等少数解析解）。
- **离散**：一个简单循环，逐段拼接，**异常稳定、简单、干净**——本质上就是耦合 ODE 的几何离散求解器。

**强烈建议**：写一段小代码，随便选 $\kappa_i,\tau_i$ 函数（如 $\kappa_i=1/n,\tau_{i+1}=(n-i)/n$）跑一下——观察从空间螺旋逐步退化为平面环的过程，是培养"曲率/挠率几何直觉"最快的方法。

### 4. 数据压缩与编辑友好

- 三种标量序列即可完整描述空间曲线。
- 改曲率影响弯曲，改挠率影响扭曲，互不干扰。
- 应用：相机路径、绳缆与树枝、毛发、脊柱/尾部动画的程序化建模。

## 九、曲率流：能量驱动的曲线演化

### 1. 通用范式

为曲线定义能量函数 $E(\gamma)$，让曲线沿能量梯度下降方向演化（**梯度流**）：
$$\frac{\partial\gamma}{\partial t}=-\nabla E$$

### 2. 两条离散化路径

| 路径 | 流程 | 备注 |
| :--- | :--- | :--- |
| 先连续微分，再离散 | 光滑能量 → 求梯度得 PDE → 把光滑算子（如 $\kappa$）离散化 | 自然但可能破坏守恒律 |
| 先离散能量，再求梯度 | 直接对离散能量（如 $\sum$边长）求顶点位置梯度 | 直接得离散更新规则 |

两条路径**通常给出不同的数值方案**——这一选择对能量/动量守恒至关重要。

### 3. 应用领域

- 物理模拟（弹性棒、电缆动力学）。
- 几何处理（最短路径/测地线、轮廓平滑）。
- 图像处理（黑白边界的特征检测、降噪）。

## 十、曲线缩短流（Curve Shortening Flow）

### 1. 定义

能量 = 曲线总长度：
$$E(\gamma)=\int_\gamma\|\gamma'(s)\|\,ds$$

长度的梯度是曲率法线 $\kappa\mathbf{N}$，因此演化方程：
$$\frac{\partial\gamma}{\partial t}=\kappa\,\mathbf{N}$$

直观：每点沿法线方向移动，速度等于当前曲率。

### 2. 关键性质

- **质心守恒**：沿曲线对位置积分，质心始终不变。
- **趋于圆形**：曲线随时间逐步"圆化"。
- **有限时间消失**：圆形成后继续收缩，**有限时间内坍缩为一点**。
  - 正反馈：法向移动 → 半径减小 → 曲率增大 → 收缩更快。
- **嵌入保持**：初始不自交则永远不自交——对处理物体轮廓/剪影特别有用。

### 3. 离散实现

**时间离散化**（前向欧拉）：
$$\frac{\gamma^{k+1}-\gamma^k}{\Delta t}\approx\kappa\cdot\mathbf{n}$$

**空间离散化**（两条路径）：

- 把 $\kappa,\mathbf{n}$ 替换为某种离散曲率定义。
- **或者**：直接对总边长能量
$$E_{\text{length}}=\sum_i\|\gamma_{i+1}-\gamma_i\|$$
求顶点位置梯度，得到离散流动规则。

最终得**时空均离散**的迭代规则。同时完成平滑与去噪，最终仍趋于圆并坍缩。

## 十一、弹性流（Elastic Flow）

### 1. 动机与能量

只想"降低曲率让曲线变光滑"，但**不希望长度也跟着缩短**。能量定义为曲率平方积分：
$$E_{\text{bend}}=\int_\gamma\kappa^2\,ds$$

为何用 $\kappa^2$（而非 $|\kappa|$ 或 $\kappa$）：曲率有符号，平方后消除符号，**无差别**地惩罚任意方向的弯曲。

### 2. 与曲线缩短流的对比

| 流 | 驱动能量 | 速度方程 | 结局 |
| :--- | :--- | :--- | :--- |
| 曲线缩短流 | $\int 1\,ds$（长度） | $\kappa\mathbf{N}$ | 坍缩为点 |
| 弹性流 | $\int\kappa^2\,ds$ | 高阶项（含 $\kappa$ 的导数） | 趋于直线/圆，**不缩短** |

### 3. 视觉与拓扑性质

- 长度不缩短，弯曲展开。
- 全程光滑，不产生折痕、不自交、微分不归零。
- **拓扑不变**：最终曲线（如"8"字）保持初始的转数（Whitney–Graustein 体现）。
- 最小能量曲线称为**弹性曲线（Elastica）**或**欧拉弹性曲线**，逼真模拟物理弹性条（木条、纸条）。
- 离散弹性流在网格细化下收敛到光滑物理解。

### 4. 离散表达：转角作为综合曲率

离散能量是**转角**（顶点外角）的函数。**注意：不是简单的转角平方和**——转角是顶点左右两侧线段上曲率的积分。

### 5. 平滑实现两种思路

**方法 A：直接梯度下降（不推荐）**

对顶点位置直接求 $E_{\text{bend}}$ 梯度并移动顶点。问题：

- 计算复杂、几何关系冗长。
- 顶点移动改变边长，引入意外膨胀/收缩。
- 闭合性不可控。

**方法 B：基于"角度-长度"的重建（推荐）**

利用离散平面曲线基本定理绕过顶点位置：

1. 从当前曲线提取转角 $\{\phi_i\}$ 和边长 $\{l_i\}$。
2. **缩小转角**（直接降低离散曲率值）。
3. 用基本定理从新的 $\{\phi_i\},\{l_i\}$ **重建曲线**。

优点：算法简单、数值稳定、精确保持边长。

### 6. 闭合曲线的可积性挑战

离散平面曲线基本定理只保证"重建一条开放曲线"——**不保证它闭合**。随意缩小转角后，新曲线大概率不再首尾相连。

> 这是微分几何中的**可积性 / 几何数据相容性**问题：给定数据何时真正能描述一个闭合几何对象？需要找到一组"闭合条件"约束转角和边长的变化。

## 十二、离散弹性杆（Discrete Elastic Rods）

### 1. 模型

将离散空间曲线应用于弹性杆（电缆、头发丝）的物理模拟，**同时最小化曲率与挠率**并跟踪时间演化。

物理机制：

- 曲线惯性地沿当前方向前进。
- 曲率与挠率作为"拉力"将其拉向不同方向。
- 弹性杆趋向于伸直、松开（减小曲率与挠率）。

### 2. 关键经验

仅用**离散折线 + 简单的离散曲率/挠率**就能得到非常丰富、有预测性的弹性杆模型——无需有限元、无需高阶曲线表示。

应用：头发模拟、海底电缆铺设。

### 3. 推广：粘性螺纹（Viscous Thread）

把"最小化曲率积分（应力）"换成"最小化曲率随时间变化率的积分（应变率）"。

经典演示：**蜂蜜被淋到传送带上的螺旋盘绕**——

| 传送带速度 | 螺纹行为 |
| :--- | :--- |
| 低速 | 来回蜿蜒 |
| 中速 | 小圈盘绕 |
| 高速 | 大循环 / 之字形 |

数值模拟与真实物理高度吻合。

## 十三、解结问题与排斥能量

### 1. 解结问题

> 给定 $\mathbb{R}^3$ 中任意闭曲线，能否在不让曲线穿过自身的前提下变形为一个圆？

**计算复杂度仍未解决**。微分几何的尝试方法：定义某种**惩罚能量**，让其极小化过程将曲线"解开"，从浸没（允许自交）推向嵌入（无自交）。

### 2. 朴素静电排斥（失败）

把曲线想成均匀分布的负电荷：
$$E_{\text{naive}}=\iint\frac{1}{\|\mathbf{x}(s)-\mathbf{x}(t)\|^2}\,ds\,dt$$

**问题**：相邻参数 $s\approx t$ 的点空间上也很近，分母趋零，积分**爆炸**——能量对局部结构过度敏感，无几何意义。

### 3. Möbius 能量（正则化）

只惩罚"空间近 + 参数远"的点对：
$$E_{\text{Möbius}}=\iint\left(\frac{1}{\|\mathbf{x}(s)-\mathbf{x}(t)\|^2}-\frac{1}{\|s-t\|^2}\right)\,ds\,dt$$

第二项按参数距离构造的"理想参考"在相邻处抵消第一项，能量不再发散。

### 4. 数值过程

1. 离散化曲线为多边形。
2. 计算 $E_{\text{Möbius}}$ 关于顶点的梯度。
3. 沿梯度下降演化（**梯度流**）。

**结果**：复杂打结自交曲线 → 经过迭代解缠为完美圆环（圆是全局最小值之一）。
**局限**：可能陷入**局部最小值**（如简单结），不保证收敛到最光滑曲线。

### 5. 切点能量（Tangent-Point Energy）

不只用欧氏距离，还利用两点的**切向量**与连接向量的几何关系定义排斥——更精确地刻画"厚度"和自避免力。

### 6. 应用

- **曲面上的空间填充曲线**：在二维曲面绘制充满表面但不自交的曲线。
- **体积内的纤维打包**：封闭体积内多条曲线互不缠绕。
- **图布局（Graph Layout）**：允许边弯曲，用排斥能避免边交叉，得到更清晰的网络图。

## 十四、涡丝动力学：桥本流（Hashimoto Flow）

### 1. 物理背景

- **涡度（Vorticity）**：速度场的旋度 $\nabla\times\mathbf{v}$，刻画流体微团的旋转。
- **涡丝（Vortex Filament）/ 涡环**：涡度高度集中的细长管，流体沿其轴线旋转——典型如**烟圈**、海豚吐出的气泡环。涡丝自身的运动由**Biot–Savart 定律**给出（沿曲线的诱导速度积分），桥本流是其在局部诱导近似下的经典极限。

### 2. 桥本流方程

涡丝中心线 $\gamma$（弧长参数化）的演化：
$$\frac{\partial\gamma}{\partial t}=\frac{\partial\gamma}{\partial s}\times\frac{\partial^2\gamma}{\partial s^2}$$

分解：

- $\partial\gamma/\partial s=T$（单位切线）。
- $\partial^2\gamma/\partial s^2=\kappa N$（曲率法线）。
- $T\times N=B$（副法线）。

故：
$$\frac{\partial\gamma}{\partial t}=\kappa\,\mathbf{B}$$

**通俗解读**：每点沿副法线方向移动，速度与曲率成正比。

### 3. 圆环涡的运动

- 圆上 $\kappa$ 处处为常数。
- $\mathbf{B}$ 垂直于圆所在平面。
- 故圆环**整体平移**，速度与曲率成正比。
- **现实对应**：**小烟圈比大烟圈跑得快**——曲率与半径成反比，半径小 → 曲率大 → 平移快（玩具涡环枪正是利用这一点）。

### 4. 离散化与一般曲线

非圆曲线（曲率不均）需要离散化：

- 用离散副法向、离散曲率（沿用之前的离散工具）。
- 显式时间步进迭代。
- 现象：椭圆环在平移的同时局部变形振荡。

### 5. 深层联系：可积系统与离散几何力学

桥本流是**离散微分几何**精彩案例之一：

- 简单离散化即可捕捉基本行为。
- 精密离散化方案（特定变换）能精确保持光滑流的不变量——对应物理中的能量/动量/辛守恒。
- 桥本流（及其离散版）与**可积系统**有深厚联系，属于**离散几何力学**前沿。

## 十五、丰富涡环模型：从中心线到带厚度

### 1. 引入半径变量

在中心线之外加入**半径**作为额外变量（沿曲线位置变化）——打破"处处均匀厚度"的理想化。

### 2. 拓扑变化与互动

新模型可预测/模拟：

- 两个气泡环移动并相互作用。
- **合并与分裂**等拓扑事件。
- 真实实验视频与数值模拟达到高度吻合。

### 3. 应用：墨水吊灯

将略重于水的小墨滴滴入水中，演化出复杂的"墨水吊灯"形态。

- 不必昂贵的全三维体积模拟。
- 用**离散曲线 + 半径函数**这套低维表示即可精确建模。
- 高效、简洁，复现经验观测。

**核心洞察**：选对离散模型（中心线 + 半径），就能用极低维表示捕捉高维流体动力学行为。
