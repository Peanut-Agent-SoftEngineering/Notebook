# CMU_DDG_p15

> 主题：从平面曲线推广到二维曲面 —— 参数化、浸入、嵌入、正则同伦、诱导度量、黎曼流形

## 一、研究曲面的两种视角

### 1. 外在描述（Extrinsic）

把曲面看作嵌入在更高维空间（通常 $\mathbb{R}^3$）中的子集，关心点在外部空间的坐标位置。

- 典型方式：**局部参数化**——用一张张"小补丁"（局部平面）映射变形到三维。
- 类比：地球的**地图集**。一本地图集由许多分页组成，每页地图描绘地球一小块；要看全貌必须拼接所有图。
- 局限：仍依赖经纬度（外部坐标），且许多复杂曲面不存在简洁的全局表达式。

### 2. 内在描述（Intrinsic）

只关心几何**自身**的局部度量信息（边长、角度、面积），完全不提及它在哪个外部空间里。

- **关键洞察**：仅凭一张二维地图，我们仍能知道：
  - 角度（地图类型正确时可精确测量）
  - 方向、面积、距离
  - 拓扑关系（哪些区域接壤）
- **内在的强大之处**：许多曲面在 $\mathbb{R}^n$ 中的全局形状难以描述，但内在视角下却非常自然（例如双曲几何）。

### 3. 网格的两种表示对照

| 表示 | 数据 | 性质 |
| :--- | :--- | :--- |
| 外在 | 顶点列表 + 每个顶点的 $(x,y,z)$ 坐标 | 直接给出嵌入位置 |
| 内在 | 仅给出**所有边的长度** | 凭长度即可推出三角形面积、角度、顶点处的**角度缺陷** |

> 内在描述足以传递几乎所有几何信息——这是 DDG 的核心切入口之一。

### 4. 方法论

继续使用**外微分**作为讨论曲面的基本语言。原因：可以非常直接地把光滑结论翻译到离散设置（DEC），构建可计算算法。

## 二、参数化曲面与微分

### 1. 参数化曲面

- **定义**：映射 $f: U \subset \mathbb{R}^m \to \mathbb{R}^n$。
  - $U$：参数域（通常是平面区域）。
  - 像 $f(U)$ 即曲面。
- **要求**：连续、可微、光滑（以便定义法向、曲率等）。
- **本质**：参数化是一种**外在描述**——把曲面写成 $\mathbb{R}^n$ 的子集。

### 2. 鞍面例子

- 参数域：$U = \{(u,v) \mid u^2+v^2 \le 1\}$。
- 映射：$f(u,v) = (u, v, u^2 - v^2)$。
- 同一鞍面也可用 $\tilde{f}(u,v)=(u+v, u-v, 4uv)$ 等多种参数化表达。
- **结论**：**形状唯一，参数化无穷**。同一形状可由多种参数化产生；参数化只是描述曲面的工具。

### 3. 微分 $df$：橡胶板上的箭头

- **几何直觉**：在橡胶板（参数域）上画一个小箭头 $X$，把橡胶板拉伸成三维曲面后，那个箭头会被推送、旋转、拉伸成新箭头。$df$ 就是描述这一变化的工具。
- **作用**：把参数域上的切向量 $X$ **推前（push forward）** 到三维空间，得到 $df(X)$。
- **形式**：$df$ 是一个**向量值 1-形式**，写作
  $$df = \frac{\partial f}{\partial u}\,du + \frac{\partial f}{\partial v}\,dv$$
  其中 $du, dv$ 是基 1-形式，提取向量在 $u,v$ 方向的分量。

### 4. 鞍面计算示范

$f(u,v) = (u,v,u^2-v^2)$，则
$$\frac{\partial f}{\partial u}=(1,0,2u),\quad \frac{\partial f}{\partial v}=(0,1,-2v)$$
$$df = (1,0,2u)\,du + (0,1,-2v)\,dv$$

对常向量场 $X = \tfrac{3}{4}\partial_u - \tfrac{3}{4}\partial_v$：
$$df(X) = \left(\tfrac{3}{4},\ -\tfrac{3}{4},\ \tfrac{3}{2}u + \tfrac{3}{2}v\right)$$

### 5. 与雅可比矩阵的关系

- $df$ 在标准基下的矩阵表示就是**雅可比矩阵** $J_f$（从 $\mathbb{R}^2 \to \mathbb{R}^3$ 时是 $3 \times 2$）。
- $df(X) = J_f X$（列向量乘法）。
- **为什么不直接用雅可比？**
  - 雅可比依赖具体坐标，不便向无限维推广（如曲率流分析）。
  - 离散设置中，$df$ 自然对应**网格边向量**；雅可比则按三角形定义，与其他离散算子不匹配。
- **跨学科同义词**：在固体力学中雅可比叫**变形梯度（Deformation Gradient）**——同一对象的不同名字。

## 三、浸入（Immersion）：曲面"良好"的局部条件

### 1. 定义

映射 $f: U \to \mathbb{R}^n$ 是**浸入**当且仅当 $df$ 处处**非退化**：
$$df(X) = 0 \iff X = 0$$

直觉：任何非零切向量都不会被压扁成零。曲面没有局部"挤压"成一个点的退化。

### 2. 浸入保证的几何量

只要 $f$ 是浸入，下列局部量都良定义：
- 切向量、切平面
- 单位法向量
- 度量、长度、角度
- 各阶微分几何量（曲率、Laplacian……）

> **重要澄清**：自交不破坏浸入性——浸入只看每点的微分。一个全局有自交的曲面，只要每个局部邻域里 $df$ 非退化，依旧可正常做微分几何。

### 3. 反例：球面经纬度参数化在两极退化

参数化 $f(u,v) = (\cos u \sin v, \sin u \sin v, \cos v)$，$u\in[0,2\pi]$ 经度，$v\in[0,\pi]$ 纬度。

在 $v=0$（北极）处 $\partial f/\partial u = 0$：站在北极说"向东走"没有意义，沿经度方向的切向量被压扁为零。

**结论**：该参数化在两极**不是浸入**，其余处处是。

## 四、嵌入（Embedding）：曲面"良好"的全局条件

### 1. 定义

参数化 $f: U \to \mathbb{R}^3$ 是**嵌入**当且仅当：
1. $f$ 是连续双射（单射 + 满到像）
2. 逆映射 $f^{-1}: f(U) \to U$ 也连续

直观：一对一、不撕裂、不粘连、不自交，且**保持域的拓扑结构**。

### 2. 反例

| 反例 | 失败原因 | 拓扑代价 |
| :--- | :--- | :--- |
| 8 字形自交 | 不是单射，两点映到同一像点 | 圆盘拓扑变成带环结构 |
| 把展开图折成立方体 | 单射但逆不连续（要"撕开"才能展平） | 圆盘拓扑变成球面拓扑 |

> **核心区分**：**嵌入 ≠ 不自交**。粘合边界即使没有自交也会破坏拓扑。

### 3. 浸入 vs 嵌入对照表

| 特性 | 浸入 | 嵌入 |
| :--- | :--- | :--- |
| 性质 | 局部（点态微分） | 全局（拓扑） |
| 允许自交 | ✅ | ❌ |
| 检查难度 | 容易（逐点查切平面） | 困难（全局碰撞检测） |
| 几何运算 | 长度、角度、曲率均可正常计算 | 同样可做，但要求更强 |
| 离散自然性 | 高（网格折叠/穿透不破坏局部度量） | 低（要求物理上不穿透） |
| 数学关系 | 任何浸入都是**局部嵌入** | 嵌入 ⟹ 浸入，反之不成立 |

### 4. "随机网格"的概率直觉

给三角网格顶点分配**完全随机**的坐标：
- 大概率得到自交（**非嵌入**）
- 极低概率得到退化三角形（即 $df$ 退化，**非浸入**）

→ 浸入比嵌入普遍得多。这也是为什么 DDG 默认在浸入框架下工作。

### 5. 浸入集合的代数性质

浸入集合更像**向量空间**：把两个浸入 $f_1, f_2$ 的顶点坐标相加，结果**大概率仍是浸入**（除非微分恰好抵消，极罕见）。但两个嵌入相加**极有可能产生自交**而失去嵌入性。

## 五、正则同伦与球面外翻

### 1. 动机：从静态到动态

不仅研究静态曲面，还要研究"融化、混合、生长"等**动态过程**。需要捕捉"好的运动"——没有尖锐折痕、没有突然捏合。

### 2. 正则同伦定义

给定两个浸入 $f_0, f_1: U \to \mathbb{R}^3$，**正则同伦**是连续映射
$$H: U \times [0,1] \to \mathbb{R}^3$$
满足：
1. **端点**：$H(\cdot, 0) = f_0$，$H(\cdot, 1) = f_1$
2. **浸入保持**：每个时刻 $t \in [0,1]$，$H(\cdot, t)$ 都是浸入

简言之：整个形变过程曲面始终光滑，无局部退化。

### 3. 直觉对比

| 概念 | 视角 | 类比 |
| :--- | :--- | :--- |
| 浸入 | 静态 | 一张抓拍的"好照片"，局部完好 |
| 正则同伦 | 动态 | 一段流畅的"好电影"，每一帧都好 |

**好例子**：球面光滑拉伸成椭球面。
**坏例子**：形变中出现尖点（如压成"8"字形的瞬间），那一刻 $df$ 部分退化。

### 4. 参数化噪声警告

参数化是**独立于几何品质**的数据。一个完美圆若用速度极不均匀的参数表达，参数导数可能在某处为零，但**几何形状本身没有折痕**。判断正则同伦时关注的是**几何**是否退化，而非参数化。

### 5. 曲线 vs 曲面：同伦类的差异

| 维度 | 完全不变量 | 能否"翻面" |
| :--- | :--- | :--- |
| 平面曲线 | **转动数**（Whitney–Graustein 定理） | 不能：逆时针圆与顺时针圆不同伦 |
| 三维曲面 | 不再是简单整数 | **可以**：球面外翻 |

**Whitney–Graustein 定理**：两条平面曲线正则同伦当且仅当转动数相同。任何尝试把逆时针圆变成顺时针圆都会在某瞬间产生速度归零（尖点）。

### 6. 球面外翻（Sphere Eversion）

**斯蒂芬·斯梅尔**博士论文证明：球面可以通过浸入正则同伦"由内向外翻"，他的导师起初认为这不可能。

- **过程**：灰色球（法线全外）→ 正则同伦 → 蓝色球（法线全内）
- **关键约束**：始终保持浸入，**允许自交**，禁止捏合/撕裂/开洞
- **关键直觉**：曲面有比曲线高得多的自由度；自交在曲线中不可行（一维太"窄"），但在曲面上可以

| 已知版本 | 特点 |
| :--- | :--- |
| Morin 外翻 | 最早的版本（Bernard Morin 发明） |
| 最小弯曲外翻 | 最小化总弯曲能量 |
| 直纹曲面外翻 | 用直线扫过的曲面构造，描述清晰 |

**Morin 的核心创意——中间对称表面**：
1. 找一个曲面浸入，它在某个对称变换下不变（一半法线朝内、一半朝外）。
2. 从普通球面正则同伦到此中间表面。
3. 对称地反播这个过程，得到完整的外翻。

后续版本（如最小弯曲）借助**曲率流**：让中间表面在不稳定平衡处加微小扰动，自然流向"全外翻"或"全内翻"。

**Bernard Morin** 是位**盲人数学家**，幼年失明，通过**触摸物理模型**完成所有这类拓扑研究。他常说："我们的空间想象力是通过操纵物体来构建的"——这一观点至今仍指导几何教学：从早期的实物模型到今天的 **3D 打印**，触觉始终是几何直觉的重要来源。

> 推荐观看 **《Outside In》**（1994，由 Geometry Center 制作）——一部用动画完整展示球面外翻过程的经典数学科普影片，是最早进入大众视野的高维拓扑可视化作品之一。

> **教训**：低维几何直觉在高维或复杂拓扑中常会失效。

## 六、黎曼度量：测量曲面上的长度与角度

### 1. 动机：参数平面上的点积无效

参数平面上的两个向量 $X, Y$，它们的标准点积 $X \cdot Y$ **不等于**这对向量被推到曲面后的实际几何（长度、夹角）。原因：$df$ 的拉伸/扭曲会改变向量的长度和方向。换不同参数化，结果还会不同。

### 2. 诱导度量定义

正确做法：先把 $X, Y$ 通过 $df$ 推到 $\mathbb{R}^3$，**在三维空间里**做点积，并把结果"拉回"到二维参数域：
$$g(X, Y) := \langle df(X),\, df(Y) \rangle_{\mathbb{R}^3}$$

- $g$ 编码了从平面到曲面的所有局部失真，但**保留在二维坐标里**。
- 称为**诱导黎曼度量**或**第一基本形式**。

### 3. 关键术语澄清

- **黎曼度量（Riemannian metric）**：作用于**切向量**的内积，定义角度和长度。
- **距离度量（point-set metric）**：作用于**点对**的距离函数，满足三角不等式。
- 二者英文都叫 metric，但数学对象不同。

### 4. 矩阵表示：第一基本形式 $\mathbf{I}$

记参数坐标基向量为 $e_1, e_2$，定义
$$I_{ij} := g(e_i, e_j) = \langle df(e_i),\, df(e_j) \rangle$$
则任意 $X, Y$ 满足 $g(X, Y) = X^T \mathbf{I}\, Y$。由内积的双线性，知道基向量对的值就完全决定 $g$。

### 5. 雅可比公式

设 $J$ 是 $f$ 的雅可比矩阵（$3 \times 2$），则
$$\boxed{\mathbf{I} = J^T J}$$
推导：$g(X,Y) = \langle JX, JY\rangle = X^T(J^T J) Y$。

### 6. 鞍面 $f(u,v)=(u,v,u^2-v^2)$ 实算

$$J = \begin{bmatrix} 1 & 0 \\ 0 & 1 \\ 2u & -2v \end{bmatrix},\qquad
\mathbf{I} = J^T J = \begin{bmatrix} 1+4u^2 & -4uv \\ -4uv & 1+4v^2 \end{bmatrix}$$

### 7. 第一基本形式的核心性质

**a. 对称性**

矩阵右上 = 左下 = $-4uv$。原因：内积本身满足 $\langle X,Y\rangle = \langle Y,X\rangle$，所以 $\mathbf{I}$ 必然对称。**对称性可作为计算正确性的快速检查**。

**b. 经典记号 $E,F,G$**

由对称性，只需三个独立分量：
$$\mathbf{I} = \begin{bmatrix} E & F \\ F & G \end{bmatrix}$$
鞍面例：$E = 1+4u^2$，$F = -4uv$，$G = 1+4v^2$。

**c. 点依赖**

$\mathbf{I}$ 是**点 $p$ 的函数**——曲面在不同位置拉伸程度不同，矩阵也不同。严谨写法 $g_p(X,Y)$；但在公式推导中常省略 $p$。

**d. 坐标依赖与几何不变性**

$E,F,G$ 的具体数值会随参数化改变，**它们没有独立的几何意义**，只是局部坐标下的计算工具。但用它们算出的**长度、角度、面积是坐标无关的**——就像墨卡托投影下格陵兰岛被拉得很大，但球面上真实面积不变。

## 七、共形参数化（Conformal Parameterization）

### 1. 长度与角度的取舍

任何 2D ↔ 3D 映射几乎总会扭曲几何。无法同时保留所有长度、角度、面积。

- **弧长参数化（Isometric）**：单位长度对应单位长度，完全保长。**对绝大多数曲面（如球面）不存在**。
- **共形参数化**：放弃保长，只**保角**——映射在每点是欧氏度量的正缩放。

### 2. 共形定义

参数化 $f$ 是**共形的**，当且仅当存在标量函数 $\phi(p)$ 使得对域中任意 $X, Y$：
$$\langle df_p(X),\, df_p(Y)\rangle_{\mathbb{R}^3} = e^{2\phi(p)}\,\langle X, Y\rangle_{\mathbb{R}^2}$$

- **共形因子** $e^{2\phi(p)}$（也常写 $\lambda^2(p)$）：仅依赖于点 $p$ 的标量，描述该点处的**拉伸或收缩程度**。
- **坐标不变性**：共形性是几何属性，与平面坐标系无关。

### 3. 共形映射的优势

- **角度完全保留**：曲线间夹角不变。
- **失真用一个数字描述**：每点只需一个共形因子，分析极为简化。
- **第一基本形式特别简单**：始终是单位矩阵的标量倍：
  $$\mathbf{I} = e^{2\phi}\,\mathbb{I}_2$$

### 4. 实例

某参数化 $f(u,v)$ 的诱导度量化简为
$$g = (u^2+v^2+1)\begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$$
这正是共形度量。可视化：均匀网格在共形参数化下，被映射后**网格线仍正交**，但单元大小被局部因子缩放（中心小、边缘大）。

### 5. 工程意义

实际网格处理中，给定离散三角网格，需要**自动算法**把它压平到二维平面，并保证诱导度量尽量共形。这种共形参数化是**几何加工**与**数字制造**（裁片、UV 展开）的关键工具，因为它最小化角度畸变。

## 八、抽象黎曼度量与黎曼流形

### 1. 提升抽象层级：内禀视角

之前我们通过映射 $f$ 把三维内积**拉回**到二维参数域。下一步：**直接定义内积，丢掉曲面来源**。

> "我不需要知道这个度量从哪个曲面来——只要在每点给我一个光滑变化的内积，我就能做所有几何。"

这就是**抽象黎曼度量**。

### 2. 共形度量的通用形式

$$\langle X, Y\rangle_g = \lambda^2(u)\,\langle X, Y\rangle_{\text{Euclidean}}$$

只要给出标量函数 $\lambda(u) > 0$，就得到一族共形度量。

### 3. 经典例子：庞加莱圆盘（双曲度量）

- **域**：开单位圆盘 $\mathbb{D} = \{p \mid \|p\| < 1\}$。
- **度量**：$\displaystyle \langle X, Y\rangle_g = \frac{4\,\langle X, Y\rangle_{\mathrm{Eucl}}}{(1-\|p\|^2)^2}$

性质：
- 与欧氏度量**共形**（仅相差标量因子）。
- 中心 $p=0$ 处分母为 $1$，度量近似欧氏。
- 边界 $\|p\| \to 1$ 时分母 $\to 0$，**度量趋于无穷**。
- 几何含义：边界在"无穷远处"——这是双曲几何的标志特性。可视化时越靠边界颜色越深，表示物理距离被无限拉伸。

### 4. 在抽象度量上的几何运算

无需任何嵌入，仅凭度量 $g$ 即可计算：

| 量 | 公式 |
| :--- | :--- |
| 向量长度 | $\|X\|_g = \sqrt{g_p(X,X)}$ |
| 夹角 | $\cos\theta = \dfrac{g_p(X,Y)}{\|X\|_g\,\|Y\|_g}$ |
| 曲线长度 | $\int \sqrt{g_{\gamma(t)}(\gamma'(t),\gamma'(t))}\,dt$ |
| 面积 | 通过 $\sqrt{\det g}$ 积分 |

## 九、嵌入定理：抽象度量何时能"画"出来

### 1. 希尔伯特定理（否定结论）

**双曲度量（庞加莱圆盘）无法被光滑全局等距嵌入到 $\mathbb{R}^3$**。

直观原因：要在三维空间模拟边界处度量爆炸，需把圆盘"起皱"并越来越密集堆叠，最终自交或爆炸。

### 2. 纳什嵌入定理（肯定结论）

**任何足够光滑的黎曼流形，都可以等距嵌入到足够高维的欧氏空间 $\mathbb{R}^N$ 中**（$C^k$ 光滑、$k$ 可指定）。

- 维度上界：$n$ 维流形通常需 $N = \tfrac{n(3n+11)}{2}$（$C^\infty$ 版本）。
- 工程意义：理论保证抽象度量**总对应某个真实的几何形状**——可以放心使用。

### 3. 纳什–Kuiper 定理（弱光滑下的奇迹）

对任何**短嵌入**（不增长距离的嵌入），存在**至少一次可微（$C^1$）的等距嵌入**，方法是不断添加越来越小的褶皱。

### 4. 经典例子：扁平环面的 $C^1$ 嵌入

- **拓扑**：环面 $\cong$ 正方形左右、上下边粘合。
- **扁平环面**：处处曲率为零，貌似无法嵌入 $\mathbb{R}^3$（甜甜圈总是要弯）。
- **纳什–Kuiper 构造**：从光滑甜甜圈出发，但两个方向环长不等。在短方向加小波纹拉长，但破坏了另一方向的长度，再加波纹修正……极限下得到一个皱巴巴的、$C^1$ 等距的扁平环面嵌入。

### 5. 工程实践启示

- **不需要显式嵌入**：渲染、模拟中直接用度量张量计算长度、角度、曲率即可。
- **避免不可行嵌入**：处理曲面、非标准空间（多层流形等）时，基于度量的计算总比寻找嵌入更通用、高效。

## 十、抽象流形：从图集到内禀几何

### 1. 局部图与诱导度量

复杂曲面整体不规则，但**局部**像平坦补丁。抽象做法：

- 一组**图（Charts）** $\{\phi_i: U_i \to \mathbb{R}^3\}$，每个 $U_i \subset \mathbb{R}^n$ 是开集。
- **覆盖**：所有 $\phi_i(U_i)$ 的并集等于整个曲面。
- 在每个 $U_i$ 上定义诱导度量 $g_i(X, Y) := \langle d\phi_i(X),\, d\phi_i(Y)\rangle$。

### 2. 重叠区一致性

同一点在多个图中的重叠区，用 $g_i$ 与 $g_j$ 算出的内积**必须相同**。否则切向量长度和角度依赖"视角"，无法良定义。

机制：在 $U_i$ 中有 $X_i, Y_i$，通过过渡映射变换到 $U_j$ 得 $X_j, Y_j$，则
$$g_j(X_j, Y_j) = \langle d\phi_j(X_j), d\phi_j(Y_j)\rangle = \langle d\phi_i(X_i), d\phi_i(Y_i)\rangle = g_i(X_i, Y_i)$$

### 3. 关键飞跃：丢掉嵌入

既然不同图的度量必须一致，我们**根本不需要知道这些图来自哪里**——任何一组：
- 开集 $\{U_i \subset \mathbb{R}^n\}$
- 可逆且双向可微的过渡映射
- 在重叠区相容的局部度量 $g_i$

构成一个**黎曼流形**。

> **过渡映射要求双射且双向可微**：保证流形的微分结构一致，使切向量能在不同图之间合法推送。

### 4. 流形的本质

- 通过**粘合（identification）**重叠点，所有局部图拼成一个完整空间。
- 不需要想象它嵌入在更高维 $\mathbb{R}^N$ 中。
- 在这个空间上"漫游"，所有几何量（长度、角度、曲线长度、面积）均**内在地、一致地**良定义。

### 5. 几何计算工作流总结

1. **局部外在图像**：$\mathbb{R}^N$ 中的参数化映射 $f$（早期曲面理论）。
2. **拉回到内禀**：在二维域上得到诱导度量 $g$。
3. **二维计算**：所有量（长度、角度、面积、曲率）在局部坐标下计算。
4. **全局拼接**：多个补丁通过过渡映射粘合，要求度量在重叠区一致。
5. **最终洞察**：所有信息都可压在二维坐标系 + 度量上，不需要外部空间。

### 6. 在曲面上"本地化"几何

记参数为 $(u,v)$，度量张量为
$$ds^2 = E\,du^2 + 2F\,du\,dv + G\,dv^2$$
所有几何运算（长度、角度、面积）**直接在二维域中**用 $E, F, G$ 完成，无需访问 $\mathbb{R}^3$。

### 7. 内禀 vs 外在的总结

- **内禀几何**：依赖度量，不依赖嵌入方式。本课目前的核心。
- **外在几何**：引入曲面法线、与外部空间的关系。是下一步——把内禀度量与曲面法线、外微分语言联系起来。

## 十一、关键术语速查

| 术语 | 含义 |
| :--- | :--- |
| 外在描述 | 把曲面看作 $\mathbb{R}^N$ 子集；通过坐标定位 |
| 内在描述 | 仅凭曲面自身度量描述几何，不需要外部空间 |
| 参数化曲面 | 映射 $f: U \subset \mathbb{R}^m \to \mathbb{R}^n$，像 $f(U)$ 即曲面 |
| 微分 $df$ | 把参数域切向量推前到目标空间的向量值 1-形式 |
| 雅可比矩阵 | $df$ 在标准基下的矩阵；固体力学中称变形梯度 |
| 浸入 | $df$ 处处非退化；曲面良好的**局部**条件 |
| 嵌入 | 浸入 + 全局单射 + 逆连续；保持拓扑的**全局**条件 |
| 局部嵌入 | 浸入的等价表述：每点附近都是嵌入 |
| 正则同伦 | 浸入之间的连续形变，每个时刻都保持浸入 |
| Whitney–Graustein | 平面曲线正则同伦 ⟺ 转动数相同 |
| 球面外翻 | $S^2$ 通过浸入正则同伦由内向外翻；曲线版本不可行 |
| 黎曼度量 | 切空间上光滑变化的内积 $g_p$ |
| 诱导度量 | $g(X,Y) = \langle df(X), df(Y)\rangle$；从嵌入拉回 |
| 第一基本形式 $\mathbf{I}$ | 诱导度量的矩阵：$\mathbf{I} = J^T J$；常记 $E, F, G$ |
| 共形参数化 | 度量是欧氏度量的标量倍：$g = e^{2\phi}\langle\cdot,\cdot\rangle$ |
| 共形因子 | 描述共形映射在每点的局部缩放 |
| 庞加莱圆盘 | 双曲度量：$g = 4\langle\cdot,\cdot\rangle/(1-\|p\|^2)^2$ |
| 希尔伯特定理 | 双曲度量无法 $C^\infty$ 等距嵌入 $\mathbb{R}^3$ |
| 纳什嵌入定理 | 任何黎曼流形可等距嵌入足够高维 $\mathbb{R}^N$ |
| 纳什–Kuiper | 短嵌入可改造为 $C^1$ 等距嵌入；扁平环面经典构造 |
| 黎曼流形 | 一组开集 + 相容过渡映射 + 重叠一致的局部度量 |
| 过渡映射 | 不同局部图之间的双射双向可微变换 |
| 角度缺陷 | 顶点处缺失/盈余的角度总和；内在可计算 |
