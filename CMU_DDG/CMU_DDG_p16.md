# CMU_DDG_p16

> 主题：曲面外在几何 —— 法向量、高斯映射、向量面积、面积形式、复结构、霍奇星算子在曲面上的推广

## 一、回顾：参数化曲面与浸入

### 1. 参数化曲面与微分

- **参数化曲面**：映射 $f: \Omega \subset \mathbb{R}^2 \to \mathbb{R}^3$。把二维"橡胶布" $\Omega$ 拉伸到三维空间。
- **微分** $df$：把 $\Omega$ 上的切向量 $X$ "推前"到曲面上的切向量 $df(X)$。它是一个**向量值 1-形式**。

### 2. 浸入条件

$f$ 是浸入 $\iff df$ 非退化（$df(X)=0 \Rightarrow X=0$）。直观：任何非零切向量不被压扁成零。

这是定义曲面上一切几何量（法线、度量、曲率）的**先决条件**。

### 3. 诱导黎曼度量

$$g(X,Y) := \langle df(X),\, df(Y)\rangle_{\mathbb{R}^3}$$

把 $\mathbb{R}^3$ 中的内积**拉回**到二维参数域，使得我们能用局部坐标算长度和角度，而无需离开二维。

> 这一节起，我们专注**外在几何**：曲面在 $\mathbb{R}^3$ 中的法向、面积形式以及与外微分的联系。

---

## 二、曲面法向量

### 1. 严格定义

点 $p$ 处的**法向量**是与所有切向量正交的向量。即对所有 $X \in T_pM$：
$$\langle n,\ df_p(X)\rangle = 0$$

几何上：法向量是"直接离开曲面"的方向，类比平面曲线切向量旋转 90° 得到的法向。

### 2. 非唯一性与单位化

仅"正交于切空间"不能唯一确定法向量：
- 长度任意
- 方向 $\pm$ 任意

修正：要求**单位**且**连续选择**。

---

## 三、高斯映射（Gauss Map）

### 1. 定义

**高斯映射**是连续映射 $N: M \to S^2$，把曲面上每点 $p$ 映到该点的单位法向量 $n(p)$。

- 几何视角：每个曲面隐含一个"法线曲面"——它的像位于单位球面 $S^2$ 上。
- 用途：是定义**曲率**（高斯曲率 = 高斯映射的局部面积变化率）和后续高斯–Bonnet 定理的基础。

### 2. 存在性与可定向性

**核心问题**：是否任何曲面都有全局连续的高斯映射？

**答**：不一定。例如**莫比乌斯带**：选定一点的法向量后沿曲面绕一圈回到起点，法线已翻转方向；再绕一圈又会突然翻回。无法选出全局连续的单位法向。

> **重要结论**：**曲面是可定向的 $\iff$ 它存在全局连续的高斯映射**。
> 不可定向曲面（莫比乌斯带、克莱因瓶）没有全局高斯映射。

### 3. 工程意义

- **法线方向决定光照**：渲染中（如 Blinn–Phong 模型）法线连续性是基本要求，否则会出现视觉断裂。
- **法线贴图**：本质是把高斯映射存为纹理，让低模拟出高模的高频细节。
- **拓扑联系**：高斯映射的度（degree）与欧拉示性数相关，是 Gauss–Bonnet 定理的核心。

## 四、用切向量叉积构造法线

### 1. 基本方法

在每点取两个切向量（通常是参数化的偏导）做**叉积**。

$$N = \frac{\partial f}{\partial u} \times \frac{\partial f}{\partial v},\qquad n = \frac{N}{\|N\|}$$

- 叉积自动垂直于两个切向量，因此是法向量。
- 长度等于由两切向量张成的**平行四边形面积**。
- 方向由参数化的**定向约定**决定（$+$ 或 $-$ 二选一）。

**前提**：两个切向量必须**线性独立**（即点处 $f$ 是浸入），否则叉积为零。

### 2. 球面例子与极点退化

球面参数化 $f(u,v) = (\cos u \sin v,\ \sin u \sin v,\ \cos v)$。

- **正常点**：偏导数线性独立，叉积给出有效法线。
- **两极**（$v=0$ 或 $v=\pi$）：$\partial f/\partial u = 0$，叉积为零，**该参数化在极点不是浸入**。

> 几何含义：北极点没有"东方"——经度全部交汇，沿经度方向的切向量被压扁成零。

整体上球面仍可定向、有全局高斯映射，只是这一参数化方案在两极有奇点。可换图覆盖或取极限处理。

### 3. 中心球的法线"反方向"

以原点为中心的球面，按上述方法算出的单位法线 $n = -p/\|p\|$，即**指向原点**。这是**约定问题**：是否取反由参数化方向决定，实际使用时任取一种即可。

## 五、高斯映射的满射性与单射性

### 1. 满射性（始终成立，对闭光滑曲面）

**结论**：闭光滑曲面上，对任何单位向量 $u \in S^2$，至少存在一点法线 $= u$。

**希尔伯特构造法**：
1. 取一张法向量为 $u$ 的平面，置于无穷远。
2. 沿 $-u$ 方向缓慢推近曲面。
3. **首次接触**时，平面与曲面相切——该接触点的法线必平行于 $u$。

直觉：任何方向，都能在曲面上找到一面"朝着该方向"的小区域。

### 2. 单射性（不一定成立）

**问**：每个 $u$ 是否仅对应曲面上一点？

**答**：**否**。例如花瓶或哑铃形曲面，多个不同点可能共享相同法线方向。

**何时单射？** 当且仅当曲面**严格凸**：
- **凸**：内部任意两点连线不出域。
- **严格**：表面无平面区域（立方体不算严格凸——它的面是平的）。
- 立方体反例：整面共享一个法向。

**例**：单位球面是严格凸的——其法线恰为位置向量的相反数，是完美的一一对应。

## 六、向量面积（Vector Area）

### 1. 直觉与动机

漂浮在空间中的小补丁（不一定平），用一个**箭头**代表它所有点法线的总趋势。

### 2. 定义与公式

补丁 $S$ 的**向量面积**定义为面元法向的积分：
$$\vec{A} := \int_S n\,dA = \int_S \vec{n}\,dA$$

其中 $\vec{n}\,dA$ 是**向量值 2-形式**：作用于两切向量时，先算它们张成的平行四边形面积，再乘以法向。

### 3. 关键身份：$\vec{n}\,dA = \tfrac{1}{2}\,df \wedge df$

**注意**：$df$ 是**向量值** 1-形式，所以楔积中的乘法替换为**叉积**。

**推导**：对一对切向量 $X, Y$
$$(df \wedge df)(X, Y) = df(X) \times df(Y) - df(Y) \times df(X) = 2\,df(X) \times df(Y)$$
由叉积反对称。所以
$$\boxed{\ \vec{n}\,dA = \tfrac{1}{2}(df \wedge df)\ }$$

### 4. 总向量面积 → 边界积分（斯托克斯）

利用 $d(f\,df) = df \wedge df + f \wedge d(df) = df \wedge df$（因 $d^2=0$）：

$$\int_S \vec{n}\,dA = \tfrac{1}{2}\int_S df \wedge df = \tfrac{1}{2}\int_S d(f \wedge df) = \tfrac{1}{2}\oint_{\partial S} f \times df$$

**沿边界以弧长 $s$ 行走**：
$$\oint_{\partial S} f \times df = \oint f(s) \times \frac{df}{ds}\,ds$$

**几何解释**：每点取**位置向量**与**边界切向量**的叉积，沿边界积分。

### 5. 平移不变性

$f$ 出现在积分里，看似破坏平移不变性。但沿**闭合环路**，常数 $c$ 给出
$$\oint c \times df = c \times \oint df = c \times 0 = 0$$
所以 $\oint f \times df$ 仍是平移不变的。

### 6. 闭曲面的总向量面积为零

闭曲面没有边界，斯托克斯立刻给出
$$\int_{\text{closed}} n\,dA = \oint_{\partial = \emptyset} f \times df = 0$$

> 不要求凸、不要求嵌入，只要闭即可。几何含义：闭曲面各处法线在向量意义上互相抵消。

### 7. 应用：非三角形多边形的法线定义

**痛点**：四边形/五边形等顶点不共面时，**没有自然的法线**——三角剖分方式不同会得到不同法线。

**解法**：用边界积分
$$\vec{N} = \tfrac{1}{2}\oint_{\partial P} r \times dr$$
- **不依赖**多边形内部如何填充或三角化。
- 任何两个共享边界曲线的补丁，总向量面积**完全相同**。

**几何解释**："投影面积最大化"
- 把边界曲线投影到不同方向的平面，计算投影面积。
- **最大投影面积所对应的方向 = 向量面积方向**。
- 即：向量面积是"边界曲线在所有方向的平均阴影最大值"。

## 七、外微分在曲面上的推广

### 1. 自然分裂：组合 vs 几何

把外微积分从平坦 $\mathbb{R}^n$ 推广到曲面（流形）时，运算分成两类：

**组合/拓扑部分**（曲面形状无关，定义不变）：
- **楔积** $\wedge$：满足 $\alpha \wedge \beta = (-1)^{pq}\beta \wedge \alpha$
- **外微分** $d$：线性 + 莱布尼兹 + $d^2 = 0$

> 在离散设置中，这两个算子只依赖**网格连接性**（关联矩阵），完全不知道顶点位置。

**几何部分**（依赖曲面形状）：
- **霍奇星** $\star$：唯一携带几何信息的算子。
- 在离散设置中，$\star$ 由**体积比**定义，必须知道顶点位置和网格形状。

> **关键洞察**：外微积分把曲面的**拓扑/代数结构**与**内禀几何**清晰分离。

### 2. 曲面霍奇星只需两份数据

定义曲面上的 $\star$ 只需：

| 数据 | 用途 | 几何意义 |
| :--- | :--- | :--- |
| **面积形式** $dA$ | 定义 0-形式 与 2-形式 上的 $\star$ | 告诉每点处区域的"大小" |
| **复结构** $J$ | 定义 1-形式上的 $\star$ | 告诉切空间内"如何旋转 90°" |

### 3. 三个等价的几何描述

以下三个对象等价，互相可推：
- **诱导黎曼度量** $g$
- **面积形式** $dA$
- **复结构** $J$

具体关系（见后第十节）：$g(X, Y) = dA(X,\, JY)$。

> **从面积和旋转出发定义几何**比直接从度量出发更几何、更直观。

## 八、面积形式 $dA$

### 1. 推导

由前面的恒等式
$$df(X) \wedge df(Y) = (X \times Y) \cdot n\,dA$$
（这里"$\wedge$"对向量值形式即叉积。）

要分离纯标量面积形式 $dA$：与单位法向量 $n$ 取内积（$n\cdot n = 1$）：

$$\boxed{\ dA = \tfrac{1}{2}\,n \cdot (df \wedge df)\ }$$

### 2. 离散对应

离散化到三角形网格：该公式精确给出**每个三角形的面积**。这是从连续到离散的关键桥梁。

### 3. 与诱导度量的区别

- **诱导度量** $g$：测量长度和角度。
- **面积形式** $dA$：测量面积。它独立于具体坐标嵌入，只关心曲面"大小感"。

---

## 九、霍奇星算子 $\star$ 在曲面上的具体定义

### 1. 类型对应

曲面 $n=2$，$\star$ 把 $k$-形式映为 $(2-k)$-形式：

| 输入 | 输出 |
| :--- | :--- |
| 0-形式（标量函数） | 2-形式 |
| 2-形式 | 0-形式 |
| 1-形式 | 1-形式（依赖复结构） |

### 2. 0-形式：$\star \varphi = \varphi\,dA$

把标量函数 $\varphi$ 升为 2-形式 $\varphi\,dA$（用 $\varphi$ 缩放面积形式）。

**几何**：$\varphi$ 告诉我们标准面积元 $dA$ 在每点应被放大多少倍。

### 3. 2-形式：$\star \omega = $ 密度函数

给定 2-形式 $\omega$，存在唯一标量 $\varphi$ 使 $\omega = \varphi\,dA$。则 $\star\omega := \varphi$。

**几何**：把 $\omega$ 表为标准面积形式的密度。

### 4. 1-形式：依赖复结构 $J$

$$\boxed{(\star\alpha)(X) := \alpha(JX)}$$

直觉：把 1-形式作用于"先把 $X$ 旋转 90° 再代入"。

但**曲面切空间不能用统一矩阵旋转**——每点切平面不同方向。需要**复结构 $J$** 来描述每点的局部 90° 旋转。

## 十、复结构 $J$：曲面上的 90° 旋转

### 1. 平面上的 $J$

平面上 90° 旋转：$(x,y) \mapsto (-y, x)$，矩阵为
$$J = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix},\qquad J^2 = -I$$

### 2. 曲面上：用法向叉积

在 $\mathbb{R}^3$ 中曲面切空间，90° 旋转可由法向 $n$ 的叉积实现：
$$n \times df(X) \in T_pM,\quad \text{且与 }df(X)\text{ 正交、等长}$$

### 3. 参数域上的 $J_f$ 定义

我们希望参数平面上有线性映射 $J_f$ 使其推前后等于这个三维旋转：
$$\boxed{\ df(J_f X) = n \times df(X)\ }$$

由这等式对所有 $X$ 成立，$J_f$ 被**唯一**确定。

### 4. 关键图景：参数平面 vs 曲面

- 在**参数平面**：$X$ 与 $J_f X$ **通常不正交**（普通平面几何无 90°）。
- 通过 $df$ 推到**曲面**：$df(X)$ 与 $df(J_fX)$ **正好正交**。

> $J_f$ 是参数平面上的"为编码三维 90° 旋转而设计的伪旋转"。

### 5. 共形浸入的优雅刻画

**$f$ 是共形浸入 $\iff$ 平面上的 90° 标准旋转 = 曲面上的 90° 旋转 $J_f$。**

即：参数平面里两个互相正交的向量，推前到曲面上仍互相正交。这意味着角度被映射保持——共形性的几何本质。

### 6. 矩阵形式（可选显式计算）

记法向叉积矩阵
$$[n]_\times = \begin{pmatrix} 0 & -n_z & n_y \\ n_z & 0 & -n_x \\ -n_y & n_x & 0\end{pmatrix}$$
雅可比 $A = df$（$3\times 2$）。需要 $2\times 2$ 矩阵 $J$ 满足
$$A J = [n]_\times A$$
解出的 $J$ 就是 $J_f$。

更紧凑的等价形式（用伪逆）：
$$J_f = (A^T A)^{-1} A^T\, [n]_\times\, A$$
共形情况下 $A^T A$ 是单位阵的标量倍，公式大幅简化。

### 7. 曲面 1-形式霍奇星完整定义

$$(\star_f\,\alpha)(X) := \alpha(J_f X)$$

几何：每点切空间中，把 $X$ 绕法向 $n$ 旋转 90° 后再让 1-形式作用。

## 十一、统一关系：度量 = 面积形式 + 复结构

### 1. 关键等式

对任意切向量 $X, Y$：
$$\boxed{\ g_f(X, Y) = dA_f(X,\, J_f Y)\ }$$

**含义**：度量内积 $g(X,Y)$ 等于由 $X$ 与 $J Y$ 张成平行四边形的**有向面积**。

### 2. 推论：内积可由叉积+旋转表达

在曲面上：
$$g(X,Y) = X \cdot (\text{Rotate}_{90°}(Y))$$
（叉积/平行四边形面积 + 90° 旋转 = 内积）

> 揭示了内积、面积、旋转的内在统一：**不需要新运算，三者互相决定**。

### 3. 计算范式

曲面上的微分演算所有公式形式上与平面一致，但每步都需要代入：
- 度量 $g_f$
- 面积形式 $dA_f$  
- 复结构 $J_f$

这三者**紧密耦合**，构成曲面几何的完整描述。

## 十二、Sharp（升）与 Flat（降）算子

### 1. 平面 vs 曲面

- **平坦欧氏空间**：Sharp/Flat 是平凡的——只是把列向量与行向量互换，数值不变。
- **曲面上**：它们**承载几何信息**——把度量 $g$ "吸收"进向量或形式自身。

### 2. Flat $\flat$：向量场 → 1-形式

给定向量场 $X$，定义 1-形式 $X^\flat$ 满足
$$X^\flat(Y) = g(X, Y)\quad \forall\,Y$$
即用**曲面的内积**而非欧氏内积。

### 3. Sharp $\sharp$：1-形式 → 向量场

给定 1-形式 $\alpha$，定义向量场 $\alpha^\sharp$ 满足
$$g(\alpha^\sharp, Y) = \alpha(Y)\quad \forall\,Y$$
即在曲面内积意义下完美代表 $\alpha$ 的向量。

### 4. 为什么不能直接"压平"

**错误想法**：把曲面向量压平到参数域，然后用平面点积。
**问题**：压平会扭曲角度和长度，平面点积不反映曲面真实几何。

**正确做法**：用 Sharp/Flat 把度量 $g$ 一并带上，确保运算在曲面意义下正确。

## 十三、研究曲面的多种"语言"

### 1. 不同语言带来不同算法

数学家发明了多种描述曲面的"方言"：

| 语言 | 适用场景 |
| :--- | :--- |
| 局部高度函数 | 局部凹凸性分析 |
| 参数化 $f$ | 直观、常用 |
| 克里斯托费尔符号 | 协变导数、测地线、PDE |
| **外微分形式** | 坐标无关、易于离散化（本课主线） |
| 移动标架（Darboux 框架等） | 类比 Frenet 框架 |
| 黎曼曲面（复分析） | 共形、全局拓扑分析 |
| 四元数函数 | 全局映射，更丰富的代数结构 |

### 2. 学习多语言的价值

- 同一个曲率问题，用微分形式或复分析可能得到**完全不同的离散化算法**。
- 工具箱越大，对几何的理解越丰富，能解决的问题越多。

### 3. 本课主线

本课**仅深入讨论微分形式**。其他语言提供在线参考资料，作为延伸阅读。

下一步将把上述连续思想转化到**离散设置**——把微分形式落到三角网格的数据结构上（DEC：Discrete Exterior Calculus）。

## 十四、关键术语速查

| 术语 | 含义 |
| :--- | :--- |
| 浸入（Immersion） | $df$ 处处非退化；曲面良好的局部条件 |
| 法向量 $n$ | 与所有切向量正交的单位向量 |
| 高斯映射 $N: M \to S^2$ | 每点映为单位法向量；连续 ⟺ 可定向 |
| 不可定向曲面 | 莫比乌斯带、克莱因瓶——无全局连续高斯映射 |
| 高斯映射的满射性 | 闭光滑曲面对任意 $u \in S^2$ 总存在法线为 $u$ 的点（希尔伯特构造） |
| 高斯映射的单射性 | 当且仅当曲面严格凸（无平面区域） |
| 严格凸 | 凸且表面无平面区域 |
| 向量面积 $\vec{A}$ | $\int n\,dA$；补丁的"平均朝向" |
| 关键身份 | $\vec{n}\,dA = \tfrac{1}{2}\,df \wedge df$（向量值楔积 = 叉积） |
| 边界积分公式 | $\int_S \vec{n}\,dA = \tfrac{1}{2}\oint_{\partial S} f \times df$ |
| 闭曲面平均法线 | 恒为零（斯托克斯，$\partial = \emptyset$） |
| 多边形法线 | 边界积分定义，与内部三角化无关 |
| 投影面积解释 | 向量面积方向 = 边界投影面积最大的方向 |
| 面积形式 $dA$ | 测量曲面面积的 2-形式：$dA = \tfrac{1}{2}\,n\cdot(df\wedge df)$ |
| 复结构 $J$ | 切空间内 90° 旋转算子，$J^2 = -I$ |
| 曲面复结构 $J_f$ | 满足 $df(J_f X) = n \times df(X)$，唯一确定 |
| 共形浸入 | 等价于平面 90° 旋转 = $J_f$ |
| 霍奇星 $\star$ | 唯一携带几何信息的离散算子，由 $dA$ 与 $J$ 确定 |
| 0-形式霍奇星 | $\star\varphi = \varphi\,dA$ |
| 2-形式霍奇星 | $\star\omega = \omega/dA$（密度函数） |
| 1-形式霍奇星 | $(\star\alpha)(X) = \alpha(J_f X)$ |
| 三量统一 | $g(X,Y) = dA(X, JY)$；度量、面积、旋转互相决定 |
| Flat $X^\flat$ | 向量场 → 1-形式：$X^\flat(Y) = g(X,Y)$ |
| Sharp $\alpha^\sharp$ | 1-形式 → 向量场：$g(\alpha^\sharp, Y) = \alpha(Y)$ |
| 几何/组合分离 | $d, \wedge$ 是组合的；$\star$ 是几何的 |
| 变形梯度 | 工程领域对雅可比矩阵 $df$ 的另一名称 |
