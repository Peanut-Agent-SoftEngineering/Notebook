# CMU_DDG_p19

> 主题：离散曲面上曲率的"整体观（积分视角）"——从平滑高斯/平均/主曲率出发，通过积分、采样、Steiner 公式与 Gauss-Bonnet 定理，推导网格上的离散曲率公式。

---

## 一、概述与基本符号

### 1. 学习目标

在三角网格上量化并计算光滑曲面的多种曲率：**平均曲率 $H$、高斯曲率 $K$、主曲率 $\kappa_1, \kappa_2$**，并构建一个**统一的数学框架**把多种看似无关的离散公式有机整合。

### 2. 两堂课的分工

| 视角 | 方法 |
| :--- | :--- |
| **整体观（本节）** | 直接积分或采样光滑曲率，推导离散公式 |
| **变分观（下节）** | 把曲率视为某种几何量变化率的极限 |

### 3. 网格基本量与符号

| 符号 | 含义 |
| :--- | :--- |
| $f_i \in \mathbb{R}^3$ | 顶点位置 |
| $e_{ij} = f_j - f_i$ | 有向边向量 |
| $l_{ij} = \|e_{ij}\|$ | 边长 |
| $A_{ijk}$ | 三角形 $\triangle ijk$ 面积 |
| $N_{ijk}$ | 三角形单位法向量 |
| $\theta_i^{jk}$ | 顶点 $i$ 在三角形 $\triangle ijk$ 中的内角 |
| $\phi_{ij}$ | 与有向边 $ij$ 关联的二面角，方向反对称 |

#### 二面角的精确计算（atan2 而非 acos）

$$\phi_{ij} = \text{atan2}\!\left(\mathbf{e}_{ij} \cdot (N_{ijk}\times N_{jil}),\ N_{ijk}\cdot N_{jil}\right)$$

- 第一参数（点积与叉积）确定正弦符号——指示**弯曲方向**（凸/凹）。
- 第二参数（法线点积）确定余弦——夹角大小。
- **acos 的局限**：只返回 $[0,\pi]$，无法区分凸凹方向。**atan2** 返回 $[0,2\pi)$，明确区分。

#### 边向量是离散 1-形式
$\mathbf{e}_{ij}$ 反转方向时取反，与微分形式的积分性质 $\int_{-a}^b\omega = -\int_a^b\omega$ 完美对应——不是简单线段，而是**曲面上微元积分的离散表示**。

## 二、欧拉示性数与拓扑约束

### 1. 定义

$$\chi = V - E + F$$

仅依赖**拓扑结构**（孔洞数），与几何形状无关——折叠、抖动顶点都不会改变它。

### 2. 典型值

| 形状 | $\chi$ |
| :--- | :--- |
| 多边形圆盘 | 1 |
| 球体（含二十面体） | 2 |
| 环面（一个把手） | 0 |
| 双柄环面 | $-2$ |

### 3. 与亏格的关系

可定向闭曲面：
$$\chi = 2 - 2g$$

其中 $g$ 是把手数。

### 4. 工程意义

欧拉示性数是**鲁棒的拓扑指纹**。后面将看到：所有顶点角度缺陷之和恰好等于 $2\pi\chi$——曲率的全局总量被拓扑限制。

## 三、离散高斯曲率：从角度缺陷到 Gauss-Bonnet

### 1. 角度缺陷（Angle Defect）

#### 定义
$$\Omega_i = 2\pi - \sum_{\triangle\ni i}\theta_i^{jk}$$

#### 几何直觉

| 顶点类型 | $\sum\theta$ vs $2\pi$ | $\Omega_i$ | 对应 $K$ |
| :--- | :--- | :--- | :--- |
| 平面顶点 | $=2\pi$ | $0$ | 零曲率 |
| 凸顶点 | $<2\pi$ | $>0$ | 正曲率 |
| 鞍点 | $>2\pi$ | $<0$ | 负曲率 |

### 2. 推导一：测地球面积偏差

#### 平滑事实
- 曲面上半径 $\epsilon$ 测地球的面积：
  $$A_{\text{曲面}}(\epsilon) = \pi\epsilon^2\!\left(1 - \frac{K}{12}\epsilon^2 + O(\epsilon^3)\right)$$
- $K>0$（尖峰）面积偏小；$K<0$（鞍）偏大；$K=0$ 相等。

#### 离散类比
顶点星上的"球"被三角形切成楔形，第 $i$ 个楔形面积 $W_i = \tfrac{\theta_i}{2\pi}\cdot\pi\epsilon^2 = \tfrac{\epsilon^2}{2}\theta_i$。

#### 积分得到角度缺陷
$$\sum W_i = \tfrac{\epsilon^2}{2}\sum\theta_i \approx \pi\epsilon^2\!\left(1 - \tfrac{K}{12}\epsilon^2\right)$$

整理后：
$$2\pi - \sum\theta_i \approx \frac{K\pi}{6}\epsilon^2 \cdot \text{(归一化常数)}$$

**结论**：**角度缺陷 $\Omega_i$ 对应高斯曲率在顶点邻域内的积分**：
$$\Omega_i = \int_{\text{邻域}}K\,dA$$

#### 重要性质
- **尺度无关**：$\Omega_i$ 与 $\epsilon$ 选择无关，定义规范。
- **内蕴**：仅依赖角度（边长关系），无需法线等外蕴量。

### 2'. 推导二：高斯映射的球面面积

#### 角度互换
通过高斯映射把每个面法线映射到 $S^2$ 上，连接相邻面法线得到球面三角剖分。原始网格与球面网格之间发生**角度互换**：
- 原始**二面角** $\to$ 球面**内角**。
- 原始**内角** $\to$ 球面**二面角**。

#### 关键发现
顶点 $i$ 的角度缺陷 $\Omega_i$ **等于球面上对应多边形的面积**。
- 球面多边形面积 $= \sum(\text{内角}) - (n-2)\pi$，通过角度互换可证。

### 3. 全局总曲率（离散 Gauss-Bonnet）

#### 闭凸/球拓扑
$$\sum_i\Omega_i = 4\pi$$
（球面总面积，对二十面体每个顶点缺陷对应球面上一个小五边形，无重叠覆盖整球。）

#### 一般闭曲面
$$\sum_i\Omega_i = 2\pi\chi = 2\pi(2-2g)$$

#### 带边界
- 内部顶点：用 $\Omega_i = 2\pi - \sum\theta$（离散高斯曲率）。
- 边界顶点：用**离散测地曲率** $\kappa_i = \pi - \sum\alpha_{ij}$（外角），与平面多边形离散曲率定义一致。
- **离散 Gauss-Bonnet**：
  $$\sum_{\text{内部}}\Omega_i + \sum_{\text{边界}}\kappa_i = 2\pi\chi$$

#### 直觉
边界顶点没有高斯曲率——可被压扁到平面而无拉伸。它们贡献的是**边界曲线的弯曲程度**。

### 4. 离散 Gauss-Bonnet vs 光滑

| 光滑 | 离散 |
| :--- | :--- |
| $\int_M K\,dA + \int_{\partial M}\kappa_g\,ds = 2\pi\chi$ | $\sum\Omega_i + \sum\kappa_i = 2\pi\chi$ |

**核心意义**：**局部到全局定理**——只看局部邻域就能推断整体拓扑。

### 5. 没有免费的午餐

**问**：角度缺陷是离散高斯曲率的唯一合理选择吗？

**答**：否，但它是**唯一精确满足离散 Gauss-Bonnet** 的方法。

#### 替代：局部二次拟合
在顶点拟合二次曲面，计算其光滑高斯曲率。
- **缺点**：不满足 Gauss-Bonnet。

#### 角度缺陷的局限
- 不一定收敛到光滑高斯曲率值——除非顶点价数恒为 4 或 6。
- 选择哪种方法取决于应用最看重的性质：**全局拓扑正确性 vs 数值收敛**。

## 四、离散平均曲率与三大基本量

### 1. 三大基本微分恒等式

设曲面 $f$、单位法向 $n$，则：

| 楔积 | 结果 | 几何意义 |
| :--- | :--- | :--- |
| $df \wedge df$ | $2\,n\,dA$ | 面积元 |
| $df \wedge dn$ | $2H\,n\,dA$ | 平均曲率 |
| $dn \wedge dn$ | $2K\,n\,dA$ | 高斯曲率 |

#### 推导（基于主曲率）
- 主方向 $\mathbf{x}_1, \mathbf{x}_2$ 满足：$dn(\mathbf{x}_i) = \kappa_i\,df(\mathbf{x}_i)$。
- $df\wedge dn$ 展开：$df_{\mathbf{x}_1}\times\kappa_2\,df_{\mathbf{x}_2} - df_{\mathbf{x}_2}\times\kappa_1\,df_{\mathbf{x}_1} = (\kappa_1+\kappa_2)\,df\wedge df = 2H\cdot 2n\,dA$。
- 类似地 $dn\wedge dn = 2\kappa_1\kappa_2\,n\,dA = 2K\,n\,dA$。

**关键启示**：所有曲率信息都隐含在 $f$ 与 $n$ 的关系里，无需显式计算曲率张量。

### 2. 离散平均曲率法线（推导一：直接积分）

#### Stokes 定理转化
$$\int_C H\,n\,dA = \tfrac{1}{2}\int_C df\wedge dn = \tfrac{1}{2}\int_C d(n\wedge df) = \tfrac{1}{2}\oint_{\partial C}n\wedge df$$

（利用 $d(n\wedge df) = dn\wedge df + n\wedge d(df) = dn\wedge df$，因 $d(df)=0$。）

#### 离散化对偶胞元 $C$ 的边界
- 对偶胞元的边界由穿过原始边中点的折线组成。
- 沿对偶边 $e_{ij}^*$ 拆分为来自三角形 $A$（外心 $a$）和 $B$（外心 $b$）的两段。

#### 中点法则求和

$$\int_C H n\,dA = \sum_j\big(N(a)\times(m-a) + N(b)\times(b-m)\big)$$

#### 旋度技巧
- $N(a)\perp$ 三角形 $A$，故 $N(a)\times(m-a)$ 在三角形 $A$ 平面内且 $\perp(m-a)$。
- 两项最终都**平行于原始边向量 $e_{ij}$**。

#### 余切化简后的最终公式
$$\boxed{\mathbf{H}_i = \tfrac{1}{2}\sum_{j\in N(i)}(\cot\alpha_{ij}+\cot\beta_{ij})(\mathbf{x}_i - \mathbf{x}_j)}$$

其中 $\alpha_{ij}, \beta_{ij}$ 是边 $(i,j)$ 两侧三角形中**对角**的角。

### 3. 三种推导路径的一致性

离散平均曲率法线（余切公式）至少由三种独立途径得到：
1. 对偶边长 + 余切公式（几何路径，上节推导）。
2. **拉普拉斯-贝尔特拉米算子**：$\Delta f = 2H\,n$，离散化 $\Delta$ 自然得余切权重。
3. **面积变分**：表面积对顶点位置的梯度即得余切公式。

**结论**：余切权重不是任意选择，而是**离散微分几何中的规范刻画**——多重独立路径殊途同归。

## 五、离散高斯曲率法线

### 1. 与平均曲率的关键区别

- **平均曲率**：把顶点位置 $f$ 视为沿直边**线性插值**。
- **高斯曲率**：把法线 $n$ 视为在单位球面上沿**大圆弧插值**——因为 $n$ 自然把曲面映射到 $S^2$。

### 2. 推导

#### Stokes 转化
$$\int_C K\,n\,dA = \tfrac{1}{2}\oint_{\partial C}n\wedge dn$$

#### 沿弧的关键观察
- 对偶边界对应高斯映射下的弧 $\gamma_{ij}$，连接相邻法线 $N_{ijk}$ 与 $N_{jil}$。
- 在球面上沿 $\gamma_{ij}$ 的切线 $T$ 满足：$N_{ijk}, N_{jil}$ 都正交于公共边 $e_{ij}$，故 $n\times T$ 方向**平行于边向量** $e_{ij}$，且模长为 1。
- **弧长 = 二面角 $\phi_{ij}$**（法线之间的角度）。

#### 单边贡献
$$\int_{\gamma_{ij}}n\times T\,ds = \phi_{ij}\cdot\hat{e}_{ij} = \phi_{ij}\cdot\frac{f_j - f_i}{l_{ij}}$$

#### 最终公式
$$\boxed{\mathbf{K}_i = \sum_{j\in N(i)}\frac{\phi_{ij}}{l_{ij}}(f_j - f_i)}$$

### 3. 三大向量曲率公式对比

| 量 | 光滑形式 | 离散公式 | 权重 |
| :--- | :--- | :--- | :--- |
| 面积矢量 $\mathbf{A}$ | $\tfrac{1}{2}df\wedge df$ | $\sum f_i\times f_j$（顶点星周围位置叉积之和） | — |
| 平均曲率法线 $\mathbf{H}$ | $\tfrac{1}{2}df\wedge dn$ | $\sum w_{ij}(f_j-f_i)$ | $w_{ij} = \tfrac{1}{2}(\cot\alpha+\cot\beta)$ |
| 高斯曲率法线 $\mathbf{K}$ | $\tfrac{1}{2}dn\wedge dn$ | $\sum w_{ij}(f_j-f_i)$ | $w_{ij} = \phi_{ij}/l_{ij}$ |

平均曲率和高斯曲率的离散公式形式相同，只是权重从余切换成"二面角/边长"。

## 六、Steiner 公式视角：把多面体"软化"

### 1. 动机

经典微分几何在多面体上失效：
- **面内部**：$\nabla^2 f = 0$，曲率为零。
- **顶点/边**：导数无定义/趋向无穷大。

### 2. 核心思路：闵可夫斯基和

#### 闵可夫斯基和定义
$$A + B = \{a+b\mid a\in A, b\in B\}$$

直观上：用 $B$ 作"画笔"沿 $A$ 扫过的区域，或把 $B$ 的副本放在 $A$ 每个点上后取并集。

#### 平滑化操作
对原始多面体取闵可夫斯基和：$M_\epsilon = M + B_\epsilon$，其中 $B_\epsilon$ 是半径 $\epsilon$ 的球。

#### 局部效果
| 原始元素 | 平滑后 |
| :--- | :--- |
| 顶点 | 球面片段（半径 $\epsilon$ 的球） |
| 边 | 圆柱面片段（半径 $\epsilon$ 的圆柱） |
| 面 | 平行的平面板（沿法线推出 $\epsilon$） |

这些片段光滑粘合，给出一个真正的光滑曲面。再用经典微分几何在该曲面上算曲率，最后令 $\epsilon \to 0$。

### 3. 高斯曲率：仅顶点贡献

#### 各部分高斯曲率
| 部分 | $\kappa_1$ | $\kappa_2$ | $K=\kappa_1\kappa_2$ |
| :--- | :--- | :--- | :--- |
| 平面板 | 0 | 0 | 0 |
| 圆柱面 | $1/\epsilon$ | 0 | 0 |
| 球面帽 | $1/\epsilon$ | $1/\epsilon$ | $1/\epsilon^2$ |

#### 球面帽的总高斯曲率
- 球面帽对应单位球上的立体角 $\Omega_i$（恰是角度缺陷）。
- 半径 $\epsilon$ 球面上对应面积 $= \Omega_i\epsilon^2$。
- 总高斯曲率 $= (\Omega_i\epsilon^2)\cdot(1/\epsilon^2) = \Omega_i$。

**惊人结果**：与 $\epsilon$ 无关。整个网格总高斯曲率 $\sum\Omega_i = 2\pi\chi$。

### 4. 平均曲率：仅边贡献（在 $\epsilon\to 0$ 极限下）

| 部分 | $H = (\kappa_1+\kappa_2)/2$ | 面积 | 总平均曲率 |
| :--- | :--- | :--- | :--- |
| 平面板 | 0 | $A_0$ | 0 |
| 圆柱片段（边 $ij$） | $1/(2\epsilon)$ | $l_{ij}\phi_{ij}\epsilon$ | $\tfrac{1}{2}l_{ij}\phi_{ij}$ |
| 球面帽（顶点 $i$） | $1/\epsilon$ | $\Omega_i\epsilon^2$ | $\Omega_i\epsilon$ |

- **边贡献与 $\epsilon$ 无关**：$\tfrac{1}{2}l_{ij}\phi_{ij}$。
- **顶点贡献含 $\epsilon$**：$\epsilon\to 0$ 时归零。

**结论**：刚性离散网格中，**平均曲率全部集中在边上**：
$$\boxed{H_e = \tfrac{1}{2}l_{ij}\phi_{ij}}$$

### 5. 软化网格的面积与体积公式

#### 面积
$$A(\epsilon) = A_0 + \epsilon\sum_{\text{边}}l_{ij}\phi_{ij} + \epsilon^2\sum_{\text{顶点}}\Omega_i$$

#### 体积
$$V(\epsilon) = V_0 + \epsilon A_0 + \tfrac{1}{2}\epsilon^2\sum l_{ij}\phi_{ij} + \tfrac{1}{3}\epsilon^3\sum\Omega_i$$

### 6. Steiner 多项式的导数链

把闵可夫斯基和的体积写成 $\epsilon$ 的多项式 $V(\epsilon) = \sum_k V_k \epsilon^k$，称为 **Steiner 多项式**；其系数 $V_k$（即 **Quermassintegrals / 截面积分**）正是各阶曲率积分。这统一推广了如下导数链：

| 链 | 含义 |
| :--- | :--- |
| $dV/d\epsilon = A$ | 体积导数 → 面积 |
| $dA/d\epsilon = 2H_{\text{tot}}$ | 面积导数 → 两倍总平均曲率 |
| $dH_{\text{tot}}/d\epsilon = K_{\text{tot}}$ | 平均曲率导数 → 总高斯曲率 |
| $dK_{\text{tot}}/d\epsilon = 0$ | 高斯曲率导数 → 0 |

> 离散版本的高阶项系数对应**Schläfli 公式**：$\sum_e \ell_e\, d\theta_e = 0$（变形多面体时，所有边乘以二面角变化之和恒为零），它是连接离散与光滑情形的关键恒等式。

**为什么链终止在 $K$？** 因为总高斯曲率是**拓扑不变量**（Gauss-Bonnet）——形状变形不改变积分值。

### 7. 闵可夫斯基视角的统一价值

- **统一光滑与离散**：一种思想覆盖所有维度的曲率定义。
- **几何直观**：曲率不再是抽象张量，而是"加厚后面积变化的响应系数"。
- **可推广**：$n$ 维流形的离散曲率统一构造。

#### 局部面积扩展公式（用主曲率 $H, K$ 表达）
对沿法向移动距离 $t$ 的曲面 $f_t = f + t\,n$：
$$da_t = (1 + 2tH + t^2K)\,da_0$$

- 一次项 $2H$ 来自 $df\wedge dn$。
- 二次项 $K$ 来自 $dn\wedge dn$。

这进一步说明 **$H, K$ 是面积变化关于厚度 $t$ 的一阶/二阶响应系数**。

### 8. 离散曲线版本

对多边形折线做闵可夫斯基和（用半径 $t$ 圆盘）：
- 直边段：面积线性增加。
- 顶点：额外贡献等于**外角 $\kappa_i$**。

**结论**：离散曲线曲率仅在顶点处出现，等于外角——与之前的转角法定义完全一致。

## 七、推广到三维流形：标量曲率与黎曼曲率

### 1. 离散标量曲率（顶点处）

衡量四面体网格中围绕顶点 $i$ 的所有立体角与单位球总面积 $4\pi$ 的偏差：
$$S_i = 4\pi - \sum_{t\in T(i)}\Psi_t(i)$$

- $\Psi_t(i)$：第 $t$ 个四面体在顶点 $i$ 处的立体角。
- 平坦欧氏空间中所有立体角和恰为 $4\pi$。

### 2. 离散黎曼曲率（边处）

衡量围绕边 $ij$ 的所有四面体的二面角之和与 $2\pi$ 的偏差：
$$R_{ij} = 2\pi - \sum_{t\in T(ij)}\phi_t(ij)$$

平坦三维空间中围绕一条边的二面角之和为 $2\pi$。

### 3. 统一框架

把 $n$ 维流形嵌入 $n+1$ 维空间，对其做闵可夫斯基加厚 $V(\epsilon)$，泰勒展开的系数对应不同维度的离散曲率：
- 二维 → 角度缺陷、边长 + 二面角。
- 三维 → 标量曲率、黎曼曲率。
- 一般维度 → 内蕴曲率张量的离散对应。

## 八、关键术语速查

| 术语 | 含义 |
| :--- | :--- |
| 二面角 $\phi_{ij}$ | 共享边的两面法线之间的有向夹角 |
| atan2 | 区分凸/凹方向的反正切函数，二面角计算必需 |
| 欧拉示性数 $\chi$ | $V-E+F$，闭可定向曲面 $=2-2g$ |
| 角度缺陷 $\Omega_i$ | $2\pi-\sum\theta_i$，离散高斯曲率 |
