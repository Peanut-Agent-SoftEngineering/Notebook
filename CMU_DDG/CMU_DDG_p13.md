# CMU_DDG_p13

> 主题：曲线的微分几何——从平面光滑曲线到空间曲线的完整理论；转数、绕数、Whitney–Graustein 定理及其在六面体网格化与广义绕数中的应用

## 一、为什么研究低维几何

### 1. 应用动机

- **曲线**：道路网络、头发丝、签名笔迹、电缆、相机路径。
- **曲面**：三维物体的边界——固体形状的几何性质几乎都体现在边界曲面上。
- **三维体积**：物体内部结构。

几何计算大多发生在 1D/2D/3D，与日常接触的对象高度吻合。

### 2. 低维并非"简单"

低维流形里藏着数学中最难的问题之一，难度往往超过高维：

- **解结问题（Unknotting）**：给定 $\mathbb{R}^3$ 中的闭曲线，判断它是否能通过连续不自交不断裂的形变变成圆圈。该问题自上世纪二十年代提出至今，**计算复杂度仍未知**（在 NP 还是 EXP？）。
- **Willmore 猜想**：对给定亏格的曲面，使 **Willmore 能量**（衡量整体弯曲程度）最小的形状是什么？亏格 0 的最小者是单位球，**亏格 1 的最小者是 Clifford 环面**（近年才被证明），更高亏格仍未解。
- **Poincaré / 几何化猜想**：三维流形可由 **8 种"几何乐高"** 拼合构造。Poincaré 猜想是其推论，是七个**千禧年大奖难题**中**唯一已解决**的，由 Perelman 完成。

### 3. 学习低维的价值

1. **不是"婴儿数学"**：低维流形是现代几何学最深刻、最具挑战性的研究对象。
2. **高维直觉来源**：n 维空间中的许多结构与 2D/3D 行为差异不大，扎实掌握低维能为高维提供直觉。
3. **警惕术语滥用**：在数据分析/机器学习中"流形"一词常被使用，但通常**缺乏**纤维丛、曲率流、辛结构等真正几何深度。
4. **"规范代表"思想**：对一类拓扑空间，寻求"最光滑/最低能量"的几何代表（如球面是亏格 0 的 Willmore 极小元）是几何处理与离散微分几何的一条核心思路——它让两个空间之间的映射、计算变得自然。

## 二、描述曲面/曲线的多种"语言"

### 1. 光滑设定下的方言

| 语言 | 描述方式 | 备注 |
| :--- | :--- | :--- |
| 高度函数与切平面 | 局部参数化为切平面上的高度函数 | 经典方法 |
| Christoffel 符号 | 描述内蕴几何变化的符号系统 | 广义相对论常用 |
| **微分形式** | **无坐标**地处理积分、曲率、拓扑 | **本课程核心** |
| 活动标架（Moving Frames） | 用适应曲面的标架沿空间变化描述 | 与微分形式紧密相关 |
| 复分析 / 四元数 | 黎曼曲面用复变；某些特殊曲面用四元数 | 专门化工具 |

最佳策略：**掌握多种语言**，以理解不同领域文献。

### 2. 离散设定下的两条主线

- **隐式表示**：曲面 = $F(x,y,z)=0$ 的零集。例：球 $x^2+y^2+z^2-r^2=0$。
  - **优点**：拓扑可变（流体的分裂/合并）；内外测试只看符号。
  - **缺点**：网格采样存储昂贵；自适应分辨率难；曲面上方程求解复杂。

- **显式表示**：直接列出点和元素，典型如多边形网格。
  - **优点**：渲染、细分、顶点属性存储都简单；空间自适应容易；存储成本低。
  - **缺点**：拓扑改变（合并/分裂网格）困难。

### 3. 不要陷入教条

没有"更好"的表示，**为任务选合适的工具**。明确算法目标和关心的物理量再做选择。课程的最终目的是把光滑视角（微分形式）与离散视角（抽象单纯复形 + 离散微分形式）统一起来——这样就能在两者间自由切换，把古老的几何定理直接搬进计算机。

## 三、平面光滑曲线的两种描述

### 1. 隐式描述（代数几何）

- 曲线 = 满足 $f(x,y)=0$ 的点集。
- 例：单位圆 $x^2+y^2-1=0$，函数 $f$ 同时给出符号距离场。
- **局限**：处理自交点比较棘手。

### 2. 参数化描述（微分几何）

- 映射 $\gamma:[0,L]\to\mathbb{R}^2$，其图像就是曲线。
- 例：单位圆 $\gamma(s)=(\cos s,\sin s),\ s\in[0,2\pi]$，是**闭曲线**。
- **优点**：天然处理自相交；编码"速度/方向"信息；切线、法线、曲率计算直接。

### 3. 嵌入 vs 浸没

| 类型 | 是否允许自交 | 拓扑 |
| :--- | :--- | :--- |
| **嵌入（Embedded）** | 否；连续双射，逆映射也连续 | 保持原拓扑（如开曲线两端点不会被粘合） |
| **浸没（Immersed）** | 允许全局自交 | 仅要求局部良好（处处可微，导数非零） |

**反例（非嵌入）**：半开区间 $[0,2\pi)$ 通过 $s\mapsto(\cos s,\sin s)$ 映到单位圆——双射但逆不连续，将"开"映成"闭"，破坏了拓扑。

## 四、微分（differential）vs 切线（tangent）

这是最容易混淆的核心概念。

### 1. 区别

- **微分** $d\gamma$ 是一个**映射**：把参数域上的任意向量"向前推"到曲线所在空间中。它编码**方向 + 拉伸幅度**。
- **单位切线** $T(s)$ 是曲线某一点上的一个**特定单位向量**。

公式：
$$T(s)=\frac{d\gamma\!\left(\partial/\partial s\right)}{\|d\gamma(\partial/\partial s)\|}=\frac{\gamma'(s)}{\|\gamma'(s)\|}$$

即：**切线 = 归一化(微分(域基向量))**。微分是函数，切线是值；说"微分 = 未归一化的切线"是不严谨的。

### 2. 弧长参数化的简化效果

若 $\|\gamma'(s)\|=1$（**等距映射**），则微分直接输出单位向量，无需归一化：
$$T(s)=\gamma'(s)$$

参数 $s$ 即弧长。

### 3. 实例：单位圆

$$\gamma(s)=(\cos s,\sin s),\quad d\gamma=(-\sin s\,ds,\cos s\,ds)$$
$$d\gamma(\partial/\partial s)=(-\sin s,\cos s),\quad \|\cdot\|=1$$

故圆已是弧长参数化，$T(s)=(-\sin s,\cos s)$。

### 4. 微分编码"拉伸"

参数区间到三维空间的映射相当于经历两次拉伸：参数化本身的变化 + 空间映射的尺度变化。单位切线只告诉方向，微分还告诉拉伸量。**$d\gamma=0$（速度为零）是灾难性的奇异情况**。

## 五、正则性的层级

可微只是基础条件，曲线还需要更强的正则性才能定义切线、法线、曲率。

### 1. 三个层级

| 概念 | 关键条件 | 几何意义 |
| :--- | :--- | :--- |
| **可微曲线** | 导数存在（可能为零） | 可能无切线 |
| **正则曲线（浸没）** | $\gamma'(t)\neq 0$ 对所有 $t$ | 切线、曲率可定义；允许自交 |
| **嵌入曲线** | 连续双射且逆连续 | 不自交，保持拓扑 |

口诀：**可微不够，正则才行；嵌入更强，拓扑不变。**

### 2. 反例：可微但几何上有"尖角"

**原始曲线**：
$$\gamma(s)=\begin{cases}(s,s)&s\le 0\\(s,-s)&s>0\end{cases}$$
有明显尖角，显然不可微。

**重新参数化** $\eta(s)=s^3$：
$$\tilde\gamma(s)=\begin{cases}(s^3,s^3)&s\le 0\\(s^3,-s^3)&s>0\end{cases}$$

惊人结论：$\tilde\gamma$ **处处可微**（$x(s)=s^3$ 光滑，$y(s)$ 也是 $s^3$ 的光滑分段函数），但 $\tilde\gamma'(0)=(0,0)$。曲线在 $s=0$ "速度为零"，切向量长度为 $0$、方向未定义——这就是为什么尽管可微，几何上仍有尖点。

## 六、重新参数化

- **定义**：通过区间到自身的映射 $a:[0,L]\to[0,L]$ 复合得新曲线 $\tilde\gamma=\gamma\circ a$。
- **几何**：图像（点集）不变，但沿曲线的**速度变了**——原本均匀分布的参数点变得疏密不均。
- **类比**：开车走同一条路，原来匀速，现在时快时慢。

### 几何信息 vs 参数信息

- 曲线的几何形状由图像决定。
- 参数化额外编码速度等信息。
- **关键洞察**：不能直接比较两个映射判断它们是否描述相同形状——这正是比较两个网格（曲面参数化）何时表示同一曲面如此困难的根本原因。

### 微分的复合规则

$$d\tilde\gamma=d\gamma\circ da$$

速度向量按 $da/ds$ 缩放并保持方向。

## 七、法向量

- 平面曲线的**单位法线** $N$ = 单位切线 $T$ **逆时针旋转 90°**。
- 用算子 $J$（90° 旋转矩阵）表示：$N(s)=J\,T(s)$。
- 二维操作：$(T_x,T_y)\mapsto(-T_y,T_x)$。
- 正交性：$T\cdot N=T_x\cdot(-T_y)+T_y\cdot T_x=0$。
- **方向约定**：内/外侧都合法，但需在整个曲线上保持一致。

### 实例：圆

$$\gamma(s)=(\cos s,\sin s),\ T(s)=(-\sin s,\cos s)\ \Rightarrow\ N(s)=(-\cos s,-\sin s)$$

法线指向圆心，与几何直觉一致。

---

## 八、曲率：三种等价定义

### 1. 弧长参数化曲线的变化率

$$\kappa(s)=N(s)\cdot\frac{dT}{ds}(s)=N(s)\cdot\gamma''(s)$$

（点积保留**符号**——左转或右转。）

### 2. 角度变化率

设切线与水平方向夹角为 $\theta(s)$：
$$\kappa(s)=\frac{d\theta}{ds}$$

直观：单位速度行驶时方向盘转动的"角速度"。

### 3. 密切圆（osculating circle）

$$\kappa(p)=\frac{1}{R(p)}$$

- **构造**：取曲线上 $p$ 附近的两点，三点定圆；让两点趋于 $p$，极限圆即密切圆。
- **直觉对证**：直线 $R\to\infty$，$\kappa=0$；急弯 $R$ 小，$\kappa$ 大。

---

## 九、平面曲线基本定理

> **弧长参数化的平面曲线，由其曲率函数 $\kappa(s)$ 唯一确定（差一个刚体运动）。**

### 1. 直觉

以单位速度行驶，$\kappa(s)$ 告诉你每时刻方向盘转多快（左正右负）。给定 $\kappa(s)$，再加**起点**和**初始朝向**，就能重现整条路径——就像微积分基本定理用变化率恢复函数。

### 2. 从曲率恢复曲线（两步积分）

**步骤 1：曲率 → 切线方向**
$$\theta(s)=\theta_0+\int_0^s\kappa(\tau)\,d\tau$$

**步骤 2：切线 → 单位切向量**
$$T(s)=(\cos\theta(s),\sin\theta(s))$$

**步骤 3：切向量 → 位置**
$$\gamma(s)=\gamma_0+\int_0^s T(\tau)\,d\tau$$

两个积分常数：
- $\theta_0$：初始角，对应**旋转**自由度。
- $\gamma_0$：起点，对应**平移**自由度。

这就是"差一个刚体运动"的来源：曲率本身**坐标不变**，描述内蕴形状，不含位置/朝向信息。

### 3. 实例：$\kappa(s)\equiv 1$ 给出圆

$$\theta(s)=s,\ T(s)=(\cos s,\sin s),\ \gamma(s)=(\sin s,1-\cos s)$$

圆心 $(0,1)$，半径 $1$。

### 4. 关键限制与启示

- **必须弧长参数化**：否则相同的 $\kappa(s)$ 会因速度变化生成不同空间路径。一般情形下要靠**曲率 + 参数化共同确定**。
- **解析积分通常无解**：除常数曲率（圆/螺旋线）等少数情况，闭式解不存在。
- **离散版本却极简**：从离散曲率重建离散曲线非常容易，是后续算法的基础。

## 十、整体不变量：转数与绕数

平面闭曲线有两个看似相似但本质不同的整数。

### 1. 转数（Turning Number）$k$

- **定义**：沿曲线走一周时，**单位切向量**绕原点的累计圈数。
- 等价：从曲线到单位圆的切线映射 $T:S^1\to S^1$ 的**拓扑度**。
- **公式**：
$$k=\frac{1}{2\pi}\int_\gamma\kappa\,ds$$
- **取值**：$k=\pm 1,\pm 2,\dots$；**也允许 $k=0$**（先顺时针、后逆时针，净旋转互相抵消）。
- 凸闭曲线（圆）：$k=+1$；"8"字形：$k=0$。

### 2. 绕数（Winding Number）$n$

- **定义**：曲线绕**外部某点 $P$** 的圈数。
- 数学定义：考虑映射 $\hat\gamma_P(s)=\frac{\gamma(s)-P}{\|\gamma(s)-P\|}:S^1\to S^1$ 的拓扑度。

| 绕数 $n$ | 行为 |
| :---: | :--- |
| 0 | 不缠绕 $P$（可推到无穷远） |
| +1 | 逆时针一圈 |
| -3 | 顺时针三圈 |

### 3. 两者对比

| 概念 | 观测对象 | 参考点 | 物理意义 |
| :--- | :--- | :--- | :--- |
| **转数** | 切向量 | 曲线自身 | 切线方向总变化圈数（内蕴） |
| **绕数** | 位置向量 | 外部点 | 曲线缠绕外部点的圈数 |

**联系**：转数 = 切向量曲线 $T(s)$ 自身相对原点的**绕数**。两者都是"拓扑度"的不同体现。

## 十一、Whitney–Graustein 定理与同伦

### 1. 正则同伦（Regular Homotopy）

两条闭曲线 $\gamma_0,\gamma_1$ 是**正则同伦**的，若存在连续族 $\gamma_t$ 把它们连起来，且**整个变形过程中 $\gamma_t'(s)$ 始终非零**（不出现尖点）。

### 2. 平面版本（Whitney–Graustein）

> 两条闭平面曲线正则同伦 $\Leftrightarrow$ 它们有相同的转数 $k$。

- **必要性**：连续不产生尖点的形变无法让切线突然多/少转一圈。
- **充分性**：相同转数足以保证可互相连续形变。
- **推论**：所有平面闭曲线的正则同伦类由整数 $k$ 标记。例如"8"字（$k=0$）**不能**正则同伦地变成圆（$k=1$）。

### 3. 经典结果：圆不能"翻面"

正向圆（$k=+1$）和反向圆（$k=-1$）正则同伦类不同。任何尝试把圆从内向外翻的连续过程，总会在某一时刻出现切线跳跃 / 尖点。Nelson Max 1972 年的电影是这一定理的可视化经典。

### 4. 球面版本：同伦类只有两类

球面允许"绕到背面"，比平面少了限制：

- **偶数转数**：可正则同伦到一个**"8"字形**。
- **奇数转数**：可正则同伦到一个**小圆圈**。

### 5. 应用：六面体网格化

把一个三维体积切成"拓扑立方体"（8 顶点、6 面）小块。

**Thurston–Mitchell 定理**（边界判据）：
> 一个亏格 0 的四边形网格是某个六面体网格的边界 $\Leftrightarrow$ 它有偶数个面。

- **必要性**：每个六面体贡献 6 个面（偶数），相邻粘合移除 2 个，总数始终为偶数。
- **充分性的证明思路（Erickson 等的存在性证明）**：
  1. 在四边形网格表面构造**对偶曲线网络**（每个四边形中心连接相邻中心，最终得一组闭曲线）。
  2. 用球面 Whitney–Graustein 定理把曲线网络通过**正则同伦**简化为一组分离的"8"字与小圆圈。
  3. 沿同伦路径**扫掠（sweep）**，每一时刻的曲线网络对应一个新四边形网格；扫出的内部曲面对应六面体的面。
  4. 拼合得到六面体网格。
- **复杂度**：最坏 $O(n^2)$ 个六面体（仅证存在性，不实用）。后续工作把界数降至 $O(n)$，并推广到高亏格——此时除偶数面外，还需满足与非平凡循环相关的同伦条件。

## 十二、广义绕数：从离散问题到连续问题

### 1. 动机

现实数据（扫描网格）常**有裂缝、孔洞**，曲线/曲面不闭合。但"点在物体内还是外"的判断不应被这些缺陷彻底破坏。

### 2. 思想

绕数本质上是"曲线投影长度（或曲面投影面积）的积分"，**这个积分定义不要求闭合**，所以可以扩展到含孔曲面。

- **二维**：对于线段 $\overline{AB}$ 与点 $P$ 的贡献 = 从 $P$ 看线段所张的**有符号角**。
  - 设 $\mathbf{u}=A-P,\ \mathbf{v}=B-P$：
$$\theta=\mathrm{atan2}(\mathbf{u}\times\mathbf{v},\ \mathbf{u}\cdot\mathbf{v})$$
  - 必须用 **`atan2`**（双参数），单参数 `atan` 有象限歧义。

- **三维**：对每个三角形 $t$，计算它在以 $P$ 为中心的单位球上的**有符号投影面积**，求和：
$$N(p)=\frac{1}{4\pi}\sum_t \text{signed projected area}(t\text{ on unit sphere around }p)$$
  - 闭合网格：结果为整数（1=内，0=外）。
  - 含缺陷网格：结果为**连续数值**，光滑反映"内外程度"。

### 3. 应用

- **防水网格提取**：对整个空间计算广义绕数场，按阈值（如 0.5）取等值面，直接得到无裂缝的封闭网格。
- **布尔运算**：对两个曲面的广义绕数取交/并，对带裂缝表面也能稳定工作。
- **核心思维**：从"是/否"的离散判断转化为"值多大"的连续函数，再阈值化——天然鲁棒。

## 十三、空间曲线

### 1. 定义与微分

参数化空间曲线 $\gamma:[a,b]\subset\mathbb{R}\to\mathbb{R}^3$；要求连续可微（光滑）。

- **微分**：$d\gamma_s(\mathbf{x})=\mathbf{x}\cdot\gamma'(s)$，是参数域向量 $\mathbf{x}$ 与切向量 $\gamma'(s)$ 的乘积——本质是**雅可比矩阵**。
- **向前推进**（Pushforward）：把参数域的向量场 $X=a(s)\,\partial_s$ 映到 $\mathbb{R}^3$ 中：
$$\gamma_*(X)=a(s)\left(\frac{\partial x}{\partial s},\frac{\partial y}{\partial s},\frac{\partial z}{\partial s}\right)$$

### 2. 平面曲线 → 空间曲线：多了一种弯曲方式

- **曲率 $\kappa$**：曲线在**当前平面内弯曲**的快慢。
- **挠率 $\tau$**：曲线**离开当前平面**、在三维空间中扭曲的快慢。

直观例子：

- 玩具刚体由若干半圆弧拼接，每段弧的曲率恒定、挠率为 0；但相邻两段弧在交界处发生扭出平面的旋转——这就是挠率。
- 螺旋线水平转圈（曲率）+ 向上爬升（挠率），上升越快挠率越大。

## 十四、Frenet 框架与 Frenet–Serret 公式

### 1. Frenet 框架

任何光滑空间曲线在每点都有一个**自然的局部正交标架**，由曲线本身决定：

- **单位切线** $T=d\gamma/ds$（弧长参数化）。
- **单位法线** $N$：由 $dT/ds=\kappa N$ 得到。
- **副法线** $B=T\times N$。

三向量正交，构成右手坐标系。

### 2. Frenet–Serret 公式

$$\frac{d}{ds}\begin{bmatrix}T\\ N\\ B\end{bmatrix}=\begin{bmatrix}0&\kappa&0\\ -\kappa&0&\tau\\ 0&-\tau&0\end{bmatrix}\begin{bmatrix}T\\ N\\ B\end{bmatrix}$$

矩阵中两个非零参数：

- $\kappa$ 控制 $T$ 转向 $N$ 的速度（**平面内弯曲**）。
- $\tau$ 控制 $B$（也即标架整体）绕 $T$ 旋转的速度（**扭出平面**）。

由公式可读出：
$$\kappa=\left\|\frac{dT}{ds}\right\|,\qquad \tau=\left\|\frac{dB}{ds}\right\|$$

### 3. 实例：螺旋线 $\gamma(s)=(a\cos s,a\sin s,bs)$

设 $a^2+b^2=1$（单位速度）。

| 量 | 表达式 |
| :--- | :--- |
| $T(s)$ | $(-a\sin s,a\cos s,b)$ |
| $dT/ds$ | $(-a\cos s,-a\sin s,0)$，模长 $a$ |
| $\kappa$ | $a$ |
| $N(s)$ | $(-\cos s,-\sin s,0)$ |
| $B=T\times N$ | $(-b\sin s,b\cos s,-a)$ |
| $dB/ds$ | $(-b\cos s,-b\sin s,0)=bN$ |
| $\tau$ | $-b$ |

曲率与挠率均为常数，符合螺旋线"等弯等扭"的几何直觉。

## 十五、空间曲线基本定理

> **弧长参数化的空间曲线，由 $\kappa(s)$ 和 $\tau(s)$ 共同唯一确定（差一个刚体运动）。**

### 1. 二维 vs 三维

| 维度 | 控制量 | 重建方式 |
| :--- | :--- | :--- |
| 平面 | $\kappa(s)$ | 两步独立积分 |
| 空间 | $\kappa(s),\tau(s)$ | 解耦合 ODE 组（Frenet–Serret） |

### 2. 求解方式

- **解析解**：仅常数曲率/挠率（直线、圆、螺旋）等少数情形可行。
- **数值解**：一般情形必须数值积分 ODE：
$$\frac{dT}{ds}=\kappa N,\quad \frac{dN}{ds}=-\kappa T+\tau B,\quad \frac{dB}{ds}=-\tau N$$

### 3. 维度上升原则

随空间维度增加，描述曲线所需"曲率类"的量也增加。这一规律将延伸到曲面情形（曲面是否可由其曲率信息恢复？后续讨论）。

## 十六、Frenet vs Bishop 框架

### 1. Frenet 的致命缺陷

在**直线段**（$\kappa=0$）上，$dT/ds=0$，导致 $N,B$ 无法唯一确定（出现零向量歧义）。

### 2. Bishop 框架（主教架）

一种**全局扭曲最小**的正交标架，不依赖局部 $dT/ds$，而是沿曲线"最少旋转"地传输坐标系。

- **全局定义**：依赖整条曲线历史，不能局部一次求得；要获得某点的 Bishop 标架必须从起点积分。
- **优点**：在整条曲线（含直线段）上光滑、无突然扭动。

### 3. 术语澄清

- **适配标架**（Adapted Frame）：任何含切线 $T$ 的正交标架的统称。
- **Frenet** 是局部（由 $\kappa,\tau$ 直接给出）；**Bishop** 是全局（积分得到）。

### 4. 选择策略

| 框架 | 定义方式 | 优点 | 缺点 | 适用 |
| :--- | :--- | :--- | :--- | :--- |
| **Frenet** | 局部微分 | 计算简单 | 直线段失效，可能扭动 | 简单曲线、快速局部计算 |
| **Bishop** | 全局积分 | 全曲线光滑、最小旋转 | 需积分，无法瞬时获取 | 管状曲面、相机路径、避免突然扭动 |

> **一句话**：Frenet 是局部的"微分驾驶"，拐弯好用、直道失灵；Bishop 是全局的"积分驾驶"，麻烦但一路平稳。
