## 一、课程定位与核心立场

### 1.1 这是怎样一门课？

- **不是一门"高级课"**：它是**"第一堂真正的几何学课程"**——重新训练你**思考几何**的方式：如何**表示**形状、如何**谈论**形状、如何**操作**形状。
- **不是定理证明课**：教学目标不是堆砌证明，也不是堆砌算法，而是培养一种"**形感（sense of shape）**"。
- **不是抽象先行**：传统微分几何的路线是"拓扑空间 → 图册 → 流形"，门槛极高。本课走**直觉优先**路线，把曲线/曲面始终当作**$\mathbb{R}^2/\mathbb{R}^3$ 中的浸入（immersion）**——这更贴近自然思考、也贴近真实计算。
- **数学与计算双向贯通**：
  - 离散视角能反过来帮你**理清光滑微分几何**中那些抽象到令人绝望的概念；
  - 光滑视角能改变你**写算法的方式**，让数值方法不再是凑出来的，而是从几何第一性原理推出来的。

### 1.2 课程的"元方法论"——这是整门课最关键的一句话

> **DDG 的核心工作流不是"近似"，而是"翻译 → 比较 → 筛选"。**

把光滑几何中的某个量（曲率、Laplacian、测地线、…）翻译到网格上，分三步：

1. **列出所有光滑等价定义**（密切圆、二阶导、变分、Steiner 公式、转角、…）。
2. **逐字翻译到离散对象**（折线、三角网格）。
3. **品鉴**：原本等价的几个公式，离散化后**不再等价**——必须根据应用场景选保留哪种性质。

这条方法论由一条更深的定理支撑：

> **"No Free Lunch" 原理**：在合理条件下，**不存在一个离散定义能同时保留全部光滑性质**。每个定义只能保住一部分。

整门课的所有"具体内容"——离散曲率、离散 Laplacian、离散测地线——都是这条原理的具体演绎。**记住它，比记住任何公式都重要**。

## 二、整体路线图（22 讲的内在逻辑）

22 节课不是松散的专题集合，而是**一条单调上升的螺旋**：

```
第一螺旋（最简单的几何对象——平面曲线）
  p01-p02  课程导论 + "光滑→离散"翻译方法论（用平面曲线曲率作 toy case）

第二螺旋（搭建网格的"骨架"——拓扑数据结构）
  p03      单纯形、单纯复形、定向、链接、TDA 应用
  p04      流形条件、邻接表/关联矩阵/半边数据结构

第三螺旋（搭建网格上的"语言"——外微积分）
  p05      外代数（楔积、k-向量、Hodge 星、坐标）
  p06      对偶空间与 k-形式（用余向量楔积出"测量器"）
  p07      微分 k-形式（k-形式的"场化"）
  p08      外微分 d（统一 grad/curl/div + d²=0）
  p09      积分、Stokes 定理、形式之间的内积
  p10      把外微积分翻译到网格上（链/上链 + Whitney 基）
  p11      DEC 完整框架（关联矩阵 + 对偶网格 + Hodge 星 + cotan 公式）
  p12      向量值微分形式（为讨论位置/法线作准备）

第四螺旋（曲线——最简单的几何对象，重做一遍但完整）
  p13      光滑曲线全景（参数化、转数、绕数、Whitney–Graustein、Frenet–Serret）
  p14      离散曲线（离散基本定理、缩短流、弹性流、桥本流、排斥能量）

第五螺旋（曲面——核心目标）
  p15      参数化曲面、浸入/嵌入、第一基本形式、共形参数化、抽象黎曼流形
  p16      法向量、高斯映射、向量面积、面积形式、复结构、Hodge 星在曲面上的实现
  p17      离散曲面（局部单射判据、对偶胞元法线、cotan Laplace、形状重建）
  p18      曲面曲率全景（法曲率、主曲率、高斯/平均曲率、第二基本形式、Gauss–Bonnet）
  p19      离散曲率·整体观（角度缺陷、cotan H、二面角 K、Steiner 公式）
  p20      离散曲率·变分观（求导链 V→A→H→K→0、Schläfli 公式、曲率流、Willmore）

第六螺旋（曲面上的"动力学"）
  p21      Laplacian / Laplace–Beltrami
  p22      测地线（最短 vs 最直、cut locus、medial axis、Exp/Log、Christoffel 符号）
```

每一螺旋都重复了同一过程——**光滑视角 → 翻译方法论 → 离散结构 → DEC 矩阵化 → 应用算法**。理解这层结构，就理解了课程的"骨架"。

## 三、贯穿全课程的五条主轴

把 22 讲压缩成 5 条贯穿始终的线索，比逐章复述更有用。

### 主轴 1：浸入而非嵌入

| 概念 | 局部 / 全局 | 允许什么 |
| :--- | :--- | :--- |
| **浸入（Immersion）** | 局部条件：$df$ 处处非退化 | 允许全局自交（如 8 字、球面外翻） |
| **嵌入（Embedding）** | 全局条件：单射 + 逆连续 + 保拓扑 | 不允许自交、不允许粘合 |

**全课程几乎只在浸入框架下工作**，原因：

- 离散网格做随机扰动几乎总能保持"逐三角形非退化"（浸入），但随机扰动经常造成自交（破坏嵌入）。
- 浸入集合更像向量空间（两个浸入相加大概率仍是浸入）。
- 球面外翻、Whitney–Graustein 这类只能在浸入框架下讨论。

### 主轴 2：微分形式作为坐标无关的几何语言

外微积分是这门课的**官方语言**。它把：

- **梯度**统一为 $d$（0-形式 → 1-形式）
- **旋度**统一为 $d$（1-形式 → 2-形式）
- **散度**统一为 $\star d \star$（…）
- **Laplacian** = $\delta d + d \delta = \star d \star d + \cdots$

打包到一个**与度量无关 / 与坐标无关**的框架里。重要分工：

| 算子 | 性质 | 离散对应 |
| :--- | :--- | :--- |
| $d$（外微分） | **拓扑**——不依赖度量 | **关联矩阵**（仅依赖网格连接性） |
| $\wedge$（楔积） | **代数** | 关联矩阵 + 符号约定 |
| $\int$（积分） | **由 De Rham 映射预积分** | 直接对单纯形上的值求和 |
| $\star$（Hodge 星） | **几何**——承载所有度量信息 | **对角矩阵**（cotan 公式） |

**这种"拓扑/几何"清晰分离是 DEC 最优雅的工程价值**：改变网格几何，只重算 $\star$；改变拓扑（远少见），才需要重建 $d$。

### 主轴 3：DEC（离散外微积分）= 关联矩阵 + 对角 Hodge 星

整个课程的"代码骨架"压缩成一句话：

```
经典向量微积分 = 欧氏空间的特例
外微分          = 任意流形的统一框架
DEC             = 它在网格上的精确离散映射
```

**实操工作流**（任何离散几何处理算法都套这个模板）：

```
1. 加载网格（顶点 + 三角形）
2. 构建基础稀疏矩阵
   - 关联矩阵 D₀ (E×V), D₁ (F×E)：仅依赖拓扑
   - 对角 Hodge 星 ⋆₀, ⋆₁, ⋆₂：依赖几何（cotan 公式）
3. 组合得到目标算子
   ▸ 梯度  = D₀
   ▸ 旋度  = ⋆₂ D₁
   ▸ 散度  = ⋆₀⁻¹ D₀ᵀ ⋆₁
   ▸ 余切 Laplace = ⋆₀⁻¹ D₀ᵀ ⋆₁ D₀
4. 求解线性系统 / 优化问题
5. 处理 / 可视化结果
```

**核心精确性**：在离散层面 $d^2 = 0$、Stokes 定理、$\partial^2 = 0$ 都**精确**成立，而不是近似——这是 DEC 比有限差分/有限元更"几何上诚实"的根源。

### 主轴 4：标量-向量曲率链与几何求导

曲率章节（p18-p20）的核心发现：

```
标量量            梯度（向量曲率法线）
─────────────────────────────
封闭体积 V    ──▶  面积矢量 A      = Σ ⅓(f_j × f_k)
总表面积 A    ──▶  平均曲率 H      = ½ Σ (cotα + cotβ)(f_i − f_j)
总平均曲率 ΣH ──▶  高斯曲率 K      = Σ (φ_ij / l_ij)(f_i − f_j)
总高斯曲率 ΣΩ ──▶  0  （拓扑不变量）
```

这条链同时具备四个特性：

1. **完美对应光滑链** $V \to A \to 2H \to K \to 0$（Steiner 公式）；
2. **由两个支撑定理确保**：离散 Gauss–Bonnet（$\sum \Omega_i = 2\pi\chi$）+ Schläfli 公式（$\sum l_{ij} \delta\phi_{ij} = 0$）；
3. **三种独立路径殊途同归**（积分对偶胞元 / 求面积梯度 / Steiner 软化），证明 cotan 公式是**唯一规范**离散化；
4. **直接落地为曲率流算法**——平均曲率流（梯度下降表面积）、Willmore 流（梯度下降 $\int H^2$）、桥本流（涡丝动力学）。

### 主轴 5：内蕴优先

课程后半段越来越强调**内蕴几何**：

- **第一基本形式**（黎曼度量 $g$，由边长决定）足以算长度、角度、面积、Laplace–Beltrami、测地线、高斯曲率（绝妙定理）。
- **第二基本形式**（由二面角决定）才需要外蕴信息。
- **离散基本定理**：边长 + 二面角 → 唯一确定网格（差刚体运动）。

工程含义：

- 不要把网格表示为"一堆顶点"——更深层的表示是"边长 + 二面角"或"曲率场"。
- 等距变形（保边长的弯折）下所有内蕴量不变——是形状对应、姿态匹配的核心利器。
- 双曲度量这类在 $\mathbb{R}^3$ 中无法等距嵌入的几何（Hilbert 定理），只能用内蕴语言处理。

## 四、各讲精炼摘要（含核心结论与公式）

### 第 1-2 讲：导论 + 平面曲线曲率作为方法论范例

- **课程机制**：录像授课 + Zoom 答疑；JS / C++ 双轨编码作业；柏拉图立体迟到记号。
- **核心方法论**（关键！）：把"曲率"翻译到折线，用四条等价光滑定义出发：
  - 路径 A（转角）：$\kappa_i = \theta_i$
  - 路径 B（长度变分）：$\kappa_i = 2\sin(\theta_i/2)$
  - 路径 C（Steiner 直线延伸）：$\kappa_i = 2\tan(\theta_i/2)$
  - 路径 D（密切圆）：$\kappa_i = 2\sin\theta_i / |\gamma_{i+1}-\gamma_{i-1}|$
- **No Free Lunch 表**：四种定义无法同时满足"总曲率守恒、质心不漂移、保圆稳态"全部三条。
- **应用**：曲线缩短流 $\partial\gamma/\partial t = \kappa N$。

### 第 3-4 讲：单纯复形 + 网格数据结构

- **单纯形 = 仿射独立点的凸包**；**单纯复形 = 满足"交的封闭性 + 面的完整性"的集合**。
- **抽象单纯复形**：纯组合、不嵌入、是拓扑空间的离散模拟。
- **解剖三件套**：闭包 Closure / 星 Star / 链环 Link。
- **流形条件（核心判据）**：$k$ 维单纯复形是流形 ⇔ 每顶点的 Link 同胚于 $\mathbb{S}^{k-1}$。三角网格的实用法则：
  - 内部边恰被 2 个三角形共享
  - 内部顶点周围三角形构成单一闭环
- **数据结构对比**：
  - 邻接表：仅文件存储用，邻域查询 $O(N)$
  - 关联矩阵 + CSC 稀疏存储：邻域查询 $O(1)$，矩阵化代数运算
  - **半边（核心）**：每条无向边拆为两条有向半边，三个映射 next/twin/face 编码全部连接性。代数视角：偶数集合 $H$ + 置换 $\rho$ + 对合无不动点 $\eta$，**任意偶数集 + 任意置换都给出一个流形网格**。
- **拓扑数据分析（TDA）应用**：球膨胀法 + 持久同调 → 持久性图 → 拓扑指纹（玻璃态识别、神经网络分析、形状匹配）。

### 第 5-9 讲：外代数 → 微分形式 → 外微分 → Stokes 定理

这五讲是整门课的**数学基础**，必须扎实。

#### p05 外代数

- **楔积 $u \wedge v$**：构造**有向 $k$ 维体积**（$k$-向量）。性质：双线性、反对称（$u \wedge u = 0$）、结合。
- **楔积 vs 叉积**：楔积适用任意维 + 不依赖度量；叉积仅 3D + 依赖度量。
- **$\bigwedge^k V$ 的维度** $= \binom{n}{k}$（帕斯卡三角）。
- **Hodge 星 $\star$**：$k$-向量 ↔ $(n-k)$-向量；定向约定 $\alpha \wedge \star\alpha = $ 正定向单位 $n$-形式。

#### p06 对偶空间 + k-形式

- **对偶空间 $V^*$ = 测量者**；向量是被测者。
- **Sharp $\sharp$ / Flat $\flat$**：在度量 $g$ 下转换向量与余向量，**音乐同构**。
- **k-形式 = $V^k \to \mathbb{R}$ 的多重线性、完全反对称函数**：
  $$(\alpha_1 \wedge \cdots \wedge \alpha_k)(u_1, \ldots, u_k) = \det[\alpha_i(u_j)]$$
- 行列式重新认识 = 有符号体积。
- **基 1-形式 $dx^i$** 与基向量场 $\partial/\partial x^i$ 互为对偶基。

#### p07 微分 k-形式（场化）

- **微分 $k$-形式 = 在每点放一个 $k$-形式的场**（"局部测量器场"）。
- **逐点性原则**：所有运算（$\star, \wedge, $ 作用于向量场）都逐点做单点 $k$-形式代数。
- 2D Hodge 星具体规则：$\star dx = dy$，$\star dy = -dx$。

#### p08 外微分 d——课程理论核心

外微分 $d: \Omega^k \to \Omega^{k+1}$ 由三条公理唯一确定：

1. $d\phi(X) = \nabla_X \phi$（0-形式上 = 方向导数测量器）
2. $d(\alpha \wedge \beta) = d\alpha \wedge \beta + (-1)^{|\alpha|} \alpha \wedge d\beta$（分级 Leibniz）
3. $d^2 = 0$（幂零）

**统一对应**：

| 经典 | 外微分 |
| :--- | :--- |
| $\nabla \phi$ | $(d\phi)^\sharp$ |
| $\nabla \times X$ | $(\star d X^\flat)^\sharp$ |
| $\nabla \cdot X$ | $\star d \star X^\flat$ |

$d^2 = 0$ 的最深刻原因：**边界的边界为空** $\partial^2 = \emptyset$（用 Stokes 两次套）。这把代数恒等式与拓扑事实联通。

#### p09 积分与 Stokes 定理

- **积分本源**：切碎、测量、求和。
- **维度匹配**：$k$-形式只能在 $k$ 维区域上积分。
- **Stokes 定理（统一一切）**：
  $$\int_\Omega d\omega = \int_{\partial\Omega} \omega$$
  特例覆盖微积分基本定理、Green、Gauss、Cauchy 积分定理。
- **$L^2$ 内积**：$\langle\!\langle \alpha, \beta\rangle\!\rangle = \int_M \star\alpha \wedge \beta$。

### 第 10-11 讲：DEC 完整框架

#### p10 链与上链 + Whitney 基

- **离散 $k$-形式 = 给每个定向 $k$-单纯形赋一个数**（不是采样，是**积分** $\hat\omega(\sigma) = \int_\sigma \omega$）。
- **链 $C_k$**：$k$-单纯形的整数线性组合；**边界算子** $\partial$ 满足 $\partial^2 = 0$。
- **上链 $C^k$**：链上的线性映射，等价于离散微分形式。
- **De Rham 映射**：连续形式 → 离散值。
- **Whitney 基函数**：把离散值插值回连续形式。最重要的：
  - 顶点：帽函数（Lagrange 基）
  - 边：$\phi_{ij} = \phi_i d\phi_j - \phi_j d\phi_i$
  - 关键恒等式 $\int_\tau \phi_\sigma = \delta_{\sigma\tau}$ → 离散→连续→离散是恒等映射

#### p11 完整 DEC 工具箱

- **离散外微分** = **关联矩阵**：源自斯托克斯定理，**精确成立**而非近似。
  - $D_0$（顶点→边）：每行 $\pm 1$ 表起点终点
  - $D_1$（边→面）：边定向 vs 面环绕
  - $D_{k+1} D_k = 0$ 精确成立
- **对偶网格（Poincaré 对偶）**：原始 $k$-单形 ↔ 对偶 $(n-k)$-胞腔。**外心对偶**保证正交性，但钝角三角形给出负有符号长度。
- **离散 Hodge 星**：体积比 $\star \hat\alpha = \frac{|\sigma^\star|}{|\sigma|} \hat\alpha$。1-形式上化为著名的 **cotan 公式**：
  $$\frac{|e^\star|}{|e|} = \tfrac{1}{2}(\cot\alpha_{ij} + \cot\beta_{ij})$$
- **完整算子组合**：cotan Laplace $= \star_0^{-1} D_0^T \star_1 D_0$ 是 DDG 最常用算子。
- **$d$ 精确 vs $\star$ 近似**：外微分由拓扑保证精确，Hodge 星总是近似——这条线"决定了误差去哪里"。

### 第 12 讲：向量值微分形式（曲面理论的关键工具）

- **核心改变**：输出从 $\mathbb{R}$ 升级为向量空间 $U$；**多线性 + 反对称**保持不变。
- **求值就是逐分量做标量值运算**。
- **向量值楔积必须用叉积**：
  $$(\alpha \wedge \beta)(u, v) = \alpha(u) \times \beta(v) - \alpha(v) \times \beta(u)$$
- **关键反差**：与标量情形不同——
  - 输入交换：仍反对称
  - **形式交换变成对称**（叉积反对称抵消）
  - **$\alpha \wedge \alpha \ne 0$**：编码 $\alpha$ 对面积的形变倍数（类比行列式之于体积）。
- 这套机制让我们能用同一套语言描述位置 $f$、切向 $df$、法向 $n$，曲面理论由此展开。

### 第 13-14 讲：曲线（先光滑后离散）

#### p13 光滑曲线

- 隐式 vs 参数化；嵌入 vs 浸入。
- 微分 $d\gamma$ vs 单位切线 $T$（前者是函数，后者是值）。
- 弧长参数化的简化作用。
- **平面曲线基本定理**：弧长参数化下 $\kappa(s)$ 唯一确定曲线（差刚体运动）。
- **转数 vs 绕数**：前者看切向量，后者看位置向量。
- **Whitney–Graustein 定理**：平面闭曲线正则同伦 ⇔ 转数相同。
- **球面同伦类只有两类**（偶/奇转数）→ 应用于六面体网格化。
- **广义绕数**：对带裂缝的曲线/曲面，用 `atan2` 和投影面积积分提取防水形状。
- 空间曲线：曲率 $\kappa$ + 挠率 $\tau$；**Frenet–Serret** 系统；空间曲线基本定理；Bishop 框架（全局最小扭曲，直线段不失效）。

#### p14 离散曲线

- **正则性的"好定义"是局部单射**（不是边长非零）；离散浸入精确保留 Whitney–Graustein。
- **离散基本定理（平面）**：边长 $\{l_i\}$ + 外角 $\{\phi_i\}$ → 唯一曲线（差刚体运动）。
- **离散空间曲线**：边长 + 顶点曲率（外角）+ 边挠率（相邻平面法线夹角）。Rodrigues 公式实现旋转。
- 课程涉及的曲率流：
  - **曲线缩短流** $\partial\gamma/\partial t = \kappa N$（保质心、趋圆、有限时间消失）
  - **弹性流** $E = \int \kappa^2 ds$（不缩短，趋向 Elastica）—— "角度-长度重建法"是数值实现的关键技巧
  - **桥本流** $\partial\gamma/\partial t = \kappa B$（涡丝/烟圈动力学）
- **解结问题** + **Möbius 能量** + **切点能量**：把"判断是否能解开"变成连续优化。
- **粘性螺纹**与"墨水吊灯"：低维表示（中心线 + 半径）就能模拟丰富流体动力学。

### 第 15-17 讲：曲面（光滑 + 离散）

#### p15 参数化曲面 + 黎曼度量 + 抽象流形

- **外在 vs 内在**：内在描述（仅靠度量）足以推导大部分几何。
- **第一基本形式** $I = J^T J$（雅可比矩阵的 Gram 矩阵），矩阵分量 $E, F, G$。
- **共形参数化**：$g = e^{2\phi} \mathbb{I}_2$，保角不保长。
- **球面外翻**（Smale）：球面**可以**通过浸入正则同伦由内向外翻——曲面比曲线自由度高得多的标志案例。
- **抽象黎曼流形**：图集 + 重叠相容的局部度量；不需要嵌入。
- **嵌入定理**：Hilbert（双曲不能等距嵌入 $\mathbb{R}^3$）+ Nash（任意黎曼流形可等距嵌入足够高维）+ Nash–Kuiper（$C^1$ 等距嵌入的"褶皱奇迹"）。

#### p16 曲面外在几何 + Hodge 星推广

- **高斯映射** $N: M \to S^2$（连续 ⇔ 可定向）。
- **向量面积**（关键身份）：
  $$\vec{n}\,dA = \tfrac{1}{2}\,df \wedge df$$
- **闭曲面的总向量面积为零**（Stokes + $\partial = \emptyset$）→ 多边形法线由边界积分定义，与三角化方式无关。
- **曲面上的 Hodge 星**只需两份数据：**面积形式 $dA$ + 复结构 $J$**。
- **复结构 $J_f$**（参数平面上的"伪旋转"）：$df(J_f X) = n \times df(X)$。
- **三量统一**：度量 $g$ = 面积形式 $dA$ + 复结构 $J$，$g(X, Y) = dA(X, JY)$。
- **共形浸入** ⇔ 平面 90° 旋转 = $J_f$。

#### p17 离散曲面 + 离散 Laplace + 形状重建

- **正则性正确判据 = 局部单射**（朴素的"边长非零"会漏掉分支点和空间重合）。
- **顶点法线**有四种合理定义（算术平均/面积加权/角度加权/向量面积法）；**No Free Lunch** 体现：
  - 角度加权独立于三角剖分；
  - 向量面积法只依赖边界；
  - 面积加权对应"网格包围体积梯度"。
- **离散 Hodge 星统一形式** = 原始/对偶体积比；**展平等距**保证曲面与平面公式相同。
- **离散 Laplace–Beltrami（cotan Laplace）**：
  $$(\Delta\phi)_i = \frac{1}{2A_i} \sum_{j \in N(i)} (\cot\alpha_{ij} + \cot\beta_{ij})(\phi_i - \phi_j)$$
- **形状重建定理**：
  - 离散：法线 + 拓扑 → 唯一确定（在缩放/平移意义下）；四边形网格反例多。
  - 凸情形：度量唯一（Cohn-Vossen / Alexandrov）。
  - 非凸：开放问题（"Shape from Metric"是当前研究方向）。

### 第 18-20 讲：曲面曲率（核心收割期）

#### p18 曲面曲率全景

- **法曲率 $\kappa_n$**（曲面被迫给的） vs **测地曲率 $\kappa_g$**（曲线主动转的）：
  $$\frac{dT}{ds} = \kappa_n N_s + \kappa_g (T \times N_s)$$
- **形状算子 $S$**：$DF \circ S = dN$；**主曲率 = 特征值**，**主方向 = 特征向量**（在诱导度量下正交，**不**在欧氏点积下正交）。
- **高斯曲率 $K = \kappa_1 \kappa_2$（内蕴）** vs **平均曲率 $H = (\kappa_1 + \kappa_2)/2$（外蕴）**。
- **极小曲面** $H \equiv 0$（肥皂膜）；**可展曲面** $K \equiv 0$（圆柱、圆锥）。
- **测地圆面积偏差**：$A_{\text{geo}} = \pi\varepsilon^2 - \frac{\pi K}{12}\varepsilon^4 + \cdots$ → 高斯曲率内蕴定义。
- **Gauss–Bonnet 定理**（最优雅的"局部到全局"）：
  $$\int_M K \, dA = 2\pi \chi(M)$$
- **曲面基本定理**：第一 + 第二基本形式唯一确定曲面（差刚体运动）+ Gauss–Codazzi 可积性条件。
- **离散基本定理**：边长 + 二面角 → 唯一确定网格（差刚体运动）。

#### p19 离散曲率·整体观（积分视角）

- **三大微分恒等式**：
  - $df \wedge df = 2n\,dA$（面积元）
  - $df \wedge dn = 2H n\,dA$（平均曲率）
  - $dn \wedge dn = 2K n\,dA$（高斯曲率）
- **角度缺陷** $\Omega_i = 2\pi - \sum \theta_i$ = 离散高斯曲率。
- **离散 Gauss–Bonnet（精确）**：$\sum \Omega_i + \sum_{\partial} \kappa_i = 2\pi\chi$。
- **离散平均曲率法线（cotan 公式）**：通过 Stokes 把面积分变成边界线积分，在对偶胞元上推出
  $$\mathbf{H}_i = \tfrac{1}{2} \sum_j (\cot\alpha_{ij} + \cot\beta_{ij})(\mathbf{x}_i - \mathbf{x}_j)$$
  **三种独立路径殊途同归**（积分 / Laplace–Beltrami / 面积变分）→ 它是规范刻画。
- **离散高斯曲率法线**：
  $$\mathbf{K}_i = \sum_j \frac{\phi_{ij}}{l_{ij}} (f_j - f_i)$$
- **Steiner 公式视角**：把多面体取闵可夫斯基和（球加厚）→ 高斯曲率仅顶点（球面帽）贡献，平均曲率仅边（圆柱）贡献，统一光滑/离散。

#### p20 离散曲率·变分观（梯度视角）

- **黄金法则**：几何求导用"方向 + 幅度"，避免冗长偏导。
  - 角度 $\nabla_A \theta = d^\perp / \|B-A\|$
  - 面积 $\nabla_P A = \tfrac{1}{2} n \times e$
  - 边长 $\nabla l = \hat{u}$
- **核心发现**：标量曲率的梯度 = 向量曲率法线。
  $$V \xrightarrow{\nabla} \mathbf{A} \xrightarrow{\nabla} \mathbf{H} \xrightarrow{\nabla} \mathbf{K} \xrightarrow{\nabla} 0$$
  最后归零正是离散 Gauss–Bonnet。
- **Schläfli 公式**：$\sum_e l_e \delta\phi_e = 0$ → 离散 Gauss–Bonnet 的微分版，在推导高斯曲率法线时关键。
- **曲率流**：
  - 法向流（体积梯度）：膨胀/收缩，不平滑。
  - 平均曲率流（面积梯度）：标准平滑流，有限时间退化。
  - 高斯曲率流（凸面专用）：滚石下山。
  - **Willmore 能量** $\int H^2 \, dA$：真正的"喜欢平坦"——抹平细节，保留体积，趋向圆球。

### 第 21 讲：Laplacian / Laplace–Beltrami

- **多重等价定义**：偏导数之和、Hessian 的迹、$\nabla \cdot \nabla$、外微分公式 $\Delta = \delta d$、Dirichlet 能量梯度、Brownian 生成元。
- **核心几何直觉**："你比邻居有多特别"。
- **五大代数性质**：常数在核中、刚性运动不变、**等距不变**、自伴、半正定。
- **谱定理**：紧致域上自伴椭圆算子 → 离散特征值 + 正交特征基（任意流形上的"傅里叶分析"）。
- **调和函数** = Dirichlet 能量极小 = 满足均值性质 = 满足最大原理。
- **三类边界条件**：Dirichlet、Neumann、Robin。**Neumann 可解性 $\oint h \, dS = 0$** 是工程上最易踩坑的地方。
- **Poisson 方程** $\Delta u = -f$ 与 Green 函数；**几何处理中的"曲面数据传播"** = 在曲面上求 $\Delta_M u = 0$。

### 第 22 讲：测地线

- **双重定义**：最直（$\kappa_g = 0$） vs 最短（局部最小长度）——光滑下等价，离散下分裂。
- **历史**：第五公设两千年困局 → 双曲几何（Poincaré 圆盘模型）。
- **Dirichlet 能量** $E_D = \int \|\gamma'\|^2$：极小化同时强制几何最短 + 弧长参数化。曲面上得到测地线方程 $\nabla_{\dot\gamma}\dot\gamma = 0$。
- **离散最短测地线**：
  - Dijkstra 不收敛（锯齿）→ 直化迭代 → 局部最短。
  - 顶点缺陷决定行为：凸顶点（$\Omega > 0$）必绕开、鞍点（$\Omega < 0$）成"伪源"。
  - **连续 Dijkstra (MMP)**：用窗口推进。
- **闭合测地线**：光滑凸面至少 3 条；**离散反差**——绝大多数凸多面体没有简单闭合测地线（Gauss–Bonnet 推矛盾）。
- **Cut Locus**（最短不唯一的点集）+ **Medial Axis**（最近边界点不唯一的点集，形状骨架）+ Voronoi 近似。
- **Exp / Log 映射**：$\mathrm{Exp}_P(X)$、$\mathrm{Log}_P(Q)$；**Karcher 均值**（Log-平均-Exp 迭代）适用任意黎曼流形（旋转空间、形状空间、姿态平均）。
- **协变导数 + Christoffel 符号**：
  $$\Gamma^k_{ij} = \tfrac{1}{2} g^{kl} (\partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij})$$
  测地线 ODE：$\ddot\gamma^k + \Gamma^k_{ij} \dot\gamma^i \dot\gamma^j = 0$。
- **No Free Lunch（再次）**：离散下不存在同时具备"最短/最直/不交叉/对尖锐顶点鲁棒"全部性质的测地线定义。

## 六、一份"做几何处理时的检查清单"

把课程精神浓缩到工程流程：

1. **明确问题类型**：寻路（最短）？平滑（曲率流）？匹配（等距不变量）？
2. **选择正确的离散定义**——不存在最优解：
   - 物理仿真追求能量守恒？→ 变分类离散
   - 拓扑正确性优先？→ 转角法 / 角度缺陷
   - 数值收敛优先？→ 二次拟合等
3. **检查正则性**——优先用"局部单射"而非"边长非零"。
4. **拆解算子**：$d$ 是拓扑、$\star$ 是几何，写代码时就把它们分离。
5. **构建稀疏矩阵**：关联矩阵（$D_0, D_1$）+ 对角 Hodge 星（$\star_0, \star_1, \star_2$）。
6. **组合算子**：梯度/散度/旋度/Laplacian 都是上述基础矩阵的乘积。
7. **求解线性系统**——这是 60 年来高度优化的领域。
8. **校验**：
   - $D_1 D_0 = 0$？（拓扑正确性）
   - $\sum \Omega_i = 2\pi\chi$？（Gauss–Bonnet）
   - 边界条件是否相容（Neumann 必须 $\oint h = 0$）？
9. **可视化与调参**：从最简单网格开始，可视化曲率场/向量场。
10. **回到理论思考**：当算法不稳定时，多半是某个光滑等价定义在离散下不再等价——回到**翻译方法论**重新选择。

## 七、一句话总结整门课

> **DDG 教你这样思考**：所有"曲率""导数""距离"在光滑设定下都有多种等价定义，**翻译到网格后这些定义会分裂**——选哪一个由你想保留的几何性质决定。
>
> **DDG 教你这样写代码**：把每一个微分算子写成 **关联矩阵**（拓扑）和 **对角 Hodge 星**（几何）的乘积；改网格只要重算 $\star$，改算子只要重组合。
>
> **DDG 教你这样看几何**：形状不是顶点的列表，而是**曲率场**、**度量**、**第一/第二基本形式**——它们才是形状的"DNA"，顶点只是它们的一种表示。
