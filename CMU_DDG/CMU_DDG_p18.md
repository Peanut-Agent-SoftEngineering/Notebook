# CMU_DDG_p18

> 主题：曲面曲率的全景——从平面曲线、空间曲线，到曲面上的法曲率/测地曲率、主曲率、高斯曲率与平均曲率，最后到高斯-博内定理与离散基本定理。

## 一、为什么要研究曲率

### 1. 曲率的两种互补画面

| 视角 | 关注点 | 直观含义 | 典型量 |
| :--- | :--- | :--- | :--- |
| **外在（Extrinsic）** | 几何体如何"坐"在外部空间中 | 切向量 / 法向量在 $\mathbb{R}^3$ 中变化得有多快 | 法曲率、主曲率、平均曲率 |
| **内在（Intrinsic）** | 几何体自身的度量与平面相比偏离多少 | 在曲面上画一个圆，其面积/周长是否符合 $\pi r^2$ / $2\pi r$；向量沿闭环路平移一圈是否回到原方向 | 高斯曲率（绝妙定理：仅靠度量即可计算） |

**关键差别**：内在视角更深刻——它不依赖外部观察者，是流形自身的属性，并能自然推广到我们看不见外部空间的高维流形。

### 2. 曲率是物理现象的核心驱动力

许多自然过程的本质是**减少或恢复某种参考曲率**：

- **肥皂膜 / 表面张力**：把曲率"抚平"到零，最终成为平面或最小曲面。
- **弹性梁、弹簧片**：弯曲后想恢复直线（参考曲率为 0）。
- **帽子、贝壳、植物叶片**：恢复某个**非零参考曲率**（即"静止形状"）。
- **广义相对论**：时空本身由曲率方程描述，微分几何正是其语言。

### 3. 曲率的两大优良性质

**（a）坐标不变性**：曲率是几何形状的内蕴属性，不依赖坐标系。
- 比较两条曲线/曲面时无需先做刚性配准。
- *平面曲线基本定理*：仅由 $\kappa(s)$（弧长参数下）就可重构曲线，差一个刚体运动：
  $$\theta(s) = \int \kappa(s)\,ds,\qquad \mathbf{r}(s) = \int \mathbf{T}(s)\,ds$$

**（b）局部 → 全局桥梁**：曲率积分给出拓扑不变量。
- *Whitney–Graustein*：闭合平面曲线 $\oint \kappa\,ds = 2\pi \cdot (\text{环绕数})$。
- *Gauss–Bonnet*：闭合曲面 $\int_M K\,dA = 2\pi\chi(M)$。

### 4. 工程意义

| 场景 | 曲率的作用 |
| :--- | :--- |
| 物理模拟（布料、柔体、流体） | 表面张力、弯曲能、回弹力都由曲率给出 |
| 几何处理 | 平滑、锐化、去噪、形状匹配、特征点检测 |
| 渲染 | 法线贴图本质是改写微表面曲率；曲率感知着色 |
| LOD / 网格简化 | 高曲率区域保留更多三角形，平坦区域大幅塌陷 |
| 图像处理 | 轮廓检测平衡"长度力"与"曲率力" |

## 二、曲线曲率的回顾

### 1. 弧长参数化

设 $\gamma:[0,L]\to\mathbb{R}^n$，**弧长参数化**指 $\|\gamma'(s)\|=1$。此时切向量 $T(s)=\gamma'(s)$，许多公式直接简化。

### 2. 平面曲线曲率的多种等价定义

| 定义 | 公式 |
| :--- | :--- |
| 二阶导数模长 | $\kappa(s) = \|\gamma''(s)\|$（丢符号） |
| 法向投影（带符号） | $\kappa(s) = \langle N(s), \gamma''(s)\rangle$ |
| 切线角变化率 | $\kappa(s) = d\theta/ds$，$\theta$ 为切向量与水平方向夹角 |
| 几何（密切圆） | $\kappa = 1/r$ |
| 变分 | $\kappa = \delta L/\delta\gamma\cdot N$ |

> 这些定义在光滑情形等价，但**离散化后会得到不同公式**——这是离散微分几何的核心方法论（详见 p02）。

### 3. 空间曲线：曲率 + 挠率

把 $\gamma''$ 进一步分解。空间曲线需要两个量：

- **曲率 $\kappa$**：在切平面内弯曲的速率（如半径 1 的圆，$\kappa=1$）。
- **挠率 $\tau$**：偏离当前切-法平面（即扭出平面）的速率。

**Frenet–Serret 公式**（弧长参数下，$T,N,B$ 为切/主法/副法向量）：

$$
\frac{dT}{ds}=\kappa N,\qquad
\frac{dN}{ds}=-\kappa T+\tau B,\qquad
\frac{dB}{ds}=-\tau N
$$

**空间曲线基本定理**：给定 $\kappa(s)$ 和 $\tau(s)$，可唯一恢复曲线（差一刚体运动）。这是曲线形状的"DNA"，无需大量点云存储。

> 几何直觉：从单位圆出发，逐渐增加 $\tau$，曲线就从平面圆"螺旋"出去变成 helix。

## 三、曲面上的曲线：法曲率 + 测地曲率

### 1. 为什么曲线曲率要重新分解

当曲线 $\gamma$ 被限制在曲面 $M$ 上，自由空间的 Frenet 框架不再自然——更有意义的是用**曲面相关的框架**：

- $T$：曲线切向量（同 Frenet）
- $N_s$：**曲面**法向（不依赖曲线方向）
- $B_s = T\times N_s$：曲面副法向，落在切平面内

### 2. 曲率向量的分解

把 $dT/ds$（即曲率向量）投影到这组正交基：

$$
\frac{dT}{ds} = \kappa_n\,N_s + \kappa_g\,(T\times N_s)
$$

| 分量 | 公式 | 几何意义 |
| :--- | :--- | :--- |
| **法曲率 $\kappa_n$** | $\langle N_s, dT/ds\rangle$ | 曲线被曲面"带弯"的程度，方向垂直于曲面 |
| **测地曲率 $\kappa_g$** | $\langle B_s, dT/ds\rangle$ | 曲线在曲面"皮肤"内自身蜿蜒的程度 |

**总曲率关系**：$\kappa^2 = \kappa_n^2 + \kappa_g^2$。

### 3. 为什么没有第三个曲率？

设想沿 $T$ 方向也存在分量 $dT/ds\cdot T$。由于 $T$ 是单位向量，$\|T\|^2=1$ 求导得 $2\langle dT/ds,T\rangle = 0$——**单位向量的变化必须正交于自身**。所以第三个分量恒为 0，无信息量。

### 4. 直观：开车在地形上

- **法曲率**：由"重力 + 地形起伏"决定，方向盘被迫跟随地势（你必须做的转向）。
- **测地曲率**：你**主动**打方向盘左右晃动的快慢。

### 5. 测地线

**测地线**：曲面上 $\kappa_g\equiv 0$ 的曲线——曲率纯由曲面"压"出，是局部最短路径。
- 球面上的**大圆**（赤道、经线）是测地线：$\kappa_n$ 大（绕地球转），$\kappa_g=0$。
- 平面上画的圆完全是 $\kappa_g$（$\kappa_n=0$）。

## 四、Gauss 映射与 Weingarten 映射

### 1. Gauss 映射

把曲面每点的单位法向 $N$ 视为单位球面 $S^2$ 上的一点：

$$ N: M \to S^2 $$

- 平坦区域 → 法线全相同 → 映到一个点。
- 弯曲区域 → 法线分散 → 映到 $S^2$ 上一片区域；**曲率越大，覆盖面积越大**。

### 2. Weingarten 映射 $dN$

**Weingarten 映射** = Gauss 映射的微分。给一个切向量 $X$，输出 $dN(X)$ = 沿 $X$ 方向法线的变化率。

**重要性质**：因为 $\|N\|=1$，所以 $\langle dN(X), N\rangle = 0$——$dN(X)$ 始终位于切平面内。所以 $dN$ 把切空间的输入映到切空间的"输出"（在 $\mathbb{R}^3$ 中表示但落在切平面内）。

### 3. 法线椭圆

把曲面上一个微小圆盘通过 Gauss 映射送到 $S^2$，得到一个**小椭圆**（一般情况）：

| 曲面类型 | 椭圆形态 | 主曲率关系 |
| :--- | :--- | :--- |
| 平面 | 一个点 | $\kappa_1=\kappa_2=0$ |
| 柱面 | 一条线段（退化） | 一个 $\kappa$ 为 0 |
| 球面 | 圆 | $\kappa_1=\kappa_2$ |
| 一般曲面 | 椭圆 | 两轴对应主方向 |
| 鞍面 | 椭圆方向**翻转**（$K<0$） | 两个主曲率符号相反 |

椭圆的两个正交轴 = **主方向**；轴长比 ∝ 主曲率比。

## 五、法曲率的计算

### 1. 法曲率作为"方向导数"

对标量函数，方向导数告诉你沿某方向函数变化多快；对曲面，**法曲率告诉你沿某切方向法线变化多快**。

**公式**（两种等价写法）：

$$
\kappa_n(X) = \frac{\langle DF(X),\,DN(X)\rangle}{\|DF(X)\|^2}
= \frac{II(X,X)}{I(X,X)}
$$

- **分子** = 曲率倾向与方向重叠（弯曲强度）。
- **分母** = 归一化，让结果只依赖**方向**，不依赖步长。

### 2. 几何画面：切片法

1. 选一点 $p$、单位法 $N$、切方向 $dX$。
2. 用同时含 $N, dX$ 的平面切割曲面，得到一条平面曲线。
3. **该平面曲线在 $p$ 点的曲率 = 法曲率 $\kappa_n(dX)$**。
4. 旋转 $dX$，切片随之旋转，$\kappa_n$ 是 $dX$ 的函数。

可视化：画该平面曲线的密切圆，$\kappa_n=1/r$。

### 3. 算例：参数化圆柱 $f(u,v)=(\cos u,\sin u,v)$

**步骤 1**：偏导数
$$f_u = (-\sin u,\cos u,0),\qquad f_v=(0,0,1)$$

**步骤 2**：单位法
$$N = f_u\times f_v = (\cos u,\sin u,0)$$
（恰好已单位化。）

**步骤 3**：法线微分
$$N_u=(-\sin u,\cos u,0),\qquad N_v=(0,0,0)$$

**步骤 4**：代入 $\kappa_n=\langle DF,DN\rangle/\|DF\|^2$：
- 沿 $u$ 方向：分子 $= \sin^2 u+\cos^2 u=1$，分母 $=1$，故 $\kappa_n^{(u)}=1$（绕圆，半径 1）。
- 沿 $v$ 方向：$N_v=0$ → $\kappa_n^{(v)}=0$（沿母线无弯曲）。

## 六、主曲率与形状算子

### 1. 主曲率 / 主方向

旋转切方向 $X$，$\kappa_n(X)$ 是 $X$ 的连续函数，必有最大、最小值：

- **主曲率** $\kappa_1, \kappa_2$ = $\kappa_n$ 的最大、最小值。
- **主方向** = 实现极值的两个切方向。
- **正交性**：两个主方向**在曲面切平面内**正交（注意：在参数域中可能看起来不正交！见后）。

### 2. 形状算子 $S$

形状算子是从切空间到切空间的**自同态**，把 Weingarten 映射"投影"回切空间：

$$ DF\circ S(X) = DN(X) $$

矩阵关系（$DF, DN$ 均为 $3\times 2$，$S$ 为 $2\times 2$）：

$$ DN = DF\cdot S $$

**主方向 / 主曲率 = $S$ 的特征向量 / 特征值**：
$$ S(\mathbf{e}_i) = \kappa_i\,\mathbf{e}_i $$

> **注意**：$S$ 的矩阵在一般参数化下**不一定对称**。其特征向量在参数域（笛卡尔点积下）可能非正交，但通过 $DF$ 推到 $\mathbb{R}^3$ 后**一定正交**——正交性来自第一基本形式（诱导度量）$G$，而非欧氏度量：
> $$\langle a,b\rangle_G = a^\top G\,b$$

### 3. 圆柱例（标准参数化）

解 $DN = DF\cdot S$ 得
$$ S = \begin{bmatrix}1 & 0\\ 0 & 0\end{bmatrix} $$
特征值 $1, 0$ ↔ 主曲率 $\kappa_1=1$（环向）、$\kappa_2=0$（轴向），完全符合直觉。

### 4. 圆柱例（剪切参数化）：参数化扭曲的现象

取 $f(u,v)=(\cos u,\sin u,u+v)$（v 方向被剪切）。
- $f_u=(-\sin u,\cos u,1)$、$f_v=(0,0,1)$，**在欧氏点积下不正交**。
- 求出的 $S$ 矩阵不对称；其特征向量 $(0,1)^\top$ 与 $(-1,1)^\top$ 在参数平面看不正交。
- **但用 $DF$ 推到 $\mathbb{R}^3$**：
  - $DF(0,1)^\top=(0,0,1)$
  - $DF(-1,1)^\top=(\sin u,-\cos u,0)$
  - 点积 $= 0$ ✓ —— 几何主方向确实正交。

> **教训**：参数平面只是局部坐标，会扭曲几何；判断正交性必须用诱导度量 $G$，不能直接用 $du,dv$ 的欧氏点积。

### 5. 形状算子 vs Weingarten 映射

两术语经常混用，但严格说：

| | 定义域 → 值域 | 关系 |
| :--- | :--- | :--- |
| Weingarten 映射 $dN$ | 切空间 → $\mathbb{R}^3$（事实上落在切平面内） | $dN = DF\circ S$ |
| 形状算子 $S$ | 切空间 → 切空间 | 自同态，由 $S=DF^{-1}\circ dN$（左逆意义下）给出 |

## 七、脐点

### 1. 定义

**脐点（Umbilic）**：$\kappa_1=\kappa_2$ 的点。所有切方向法曲率相同，没有可区分的主方向。

形状算子在脐点处仍**满秩**，但有重复特征值：
$$S = \frac{1}{R}I$$
（球面是处处脐点的最简单例子。）

### 2. 主方向作为"规范标架"——但脐点除外

非脐点上，主方向给出曲面的**自然正交标架**；脐点处必须**任意选择**一对正交向量，这个选择不再由几何唯一决定。

### 3. 脐点的局部分类（一般稳定情形）

主曲率线在脐点附近有三种**拓扑稳定**配置：

| 类型 | 描述 | 指标 |
| :--- | :--- | :--- |
| **柠檬（Lemon）** | 曲率线呈柠檬纹 | $+1$ |
| **星形（Star）** | 曲率线辐射出 | $+1$ |
| **Mon-Star** | 介于两者之间 | $-1$ |

### 4. 拓扑约束（Poincaré–Hopf 在脐点的体现）

设 $L,M,S$ 分别为柠檬、Mon-Star、星形脐点数：

$$ L - M + S = 2\chi $$

- 球面 $\chi=2$ → 总数 $= 4$。
- 椭球验证：4 个柠檬脐点 → $4-0+0=4$ ✓。

> 不同曲面的主曲率线网络可能完全不同，但脐点类型的代数计数是**拓扑身份证**。

## 八、主曲率线与四边形网格

### 1. 主曲率线网络

沿主方向场积分 ODE
$$\frac{d\gamma}{dt} = \mathbf{e}_1 \quad\text{或}\quad \mathbf{e}_2$$
得到两族正交曲线，几乎处处构成四边形网格。

### 2. 行为与局限

| 情况 | 行为 |
| :--- | :--- |
| 椭球等良好曲面 | 主曲率线终止于脐点，划分曲面为规则区块 |
| 圆柱等可展曲面 | 出现闭合环（绕轴一圈），不终止 |
| 一般曲面 | 主曲率线常**无限螺旋**，永不闭合，网络极其混乱 |

### 3. 工程启示

- 主曲率线网格是**四边形重网格化（Quad Remesh）** 的天然候选——网格边自然对齐曲面"主轴"。
- 但直接追踪流线在通用曲面上不可靠，现代算法**借鉴主方向场但避免直接积分**（用全局优化 / 矢量场设计技术）。

## 九、高斯曲率与平均曲率

### 1. 定义

| 量 | 公式 | 几何含义 |
| :--- | :--- | :--- |
| **高斯曲率** $K$ | $\kappa_1\kappa_2$ | 曲面"内蕴弯曲"程度；可仅由度量算出 |
| **平均曲率** $H$ | $\tfrac{1}{2}(\kappa_1+\kappa_2)$ | 平均弯曲；外蕴量，依赖嵌入 |

### 2. 典型曲面的 $K, H$

| 曲面 | $K$ | $H$ | 几何 |
| :--- | :--- | :--- | :--- |
| 平面 | 0 | 0 | 完全平 |
| 球面（半径 $r$） | $1/r^2 > 0$ | $1/r\neq 0$ | 处处脐点，凸 |
| 圆柱面（半径 $r$） | 0 | $1/(2r)\neq 0$ | 一向弯一向直，可展 |
| 鞍面 | $<0$ | 视情况，可为 0 | 一向凸一向凹 |
| 极小曲面 | $\le 0$ | $\equiv 0$ | 肥皂膜 |

### 3. 三大曲面类别

| 类别 | $K$ 符号 | 几何 | 应用 |
| :--- | :--- | :--- | :--- |
| **凸曲面** | $K>0$ | 球、椭球，主曲率同号 | 碰撞体、地形隆起 |
| **可展曲面** | $K=0$ | 柱、锥，可平摊到平面而不拉伸 | UV 展开、布料裁剪 |
| **鞍形（双曲）曲面** | $K<0$ | 主曲率异号 | 流体、布料褶皱、极小曲面 |

### 4. 内蕴 vs 外蕴

- **$K$ 是内蕴的**（绝妙定理）：圆柱卷起或展开，$K$ 都为 0。
- **$H$ 是外蕴的**：必须知道曲面如何嵌入 $\mathbb{R}^3$ 才能算。

### 5. 极小曲面与 $H=0$

$H = 0$ ⟺ 两个主曲率符号相反、大小相等 ⟺ 曲面在所有点上"两向弯曲互相抵消"。
- **物理实现**：肥皂膜（表面张力让面积最小）。
- **重要性质**：极小曲面的网格非常适合做物理仿真。

## 十、高斯曲率的内蕴定义：测地圆面积偏差

### 1. 直观

在曲面某点画**半径 $\varepsilon$ 的测地圆盘**（沿测地线走 $\varepsilon$ 距离）：
- 平面：面积 $= \pi\varepsilon^2$。
- 正曲率（球状凸起）：实际面积 **小于** $\pi\varepsilon^2$。
- 负曲率（鞍）：实际面积 **大于** $\pi\varepsilon^2$。

### 2. 精确公式（泰勒展开）

$$
A_{\text{geo}}(\varepsilon) = \pi\varepsilon^2 - \frac{\pi K}{12}\varepsilon^4 + O(\varepsilon^6)
$$

由此反解：

$$
K = \lim_{\varepsilon\to 0}\frac{12\,(\pi\varepsilon^2 - A_{\text{geo}}(\varepsilon))}{\pi\varepsilon^4}
$$

### 3. 为何这是"内蕴"的强证据

- 完全**不需要法向量**或外部空间：曲面上的二维生物只测面积、长度、角度即可算出 $K$。
- 这与 $K=\kappa_1\kappa_2$ 等价——同一概念的两个面孔。

### 4. 离散对应：角亏（Angle Defect）

平面绕一点总角度 $2\pi$；曲面顶点 $v$ 处相邻三角形内角和 $\sum_f\theta_f$ 通常不等于 $2\pi$：

$$
K(v) = 2\pi - \sum_{f\in\text{faces}(v)}\theta_f
$$

这是内蕴定义在网格上的严格离散版本（详见 DDG 后续章节）。

## 十一、Gauss–Bonnet 定理

### 1. 闭合曲面版本

$$
\boxed{\int_M K\,dA = 2\pi\,\chi(M)}
$$

- $\chi$ 为欧拉示性数，纯**拓扑不变量**：$\chi = 2 - 2g$（$g$ 为亏格 / 把手数）。
- 球面 $\chi=2$，环面 $\chi=0$，双环面 $\chi=-2$，等等。

**惊艳之处**：左边是逐点测量曲率的几何积分，右边只依赖拓扑——把曲面任意挤压扭曲（不撕裂），积分值不变。

### 2. 带边界版本

$$
\int_M K\,dA + \int_{\partial M} \kappa_g\,ds = 2\pi\,\chi(M)
$$

带边界曲面 $\chi = 2 - 2g - B$（$B$ 为边界圆个数）。

**经典退化案例**：平面拓扑圆盘
- $K\equiv 0$、$\chi=1$，故 $\int_{\partial}\kappa_g\,ds = 2\pi$——恢复了平面闭合曲线的转角数定理。

### 3. 与一维曲线定理的类比

| 维度 | 曲率积分 → 拓扑 |
| :--- | :--- |
| 1D 闭合曲线 | $\oint\kappa\,ds = 2\pi\cdot(\text{环绕数})$ |
| 2D 闭合曲面 | $\int_M K\,dA = 2\pi\chi$ |

### 4. 工程价值

- 网格简化、变形如保拓扑，则总曲率积分守恒——可用作正确性校验。
- 离散网格上的角亏满足离散 Gauss–Bonnet：$\sum_v K(v) = 2\pi\chi$。

## 十二、平均曲率：Minkowski 不等式

平均曲率 $H$ 是外蕴量，没有 Gauss–Bonnet 那样普适的等式定理，**唯一干净的结论**是 Minkowski 不等式（仅适用于凸闭曲面）：

$$
\int_S H\,dA \;\geq\; \sqrt{4\pi\,A}
$$

**取等条件**：曲面为球面。
- 半径 $r$ 球面：$\int H\,dA = (1/r)\cdot 4\pi r^2 = 4\pi r$；右边 $\sqrt{4\pi\cdot 4\pi r^2}=4\pi r$。✓

## 十三、第一与第二基本形式

### 1. 定义

| | 别名 | 作用 | 类比 |
| :--- | :--- | :--- | :--- |
| **第一基本形式 $I$** | 黎曼度量 $g_{ij}$ | 切向量内积，给出长度/角度/面积 | 度量（内蕴） |
| **第二基本形式 $II$** | — | $II(X,Y)=\langle S(X), DF(Y)\rangle$，描述法线弯曲 | 曲率（外蕴） |

**与法曲率的关系**：
$$ \kappa_n(X) = \frac{II(X,X)}{I(X,X)} $$
即第二基本形式按第一基本形式归一化。

矩阵展开：若 $I=\begin{pmatrix}E&F\\F&G\end{pmatrix}$、$II=\begin{pmatrix}L&M\\M&N\end{pmatrix}$、方向 $(a,b)$：

$$
\kappa_n = \frac{La^2+2Mab+Nb^2}{Ea^2+2Fab+Gb^2}
$$

### 2. 几何意义：把曲面沿法向偏移

把曲面沿法线方向小量推动，**第一基本形式（度量）的变化率 = 第二基本形式**。这是它们"同等基本"地位的来源。

### 3. 曲面基本定理

> **$\mathbb{R}^3$ 中两个曲面具有相同的 $I$ 和 $II$ ⟺ 它们差一个刚体运动**（旋转 + 平移）。

类比曲线基本定理（$\kappa,\tau$ 决定空间曲线），$I$ 和 $II$ 共同构成曲面的"DNA"。所有其他曲率（$K, H, \kappa_1, \kappa_2$）都可由它们推出。

### 4. 可积性条件：Gauss–Codazzi 方程

并非任意 $(I, II)$ 组合都对应一个真实存在的曲面——必须满足 **Gauss–Codazzi 方程**作为可积性条件。
- 类比曲线情形：闭合曲线必须满足 $\oint\kappa\,ds = 2\pi k$，且即使如此也不保证闭合（如螺旋线）。
- 曲面上的可积性更严格：由两个非线性 PDE 描述。

## 十四、离散曲面基本定理

### 1. 离散数据的两块

光滑情形需要 $I, II$；离散情形被自然拆为：

| 离散数据 | 对应光滑量 | 编码 |
| :--- | :--- | :--- |
| **边长（Edge Lengths）** | 第一基本形式 | 度量（沿曲面的距离） |
| **二面角（Dihedral Angles）** | 第二基本形式 | 弯曲（相邻三角形间夹角） |

### 2. 离散曲面基本定理

> **给定网格的所有边长 + 所有二面角，可唯一恢复网格在 $\mathbb{R}^3$ 中的形状（差一刚体运动）。**

### 3. 重建算法（直接、简单）

1. 任取一三角形：边长决定其形状（SSS 定理 + 余弦定理 $\cos C=\tfrac{a^2+b^2-c^2}{2ab}$），任意放在空间中。
2. 沿每条边折叠相邻三角形：边长决定其形状，公共边的二面角决定其朝向。
3. 重复迭代直到所有三角形拼好。

**类比**：剪好所有纸三角形 → 按指定折角粘合。

### 4. 数据合规性 = 离散 Gauss–Codazzi

- 随便给定的边长可能违反**三角不等式**。
- 即便每个三角形合法，全局拼合时合拢处可能出现**几何冲突**（顶点不重合、面相交）。
- **离散 Gauss–Codazzi 方程**是确保数据自洽的可积性条件——尚未完全简洁刻画，是 DDG 的活跃研究问题。

### 5. 几种"用更少数据决定曲面"的特例

| 已知 | 能否唯一确定曲面 |
| :--- | :--- |
| $I + II$ | ✓（基本定理） |
| $H + I$（度量） | 几乎唯一（除若干退化情形） |
| 仅 $I$，凸曲面 | ✓（凸曲面被度量唯一决定） |
| 仅 $K$，凸曲面 | ✓（$K\Rightarrow$ 度量 $\Rightarrow$ 曲面） |
| 离散：仅边长 | 一般不行；**凸离散曲面**则可 |
| 离散：仅二面角 | **开放问题**——不清楚是固定到一族还是唯一 |
| 拉普拉斯–贝尔特拉米算子谱 | 蕴含大量信息（"听形状") |

## 十五、用曲率而非位置编码形状

### 1. 视角切换

| 编码方式 | 数据 | 性质 |
| :--- | :--- | :--- |
| 位置编码 | 顶点 $(x,y,z)$ | 直观、原始；某些操作笨重 |
| 曲率编码 | $\kappa_1,\kappa_2,K,H,\ldots$ | 抽象、内蕴；某些操作天然简洁 |

类比：时域波形 ↔ 频域谱。

### 2. 优势

- **坐标不变性**：比较两形状无需刚性配准——直接比较曲率函数即可。
- **特征增强**：放大曲率值 → 重建几何 → 棱线、山峰更突出（用顶点很难表达"放大尖锐特征"）。
- **形状编辑**：直接降低曲率绝对值 = 平滑；提升 = 起伏。
- **简洁数学**：许多定理（如 Gauss–Bonnet）在曲率语言里非常优雅。
- **效率提升**：基于位置的迭代优化，在曲率空间常有闭式解。

### 3. 标准曲率-几何流程

1. **测量**：从原始网格计算主曲率/平均曲率/高斯曲率场。
2. **操纵**：对曲率场做滤波/编辑（标量场上的简单操作）。
3. **恢复**：由编辑后的曲率场重建顶点位置（涉及离散基本定理或 PDE 求解）。

### 4. 一句话总结

> "任何一组新变量都可能打开一扇新门"——遇到几何处理难题先想：**它在曲率空间里是否更自然？**

## 十六、关键术语速查

| 术语 | 含义 |
| :--- | :--- |
| 外在 / 内在曲率 | 看曲面在外部空间的弯曲 / 自身度量与平面的偏离 |
| 法曲率 $\kappa_n$ | 沿某切方向，曲面被法向"带弯"的速率；用切片平面与曲面的交线曲率给出 |
| 测地曲率 $\kappa_g$ | 曲线在曲面切平面内的"蜿蜒"速率；测地线 $\Leftrightarrow \kappa_g\equiv 0$ |
| 主曲率 $\kappa_1,\kappa_2$ | 沿所有方向法曲率的最大、最小值 |
| 主方向 | 取得主曲率的两个正交切方向（在诱导度量下正交） |
| Gauss 映射 $N:M\to S^2$ | 把曲面每点的单位法向作为 $S^2$ 上的点 |
| Weingarten 映射 $dN$ | Gauss 映射的微分；切向量 → 法线变化率（落在切平面） |
| 形状算子 $S$ | 切空间到切空间的自同态，$DF\circ S = dN$；其特征值/向量为主曲率/方向 |
| 高斯曲率 $K$ | $\kappa_1\kappa_2$；内蕴量；面积偏差视角：$A_{\text{geo}} = \pi\varepsilon^2 - \tfrac{\pi K}{12}\varepsilon^4 + \cdots$ |
| 平均曲率 $H$ | $\tfrac{1}{2}(\kappa_1+\kappa_2)$；外蕴量；$H\equiv 0$ ⟺ 极小曲面 |
| 脐点 | $\kappa_1=\kappa_2$ 的点；分柠檬/星形/Mon-Star 三型 |
| 主曲率线 | 沿主方向积分得到的曲面"经纬线"；可展曲面上常成闭环 |
| 第一基本形式 $I$ | 黎曼度量；编码长度/角度 |
| 第二基本形式 $II$ | 编码法线弯曲；$\kappa_n = II/I$ |
| 曲面基本定理 | $I+II$ 唯一决定曲面（差刚体运动） |
| Gauss–Codazzi | $(I,II)$ 必须满足的可积性条件 |
| Gauss–Bonnet | 闭合：$\int K\,dA=2\pi\chi$；带边界：再加 $\int_{\partial}\kappa_g\,ds$ |
| Minkowski 不等式 | 凸闭曲面：$\int H\,dA\ge\sqrt{4\pi A}$，球面取等 |
| 角亏（离散 $K$） | $K(v) = 2\pi - \sum_f\theta_f$，离散面积偏差视角 |
| 离散基本定理 | 边长 + 二面角唯一确定网格（差刚体运动） |
| 可展 / 极小曲面 | $K\equiv 0$（圆柱/圆锥）/$H\equiv 0$（肥皂膜） |
