# CMU_DDG_p22

> 主题：测地线（Geodesics）——弯曲空间中"直线"的两套等价光滑定义、它们在离散网格上的分裂、相关算法（cut locus, medial axis, exp/log 映射, Christoffel 符号）以及"没有免费午餐"的工程取舍。

## 一、什么是测地线

### 1. 双重定义

把平面直线推广到曲面，要保留它的两个核心性质：

| 性质 | 平面 | 曲面 |
| :--- | :--- | :--- |
| **最直**（无加速度） | 切向量方向不变；曲率 = 0 | 测地曲率 = 0；切向量协变导数 = 0 |
| **最短**（局部极小长度） | 直线段是连接两点的最短路径 | 局部小邻域内最短 |

光滑曲面上这两个性质**等价**——任一即可推出另一。

### 2. 局部 vs 全局

- 测地线只保证**局部**最短：任意小段是其端点之间的最短路径。
- 球面上从北极沿经线绕回北极也是测地线，但不是全局最短的。
- 区分**最小测地线**（minimal geodesic）：两点之间真正最短的那一段。
- "局部最短"的范围由**内射半径**（injectivity radius）刻画：在此半径内最短路径唯一。

## 二、历史背景：从欧几里得到非欧几何

### 1. 五大公设

1. 任意两点连一直线段。
2. 直线段可无限延伸。
3. 给定线段为半径可作圆。
4. 所有直角全等（即"垂直"是绝对一致的）。
5. **平行公设**：过直线外一点存在且唯一存在一条不与原直线相交的直线。

### 2. 平行公设的两千年困局

欧几里得本人就怀疑第五公设——他在证明中尽量避免使用，希望它能从前四个推出。两千多年间无数尝试全部失败，直到 19 世纪明白：**它根本无法从前四个推出**，因为存在多个满足前四个但不满足第五的几何系统。

### 3. 三种几何对比

| 几何 | 曲率 | 平行线数（过线外一点） |
| :--- | :--- | :--- |
| 欧几里得（平面） | $0$ | **唯一**一条 |
| 椭圆（球面） | $> 0$ | **0 条** —— 任意两条直线（大圆）必相交 |
| 双曲 | $< 0$ | **无穷多条** |

球面其实早就给出过提示：地球上的大圆"直",且任两个大圆必交。但真正深刻的革命来自双曲几何（恒定负曲率，类马鞍）。

### 4. 双曲几何的可视化

由于负曲率不能在 $\mathbb{R}^3$ 中等距嵌入，常用 **Poincaré 圆盘模型**：双曲空间内的"直线"（测地线）画成与圆盘边界正交的圆弧。模型扭曲了视觉，但保留了内蕴几何。

> **工程类比**：墨卡托投影把地球大圆扭成弧线；圆盘模型把双曲直线扭成圆弧。可视化方式与数学本质可以分离。

## 三、应用与典型实例

### 1. 球面大圆与航班路径

- 大圆 = 球面 ∩ 过球心的平面。
- 跨洋航线在墨卡托地图上是弧线，**沿球面是大圆段**——真正的最短路径。
- 路径搜索若直接用欧氏距离作启发式，结果将偏离曲面真实最短。

### 2. 引力透镜

广义相对论：质量弯曲时空，光沿弯曲时空的测地线传播。中间星系把远方光线"绕弯"，造成放大效应——即使被遮挡也能观测。

### 3. 几何处理中的应用

- **表面网格生成**：种子点 + 测地圆生长 + 碰撞 → Voronoi 切分 → 三角化。点越密三角形越规整。
- **计算制造（测地带）**：把复杂曲面分解为沿单方向几乎不弯的条带（金属板、玻璃易加工）。
- **形状分析与对应**：测地距离在**等距变形**下不变（不拉伸的弯折）；用它匹配同一物体的不同姿态扫描。等距下黎曼度量、面积、角度全保留——因此最短路径也保留。

## 四、Dirichlet 能量与变分推导

### 1. 重参数化与弧长

任意参数化曲线 $\gamma(t),\ t\in[0,1]$，其几何形状不变但"行进速度"可变。设累积弧长 $C(t)\in[0,L]$，弧长参数化 $\hat\gamma(s)$ 满足 $\|\hat\gamma'(s)\| = 1$，则
$$\gamma(t) = \hat\gamma(C(t)),\qquad \|\gamma'(t)\| = |C'(t)|.$$

### 2. Dirichlet 能量与长度的关系

$$E_D[\gamma] = \int_0^1 \|\gamma'(t)\|^2\,dt$$

由 Cauchy-Schwarz：

$$L^2 = \left(\int_0^1\|\gamma'\|\,dt\right)^2 \le \int_0^1\|\gamma'\|^2\,dt = E_D[\gamma]$$

等号成立 $\iff$ $\|\gamma'\|$ 为常数 $\iff$ 弧长（匀速）参数化。

> **核心结论**：最小化 $E_D$ 同时强制 (a) 几何长度最小 (b) 参数化匀速。这是把"最短路径"问题变成无约束变分问题的关键。

### 3. 端点固定下的极小：γ'' = 0

变分：$\gamma_\varepsilon = \gamma + \varepsilon\eta$，$\eta(0) = \eta(1) = 0$。
$$\delta E_D = \int_0^1 \gamma'\cdot\eta'\,dt = -\int_0^1 \gamma''\cdot\eta\,dt = 0\;\;\forall\eta\;\Rightarrow\;\gamma''(t) = 0$$

平面上：$\gamma$ 是从 $P$ 到 $Q$ 的匀速直线段，$\min E_D = \tfrac12\|Q-P\|^2$。

### 4. 推广到曲面：临界点未必全局最小

曲面上 $|\gamma'|^2 = g(\gamma',\gamma')$，把 $\gamma''$ 替换为协变导数 $\nabla_{\dot\gamma}\dot\gamma$，得到测地线方程：

$$\boxed{\;\nabla_{\dot\gamma}\dot\gamma = 0\;}$$

测地线仍是 $E_D$ 的临界点，但**不再唯一是全局极小**：

- **球面上同纬度两点**：短弧（极小）+ 长弧（**鞍点**，沿某些方向能量上升、沿另一些下降）都满足测地线方程。
- **意义**：测地线 = 临界点（一阶变分为零），不一定是极小。光滑情形是凸问题，曲面情形非凸。
- **绝大多数曲面没有测地线的解析公式**——必须数值求解。

## 五、离散测地线：两条不再等价的路径

光滑下"最直"="最短"，进入离散三角网格立刻分裂。

### 1. 视角对比

| 视角 | 离散对应 | 算法风格 |
| :--- | :--- | :--- |
| **最短**（locally shortest） | 长度极小（含展开内部直线段） | 边值问题（已知两点） |
| **最直**（straightest） | 顶点两侧角度平衡 | 初值问题（已知点+方向） |

### 2. 没有免费午餐

光滑下成立的多个性质（最短、最直、不交叉、过任意点的指数映射满射、对尖点鲁棒……）**在离散下无法被同一定义同时保留**。应用场景决定选哪种：

- 已知两点求路径 → 最短测地线。
- 已知起点和方向追踪 → 最直测地线。
- 物理仿真（追求局部直线性）→ 最直。
- 寻路 / 距离场 → 最短。

## 六、最短测地线：算法与顶点行为

### 1. Dijkstra 不是答案

把网格当成图、边当成权，Dijkstra 找出来的路径只能沿边走，与真正穿过三角形内部的对角线相比，长度差不会随细化收敛——典型的"水平 + 垂直锯齿"，永远比对角线长。

### 2. 直化迭代

从 Dijkstra 路径出发，对每对相邻边检查能否"切入三角形内部"以缩短：

```
Input: Dijkstra 路径 P
While P 可缩短:
  For 每对相邻边: 让其穿过三角形以缩短
Output: 局部最短测地线 P'
```

收敛到**局部**最短。**不保证全局**——初始猜测决定收敛终点（典型的下山陷阱）。

### 3. 顶点角度缺陷决定行为

在顶点 $i$ 周围所有面的内角和为 $\sum\theta$，**角度缺陷**：

$$\Omega_i = 2\pi - \sum\theta$$

| 缺陷 | 几何 | 展开特征 | 测地线行为 |
| :--- | :--- | :--- | :--- |
| $\Omega = 0$（平坦） | 局部可展开 | 无切割、无缺口 | **径直穿过** |
| $\Omega > 0$（凸 / 锥顶点） | 角度和 < $2\pi$ | 沿一条边切开后留缺口 | **绕过**——穿过比绕行长，所以最短测地线**不经过凸顶点** |
| $\Omega < 0$（鞍点） | 角度和 > $2\pi$ | 切开后展平面自重叠 | **多条**局部最短测地线汇聚于同一顶点（"伪源 / pseudo-source"） |

> **类比双曲几何**：负缺陷顶点扮演双曲点角色，存在"一族平行测地线"经过同一点。

### 4. 连续 Dijkstra（MMP 算法）

传统 Dijkstra 沿图边，**连续 Dijkstra**（Mitchell-Mount-Papadimitriou）允许路径穿过三角形内部：

- **基本对象**：不是边，而是**窗口（window）**——三角形边上的一个连续区间，区间内每点经过同一系列展开三角形后，与源点有同样的"局部最优直线段"。
- **传播**：按距离顺序展开窗口直至覆盖全曲面。
- **复杂度**：最坏 $O(N^2\log N)$；可通过修剪、并行、近似改进；实际网格往往远低于最坏情况。
- **可视化**：从源点画出"射线"图样，等距集合就是测地距离场的水平集。

### 5. 非交叉性

**光滑情形**：从源 $P$ 到不同点的两条最小测地线 $\gamma_1,\gamma_2$ 只在一条完全包含另一条时相交。直观上：从同一源出发的不同最短路径不交叉——保证距离场的水平集结构简单。

**离散情形**：在鞍形（伪源）处此性质受损——多条最短路径可在同一顶点重合或分叉。检验离散算法是否忠实于光滑几何，**非交叉性**是常用的判据。

## 七、闭合测地线

### 1. 光滑结论

任意闭合凸光滑曲面上至少存在 **3 条**简单闭合测地线（不自交的环路）。其中之一称为 **Birkhoff 赤道**。证明依赖变分法和稳定性分析。

### 2. 离散反差

**绝大多数凸多面体没有任何简单闭合测地线**。立方体等高度对称的多面体可有"赤道线"，但对其顶点做任意微小扰动就立即破坏所有闭合测地线。

### 3. 离散 Gauss-Bonnet 推矛盾

设凸多面体上存在简单闭合测地线 $\gamma$。
- $\gamma$ 不能经过凸顶点（最短测地线不绕过它们）。
- 沿 $\gamma$ 切开 → 两个圆盘 $A, B$（$\chi = 1$）。
- 每个圆盘上的离散 Gauss-Bonnet：内部顶点角度缺陷之和 + 边界（包括外角）= $2\pi\chi = 2\pi$。
- 由于 $\gamma$ 不过任何顶点、本身又是测地（边界测地曲率为 0），相当于要求**每块的内部角度缺陷之和恰好为 $2\pi$**。
- 凸多面体总缺陷 $4\pi$，要求把它干净分成两个 $2\pi$——对一般几何是极端巧合，扰动几何即破坏。

### 4. 工程含义

- 在普通三角网格上搜索闭合测地线一般无解；算法多为近似或要求特定对称。
- 网格简化、去噪等操作会轻易破坏原本存在的闭合测地线——是网格处理稳定性的重要考量。

## 八、Cut Locus 与内射半径

### 1. Cut Locus（切割轨迹）

源点 $P$ 的 cut locus = 不存在唯一最小测地线连接 $P$ 与 $Q$ 的所有 $Q$ 的集合。

直观图像：从 $P$ 同时向所有方向发射波前，cut locus 是**波前首次自相遇**的地方。

### 2. 内射半径

$P$ 到 cut locus 的最短距离。半径以内最短路径唯一，指数映射保持单射。

### 3. 经典例子

- **球面**：任一点 $P$ 的 cut locus 就是其对极点 $-P$（无数条等长大圆）。
- **复杂曲面（带凸起的环面等）**：cut locus 可以是一系列曲线段。

### 4. 离散曲面：cut locus = 树

**关键差异**：在离散曲面上，绕过任何凸（正缺陷）顶点总比直穿短 → cut locus **必然包含每个圆锥顶点**，并形成连接它们的**树状结构**。即使该多面体在视觉上接近一个光滑曲面（其 cut locus 是一条简单线段），离散版的 cut locus 也呈现出复杂的分支树。

> **警告**：精确的离散逼近不一定忠实复现光滑结构。离散切割轨迹与光滑切割轨迹形态可能完全不同。

## 九、Medial Axis（中轴）

### 1. 定义

带边界域 $\Omega$ 的中轴 = 不存在唯一最近边界点的所有 $P$ 的集合。直观为形状的"骨架"。

### 2. Medial Ball

球心在中轴、半径 = 球心到边界的最短距离。中轴球与边界**至少在两个点等距相切**。

$$r(c) = \min_{p\in\partial\Omega}\|c - p\|$$

### 3. 双重表示

中轴 + 半径场 = 形状的完整等价表示：

$$\Omega = \bigcup_{c\in M}\text{Ball}(c, r(c))$$

应用：碰撞检测、形状简化、流体粒子重采样、骨骼生成。

### 4. 实例

| 形状 | 中轴 |
| :--- | :--- |
| 圆盘 | 单点（圆心） |
| 椭圆 | 长轴上的线段（中心也在内——中心球与边界两侧相切） |
| 正方形 | X 形（4 条对角分支） |
| 矩形 | 中央线段 + 四条对角分支（"工字"形） |
| 非凸多边形 | 凸顶点处分支吸引；凹顶点处出现弧线 |
| 任意多边形 | 永不收缩为单点（除真圆）；总有星状分支指向每个凸顶点 |

### 5. 三分支稳定性

中轴典型分支点对应**三个等距接触点**（局部不可再退化）。
- 两个接触点：普通中轴线上的点。
- 三个接触点：稳定分支点。
- 四个或更多：理论上不稳定（"四腿桌子"——任何边界扰动会退化为三）。

### 6. 离散逼近的伪影

直接对多边形求中轴会冒出大量"细短分支"靠近角落，与光滑形状的理想中轴差异极大；细化网格不会自然收敛。简单阈值过滤过于 ad hoc。学术界投入大量工作寻找更鲁棒的离散骨架定义。

### 7. 高维

中轴定义在任意维都成立。3D 中骆驼形状的中轴是**内含曲面片**而不是曲线——结构更复杂但原理一致。这些曲面片通常是**二次曲面片段**（如抛物面、双曲面），形态远比 2D 的"骨架曲线"丰富。

### 8. Voronoi 近似

**关键观察**：在边界上密采样后构造 Voronoi 图，每个 Voronoi cell 形状极"瘦高"——**长边**与**短边**界限明显。

| 应用 | 保留 | 输出 |
| :--- | :--- | :--- |
| 中轴提取 | **短边** | 形状骨架 |
| 曲面重建 | **长边**（连相邻瘦 cell 的采样点） | 三角网格 |

收敛性：采样点 $|P|\to\infty$ 时，保留的短边图收敛于真实中轴；保留的长边连接收敛于真实曲面。

## 十、最直测地线视角与曲率分解

### 1. 切向无加速度

平面直线刻画：切向量恒定 ⇔ 速度向量不"转弯不加速"。在曲面上不能要求切向量在 $\mathbb{R}^3$ 中恒定（那将离开曲面），需要**只在切平面内**衡量变化。

### 2. 曲率的正交分解

沿曲面上的曲线 $\gamma$，定义局部坐标架 $T$（切向量）、$n$（曲面法向）、$B = T\times n$（副法线，仍在切平面内）：

$$\frac{dT}{ds} = \kappa_n\,n + \kappa_g\,B$$

| 分量 | 含义 |
| :--- | :--- |
| $\kappa_n$（**法曲率**） | 曲线被迫跟随曲面弯曲的部分（"地球开车"必然产生） |
| $\kappa_g$（**测地曲率**） | 曲线在曲面切平面内主动转向的部分（驾驶员转方向盘） |

**测地线 ⇔ $\kappa_g = 0$**：曲线只随曲面被动弯，不主动转向。

> 在地球表面笔直行驶（不转方向盘）时，路径的总曲率非零（法曲率），但测地曲率为零——你"感觉"在直行，物理路径是大圆。

## 十一、离散最直测地线

### 1. 离散曲线表示

三角剖分曲面 $M$ 上的离散曲线 $\gamma$ 是分段线性连续曲线，**不必沿网格边**。它可以穿过面内、跨边、经过顶点。

**重心坐标编码**：每个段记录所在单纯形（顶点 / 边 / 面）和段内位置。例如三角形 $[i,j,k]$ 内一点 $p = \lambda_i v_i + \lambda_j v_j + \lambda_k v_k$，$\sum\lambda = 1$，$\lambda \ge 0$。

### 2. 内在视角下的角度测量

**关键事实**：每个三角形本身平面，相邻三角形可沿共用边**无扭曲展开**到同一平面（不改变内蕴角度）。所以离散测地曲率纯由角度决定，与曲面在 $\mathbb{R}^3$ 中如何弯折无关。

### 3. 三种位置的处理

| 曲线在该处 | 测地曲率定义 |
| :--- | :--- |
| 三角形面内 | 平面定义：转角 = $\pi - $ 内角 |
| 边上 | 把两个相邻三角形展开到同一平面 → 同上 |
| 顶点上 | 见下文角度平衡 |

### 4. 顶点处：从"曲率为零"到"角度平衡"

光滑直线的等价描述：**两侧夹角相等**（在直线一点画小圆，两侧弧长相等）。这避免了对顶点"展平"——离散网格上顶点处无法保角展开。

**离散最直测地线在顶点的判据**：

$$\theta_L = \theta_R = \frac{1}{2}\sum_i \theta_i$$

其中 $\sum\theta_i$ 是该顶点周围所有面的内角和。
- 平坦顶点（$\sum = 2\pi$）：$\theta_L = \theta_R = \pi$，化为光滑直线条件。
- 一般顶点：$\theta_L = \theta_R \ne \pi$——这是离散特有现象，但精神（局部"对称穿过"）与光滑直线一致。

### 5. 为何顶点穿越很重要

虽然单条曲线"恰好穿顶点"概率为零，但在**模拟从源点向外传播的连续波前**时，无可避免会频繁经过顶点。处理不当将引入伪影。
- **共轭点**（conjugate points）：波前首次与自身相遇的尖锐点；理解焦散 / 能量聚焦的关键。

## 十二、指数映射与对数映射

### 1. 指数映射 Exp

$\mathrm{Exp}_P(X) = \gamma(1)$，其中 $\gamma$ 是满足 $\gamma(0) = P$、$\dot\gamma(0) = X$ 的测地线。

直观：从 $P$ 沿切向量 $X$ 方向走 $\|X\|$ 长度后落在哪。

**满射性**：紧致无边光滑曲面上 Exp 是满射——任一点 $Q$ 都可由某切向量 $X$ 达到（取 $P\to Q$ 最短测地线初速度即可）。

**非单射性**：不同 $X_1, X_2$ 可映到同一 $Q$（球面绕赤道多绕几圈仍回到同一点）。

### 2. 对数映射 Log

$\mathrm{Log}_P(Q) = X$ 使 $\mathrm{Exp}_P(X) = Q$。一般取**主值**（最短测地线对应的最小范数向量）。

```
Exp_P(X) = Q  正向：方向 + 距离 → 终点
Log_P(Q) = X  逆向：终点 → 方向 + 距离
```

由于 Exp 非单射，Log 可能多值。

### 3. 离散实现

**Exp**：从 $P$ 所在三角形沿方向 $X$ 发射线 → 与边相交 → 跨边时保持夹角进入下一个三角形。本质上等价于"把途经三角形展开到同一平面、画一条直线、再折回曲面"。

**Log**：更复杂的逆问题。注意起点终点同三角形 / 终点在边上的退化情形需明确规则。

### 4. Karcher 均值

弯曲空间中"平均"无法用欧氏算术均值（结果跑出曲面）。Karcher 均值定义为：

$$\mu = \arg\min_X \sum_i d_g(X, Y_i)^2$$

迭代求解（**Log-平均-Exp**）：

```
choose initial μ
repeat:
  v_i = Log_μ(Y_i)        # 拉到切平面
  v̄ = (1/N) Σ v_i           # 切平面线性平均
  if ‖v̄‖ < ε: break
  μ = Exp_μ(v̄)             # 沿测地走一步
```

收敛公式：

$$\mu_{k+1} = \mathrm{Exp}_{\mu_k}\Big(\frac{1}{N}\sum_i \mathrm{Log}_{\mu_k}(Y_i)\Big)$$

**注意**：Karcher 均值不唯一（球面上对极点的 Karcher 均值是整个赤道），算法收敛到的是其中一个。适用于任意黎曼流形——旋转空间 $SO(3)$、形状空间、姿态平均等都依赖这套机制。

## 十三、离散 Exp/Log 的局限

### 1. 内射半径很小

离散网格上波前快速自交。极限 = 到最近正缺陷顶点的距离。光滑曲面内射半径由曲率主导（一般不小）；离散曲面所有曲率集中在顶点 → 内射半径被严重压缩。

### 2. 离散 Exp 不满射

鞍点（角度和 $> 2\pi$）使得波前在该点附近**发散**，留下未被覆盖的"阴影区域"——某些点无法从源点经一条测地线到达。

### 3. "没有免费午餐"

离散世界不存在同时具备 (a) 局部最短 (b) 局部直 (c) 全局非交叉 (d) 对尖锐几何鲁棒 等所有性质的定义。**应用决定取舍**：

- 寻路 / 距离场：最短。
- 物理仿真 / 波传播：最直。
- 跨尖锐顶点：最直更自然。
- 形状对应 / 网格匹配：等距不变性最重要。

## 十四、协变导数（动态视角）

### 1. 为何普通导数失败

测地线动态定义：切向加速度为零。但 $\gamma''(t)\in\mathbb{R}^3$ 一般不在切平面里——既含切向也含法向分量，**法向分量是曲面强迫的**，不是真"加速"。

### 2. 外蕴定义

$$\nabla_X Y := \pi_T(\partial_X Y)$$

把 $\mathbb{R}^3$ 中的普通导数 $\partial_X Y$ 投影到切平面 $T_pM$。这是"扣除曲面强迫部分"。
- **测地线方程**：$\nabla_{\dot\gamma}\dot\gamma = 0$。

### 3. 内蕴公理化（Levi-Civita 联络）

仅靠度量 $g$ 唯一确定 $\nabla$（不需要曲面如何嵌入）。公理：

| 公理 | 公式 |
| :--- | :--- |
| 对方向场函数线性 | $\nabla_{fX}Y = f\nabla_X Y$ |
| 对被求导场加性 | $\nabla_X(Y+Z) = \nabla_X Y + \nabla_X Z$ |
| Leibniz | $\nabla_X(fY) = (X[f])\,Y + f\nabla_X Y$ |
| 与度量相容 | $Z\,g(X,Y) = g(\nabla_Z X, Y) + g(X, \nabla_Z Y)$ |
| 无扭转 | $\nabla_X Y - \nabla_Y X = [X, Y]$ |

满足以上公理的 $\nabla$ 唯一存在——这是**黎曼几何基本定理**。

### 4. Lie bracket 与扭转

$[X, Y]$ 衡量两个向量场的流"通勤失败"程度：从 $P$ 沿 $X$ 走 $\tau$、沿 $Y$ 走 $\tau$、沿 $-X$ 走 $\tau$、沿 $-Y$ 走 $\tau$，得到点 $Q$。

$$[X,Y] = \lim_{\tau\to 0}\frac{\overrightarrow{PQ}}{\tau}$$

局部坐标：$[X,Y]^i = X^j\partial_j Y^i - Y^j\partial_j X^i$。坐标基向量两两 $[X_i, X_j] = 0$（流通勤），是后续推导的关键简化。

## 十五、Christoffel 符号与测地线 ODE

### 1. 从公理到显式公式

公理只刻画 $\nabla$ 的存在性，要计算需用度量及其导数。把 $\nabla_X Y$ 与一向量 $W$ 内积，对三个相关方程 (1) + (2) − (3) 利用相容性 + 无扭转，最终得：

$$2\,g(\nabla_U V, W) = U\,g(V,W) + V\,g(W,U) - W\,g(U,V) + g([U,V], W) + g([W,U], V) + g([V,W], U)$$

由于 $W$ 任意、$g$ 可逆，$\nabla_U V$ 被唯一解出。

### 2. Christoffel 符号

把基向量场之间的关系编码为系数：

$$\nabla_{X_i} X_j = \Gamma^k_{ij}\, X_k\qquad(\text{对 } k \text{ 求和})$$

显式公式（在坐标基 $[X_i, X_j] = 0$）：

$$\boxed{\;\Gamma^k_{ij} = \frac{1}{2}\,g^{kl}\big(\partial_i g_{jl} + \partial_j g_{il} - \partial_l g_{ij}\big)\;}$$

其中 $g_{ij} = g(X_i, X_j)$，$g^{kl}$ 是 $g$ 的逆矩阵分量。这是从度量到协变导数的**纯代数操作**——只要能写出 $g$，就能写出 $\Gamma$。

### 3. 测地线 ODE

$\nabla_{\dot\gamma}\dot\gamma = 0$ 在坐标分量下：

$$\boxed{\;\ddot\gamma^k + \Gamma^k_{ij}\,\dot\gamma^i\,\dot\gamma^j = 0\;}$$

这是一个标准二阶 ODE，可直接喂给 Forward Euler、Runge-Kutta 等数值积分器。

### 4. 参数曲面流水线

对参数曲面 $F(u,v)\to\mathbb{R}^3$：

```
1. 雅可比 J = [∂F/∂u  ∂F/∂v]
2. 第一基本形式 g = JᵀJ（2×2）, 求 g⁻¹
3. 由 g 与 ∂g 计算 Γ^k_{ij}
4. ODE：γ̈ + Γ γ̇γ̇ = 0；初值 (γ(0), γ̇(0))
5. RK / Euler 积分；映射回 ℝ³：F(γ(t))
```

参数空间中的曲线看起来未必"直"，但通过 $F$ 推到 $\mathbb{R}^3$ 上确是平滑测地线。

## 十六、两种数值方法对比

| 方法 | 离散追踪 | Christoffel + ODE |
| :--- | :--- | :--- |
| 几何前提 | 三角网格 | 光滑参数化曲面 |
| 在尖点表现 | 直接处理（顶点角度规则） | 失败（$\Gamma$ 求导失效） |
| 实现复杂度 | 顶点 / 边特例多 | 一旦写出 $g$，就是标准 ODE |
| 准确性 | 取决于网格分辨率 | 光滑区域很高 |
| 通用性 | 跨光滑 / 离散统一 | 仅光滑 |

通常工程实践偏好离散追踪——通用性与代码简洁。

## 十七、最短 vs 最直：最终对比

| 性质 | 最短 (shortest) | 最直 (straightest) |
| :--- | :--- | :--- |
| 经过凸顶点 | **不能**（绕开比直穿短） | **可以**（角度平衡规则） |
| 经过鞍点 | 可以；多条同时汇聚（伪源） | 可以 |
| 凸多面体上 | 同时也是最直 | 不一定最短 |
| 非凸多面体上 | 在 cut locus 等结构上行为奇特 | 指数映射可能不满射 |
| 自然问题 | **边值**（已知两点） | **初值**（已知点 + 方向） |
| 应用 | 寻路、距离场、形状对应 | 波前传播、纹理流、物理仿真 |

> **总结**：在凸面上两者大致一致；非凸（尤其鞍点）必分道扬镳。根据问题类型选最合适的工具，比追求"万能定义"更实用。多视角是几何处理工程师的关键资本。

## 十八、关键术语速查

| 术语 | 含义 |
| :--- | :--- |
| 测地线 | 曲面上"直线"的推广；最直 = 最短（光滑下等价） |
| 最小测地线 | 两点间真正最短的测地线段 |
| 内射半径 | 从源点出发，最短测地线保持唯一的最大距离 |
| 测地曲率 $\kappa_g$ | 切向量在切平面内的弯曲；$\kappa_g = 0$ ⇔ 测地线 |
| 法曲率 $\kappa_n$ | 切向量被曲面强迫沿法向变化的部分 |
| 角度缺陷 $\Omega = 2\pi - \sum\theta$ | 顶点处离散曲率；正 = 凸 / 锥；零 = 平坦；负 = 鞍 |
| 角度平衡 | 离散最直测地线在顶点判据：两侧夹角相等 |
| 伪源 (pseudo-source) | 鞍点处汇聚的多条最短测地线"虚拟起点" |
| Birkhoff 赤道 | 闭凸光滑曲面上必然存在的简单闭合测地线 |
| 离散 Gauss-Bonnet | $\sum$ 内部缺陷 + 边界外角和 = $2\pi\chi$ |
| Cut locus | 最短测地线不唯一的点的集合 |
| Medial axis | 到边界最近点不唯一的点的集合（形状骨架） |
| Medial ball | 球心在中轴、与边界至少两点等距相切 |
| 双重表示 | 形状 = 中轴上所有 medial ball 的并 |
| Dijkstra 锯齿 | 边路径不收敛于真实测地线；需直化迭代 |
| Continuous Dijkstra (MMP) | 用窗口推进的最短测地线算法 |
| 窗口 (window) | 三角形边上的连续区间，共享一组等价测地路径 |
| Exp 映射 | $\mathrm{Exp}_P(X) = \gamma(1)$；切向量 → 曲面点 |
| Log 映射 | Exp 的逆；曲面点 → 切向量（一般取主值） |
| Karcher 均值 | 弯曲空间中最小化测地距离平方和的"平均点"；迭代 Log-平均-Exp |
| 协变导数 $\nabla_X Y$ | 曲面切空间中的方向导数；扣除法向分量 |
| Levi-Civita 联络 | 满足公理（线性、Leibniz、相容、无扭转）的唯一 $\nabla$ |
| Lie bracket $[X,Y]$ | 两向量场流的通勤失败量 |
| Christoffel 符号 $\Gamma^k_{ij}$ | $\nabla_{X_i}X_j = \Gamma^k_{ij}X_k$；由度量及其导数构造 |
| 测地线 ODE | $\ddot\gamma^k + \Gamma^k_{ij}\dot\gamma^i\dot\gamma^j = 0$ |
| 等距不变性 | 保距弯曲下黎曼度量、测地线、测地距离全部不变 |
| 共轭点 | 波前首次自相遇的奇异点；焦散行为 |
| No Free Lunch | 离散下不存在同时保留所有光滑性质的测地线定义 |
