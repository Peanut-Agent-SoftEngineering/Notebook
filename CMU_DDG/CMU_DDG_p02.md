# CMU_DDG_p02

> 主题：离散微分几何（DDG）课程导论 + 以平面曲线曲率为案例演示"光滑 → 离散"的翻译方法论

## 一、课程定位与核心思想

### 1. 为什么学几何？

- **几何数据无处不在**：3D 是继图像、声音、视频之后的下一类自然信号。
- **硬件驱动**：消费级深度扫描仪、3D 打印机、AR/VR 头显已经普及，对应用算法的需求随之爆发。
- **应用领域**：计算机视觉、计算生物学、工业设计、机器学习、建筑、计算力学、医学成像、时尚等。

### 2. 几何处理：信号处理的三维推广

- **共同问题**：降噪、上/下采样。
- **独特问题**：曲面参数化、对应点匹配、形状分析（聚类、对称性检测、关键点匹配）等。
- **机器学习挑战**：CNN 这类网络是为**规则采样**（像素阵列）设计的，而 3D 数据是**不规则采样、不规则三角化**的，必须用全新视角看待 → 这正是 DDG 发挥作用的地方。

### 3. 双向思维

| 方向 | 描述 |
| :--- | :--- |
| 连续 → 离散 | 传统方向：从方程出发，离散化后用于计算/模拟 |
| 离散 → 连续 | DDG 独特视角：从天然离散现象（碳晶格、自旋）出发，宏观上视为连续几何 |

> **核心收益**：通过对比两种视角，能更深刻理解几何**本质**——它不依赖微积分这一具体工具，而是更基本的自然规律。

### 4. 课程目标：培养"形感"

- **不是教一个算法库**，而是培养**元能力**：
  - 看到点云/网格 → 想到背后的连续几何量（曲率、法向、测地线）
  - 看到曲面方程 → 立刻意识到如何**自然地离散化**而不丢几何本质
- **作业模式**：从第一性原理推导算法，再在代码框架中实现。

## 二、微分几何 vs 离散微分几何

### 1. 微分几何

- **定义**：用微积分语言讨论形状的**局部性质**，并连接到**全局性质**。
- **局部量**：切向量、曲率、法向、测地距离等。
- **全局量**：拓扑亏格（孔洞数）等。
- **核心关系**：经典例子——边界曲率的积分等于内部孔洞数（Gauss-Bonnet 雏形）。
- **地位**：20 世纪几何、物理、统计的**通用语言**。

### 2. 离散微分几何（DDG）

- **核心**：用**有限**的长度、角度等量，描述局部和全局几何性质，**不用极限和无穷小**。
- **惊人之处**：信息损失极少——多数关键定理（如 Gauss-Bonnet）能用纯离散语言重述。
- **两大优势**：
  1. **直观**：所需数学基础更少，更快接触定理本质。
  2. **天然适配计算**：计算机本就只能处理有限存储和操作。

### 3. "形状"的普遍性

任何满足约束 $f(x)=0$ 的解空间本质上是**流形**。所以 DDG 的工具不仅服务图形学，**任何优化、约束求解、计算任务背后都可能藏着几何**。

## 三、离散化方法论："翻译"三步法

DDG 的核心工作流不是"近似"，而是 **翻译 → 比较 → 筛选**。

### 步骤一：列出所有等价定义（光滑设定）

针对一个概念（如"曲率"），在光滑设定下列出**所有数学等价**的定义：
1. 几何定义：密切圆半径的倒数。
2. 微分定义：切向量变化率 $\kappa = \|d\mathbf{T}/ds\|$。
3. 变分定义：长度对位置变化的敏感度。
4. Steiner 定义：法向偏移后长度变化率。

### 步骤二：对每个定义独立离散化

把每个定义"逐字逐句"应用到离散对象（折线、三角网格）上。需要按离散对象的特性（直线段、平面三角形）重新解释"切线""导数""极小化"等元素。

> **关键现象**：原本等价的几个定义，**离散化后不再等价**——会得到多个互不相等的离散公式。

### 步骤三：品鉴与选择（"No Free Lunch"）

不存在一个完美的离散定义同时保留**所有**光滑性质。必须根据应用场景做权衡：
- 哪个数值更稳？
- 哪个收敛更快？
- 哪个保某种守恒律（能量、面积、拓扑）？

## 四、平面曲线基础（光滑设定）

### 1. 参数化曲线

- **定义**：映射 $\gamma: [0, L] \to \mathbb{R}^2$，连续且光滑。
- **示例**：单位圆 $\gamma(s) = (\cos s, \sin s)$，$s \in [0, 2\pi]$。
- **离散类比**：**分段线性参数化曲线**——一串顶点 $\gamma_0, \gamma_1, \ldots$ 用直线段连接，每个子区间内映射是仿射的。

### 2. 切向量

- **直觉**：沿曲线方向的向量。
- **定义**：单位切向量 $T(s) = \gamma'(s)/\|\gamma'(s)\|$，要求 $\gamma'(s) \neq 0$（曲线不"减速到零再重启"）。
- **弧长参数化**：若 $\|\gamma'(s)\| = 1$，则 $T(s) = \gamma'(s)$，省去归一化。
  - 圆的例子已天然弧长参数化：$\gamma'(s) = (-\sin s, \cos s)$，模长恒为 1。

### 3. 法向量

- **定义**：单位法向量 $N(s)$ 由切向量逆时针旋转 90° 得到。
- **二维变换**：$(x, y) \to (-y, x)$，与切向量正交。
- **方向约定**：内侧 / 外侧都合法，**保持一致**即可——后续的曲率符号、Laplacian 符号都依赖这个约定。

### 4. 光滑曲率

- **直觉**：弯曲剧烈程度。直线 $\kappa = 0$；半径 $r$ 的圆 $\kappa = 1/r$。
- **公式**（弧长参数化下）：
  $$\kappa(s) = \langle N(s), T'(s)\rangle = \langle N(s), \gamma''(s)\rangle$$
- **为什么用点积而不是模长？**
  - 模长 $\|\gamma''(s)\|$ 总是非负，**丢失方向信息**。
  - 点积保留**符号**：正值/负值表示向哪侧弯。
- **等价形式**（用切线角 $\phi$ 表达）：$\kappa(s) = d\phi/ds$，其中 $\phi$ 是切向量与水平方向的夹角。这个一阶形式为离散化提供了关键起点。

## 五、离散曲率：四种推导路径

### 0. 为什么不能直接套用光滑公式？

- **直线段上**：$\gamma'' = 0$，曲率为零。
- **顶点上**：曲线不可导，二阶导数无定义（"无穷大"）。
- **结论**：直接套用光滑公式只会得到 0 和 ∞，毫无信息量。必须找一个**等价但适合离散**的光滑定义。

### 1. 路径 A：转角法（Turning Angle）—— 从积分视角

**思想**：把光滑曲率从"二阶导数"重新解释为"角度变化率 $d\phi/ds$"，再积分。

**关键观察**（微积分基本定理）：
$$\int_a^b \kappa(s)\,ds = \phi(b) - \phi(a)$$

> **总曲率 = 两端切线的角度差**，与曲线内部细节无关。

**离散版本**：对于折线，相邻两条边之间的**外角** $\theta_i$ 直接等于这一段的总曲率：

$$\kappa_i = \theta_i$$

或者按弧长归一化：$\kappa_i = \theta_i / l_i$，其中 $l_i = \tfrac{1}{2}(|e_{i-1}| + |e_i|)$ 是 Voronoi 长度。

**性质**：
- ✅ 完美满足离散 Gauss-Bonnet：闭合曲线 $\sum_i \theta_i = 2\pi$
- ✅ 收敛性好（均匀网格下二阶）
- ✅ 计算 O(1)
- ✅ 综合最优，**工程首选**

### 2. 路径 B：长度变分法

**思想**：光滑曲率等于长度对法向扰动的变分，$\kappa = \delta L / \delta \gamma \cdot N$。

**离散推导**：
1. **单线段长度梯度**：从 $a$ 到 $b$ 的线段，长度 $l = \|b - a\|$ 对 $b$ 的梯度是单位向量 $\nabla_b l = (b - a)/l$。
2. **顶点处总梯度**：移动 $\gamma_i$ 只影响相邻两条边，梯度 = 两个单位向量之和。
3. **三角恒等式化简**：
   $$\nabla_{\gamma_i} L_{\text{total}} = 2\sin(\theta_i/2)\,\mathbf{n}_i$$
   其中 $\mathbf{n}_i$ 沿外角平分线方向。
4. **类比光滑公式** $\nabla L = \kappa N$，反推得：
   $$\kappa_i = 2\sin(\theta_i/2)$$

**性质**：
- ✅ 适合能量最小化、物理仿真
- ⚠️ 当 $\theta \to 0$ 时退化到转角法（$2\sin(\theta/2) \approx \theta$），仅在粗网格上有差异
- ⚠️ 收敛阶在非均匀网格下可能下降

### 3. 路径 C：Steiner 公式（平行曲线偏移）

**光滑形式**：曲线沿法向偏移 $\varepsilon$ 后长度变化为
$$L(\gamma_\varepsilon) = L(\gamma) - \varepsilon \int \kappa\,ds$$

**离散难点**：顶点处法向无定义。**解法**：先把每条边沿其自身法向偏移，再选择如何连接相邻偏移段——**三种连接方式给出三种公式**：

| 连接方式 | 长度变化 | 离散曲率 | 备注 |
| :--- | :--- | :--- | :--- |
| 圆弧（半径 $\varepsilon$，弧度 $\theta_i$） | $-\varepsilon \sum \theta_i$ | $\kappa_i = \theta_i$ | 与转角法一致 |
| 直线 | $-\varepsilon \sum 2\sin(\theta_i/2)$ | $\kappa_i = 2\sin(\theta_i/2)$ | 与变分法一致 |
| 延长相交 | $-\varepsilon \sum 2\tan(\theta_i/2)$ | $\kappa_i = 2\tan(\theta_i/2)$ | 第三种新定义 |

> **关键启示**：Steiner 公式视角揭示了为什么前两种定义会"碰头"——同一个公式从不同路径得到，是它"典范性"的强信号。同时它也产生了第三个候选 $2\tan(\theta_i/2)$。

### 4. 路径 D：密切圆（Osculating Circle）

**光滑定义**：$\kappa = 1/r$，其中 $r$ 是密切圆半径（最佳逼近圆）。

**离散版本**：取连续三点 $\gamma_{i-1}, \gamma_i, \gamma_{i+1}$，三点定一个圆，半径 $r_i$，定义：
$$\kappa_i = \frac{1}{r_i} = \frac{2\sin\theta_i}{|\gamma_{i+1} - \gamma_{i-1}|}$$

**性质**：
- ✅ 几何直观（直接给出曲率半径）
- ❌ 三点近共线时数值不稳定（$r \to \infty$）
- ❌ 噪声敏感
- ❌ 积分不严格等于 $2\pi$

### 5. 四种定义对比表

| 视角 | 离散公式 | 拓扑保持 | 收敛性 | 计算成本 | 稳定性 | 主要用途 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 转角法 | $\kappa_i = \theta_i$ | ✅ 完美 | 好 | 低 | 高 | **默认首选**，平滑、曲率流 |
| 长度变分 | $\kappa_i = 2\sin(\theta_i/2)$ | ✅（与转角等价极限） | 中 | 低 | 中 | 物理仿真、能量法 |
| Steiner（直线延伸） | $\kappa_i = 2\tan(\theta_i/2)$ | — | — | 低 | — | 特定数值场景 |
| 密切圆 | $\kappa_i = 1/r_i$ | ❌ 不严格 | 噪声敏感 | 低 | 低 | 曲率半径可视化 |

## 六、No Free Lunch 定理

### 1. 三个理想性质

类比光滑曲线缩短流 $\partial \gamma / \partial t = \kappa \mathbf{N}$ 的三大性质：

1. **总曲率守恒**：闭合曲线 $\sum \theta_i$ 恒为 $2\pi$。
2. **质心不漂移**：曲线质心位置稳定。
3. **圆是稳态**：所有顶点在一个圆上时，流动只导致均匀缩小，不破坏圆形。

### 2. 四种定义的实测表现

| 定义 | 总曲率 | 质心 | 保圆 |
| :--- | :--- | :--- | :--- |
| 转角法 | ✅（构造性） | ❌ | ❌ |
| 正弦类（变分） | ❌ | ✅ | ❌ |
| Steiner（tan 类） | — | — | — |
| 密切圆 | — | — | ✅ 唯一保圆 |

### 3. No Free Lunch 定理的内容

> **在合理条件下，可以严格证明：不存在一个离散曲率定义能同时满足全部三条性质。**

每个定义只能保住一条。这是离散微分几何（也是数值方法中能量/动量/辛形式守恒）的基本限制。

### 4. 工程含义

**"为工作选合适的工具"**——决策依据：
- 物理仿真追求能量守恒？→ 选变分类
- 控制旋度（如流体）？→ 看具体应用
- 只是降噪？→ 转角法即可
- 设计要保圆度？→ 密切圆
- 要拓扑正确？→ 转角法

**核心方法论**：混合光滑与离散视角能激发新算法——许多优秀算法的形态远超纯离散思维所能得到的。

## 七、应用案例：曲率流（Curvature Flow）

### 1. 光滑曲线缩短流

$$\frac{\partial \gamma(s, t)}{\partial t} = \kappa(s, t)\,\mathbf{N}(s, t)$$

- 每个点沿法向移动，速度等于该点曲率。
- 弯曲剧烈处快速被抹平，最终曲线收缩为一点。

### 2. 离散版本

$$\gamma_i^{t+1} = \gamma_i^t + \tau\,\kappa_i\,\mathbf{N}_i$$

- $\tau$：时间步长
- $\kappa_i, \mathbf{N}_i$：选用某种离散定义

不同 $\kappa$ 的选择直接决定了三大性质中哪一条得以保持。

### 3. 典型用途

| 场景 | 目的 |
| :--- | :--- |
| 几何降噪 | 抹平 3D 扫描噪声，保留整体形状 |
| 曲面参数化 | 把曲面平摊到平面（服装裁片、UV 展开） |
| 拓扑分析 | 让特征边界更清晰，辅助切割模式生成 |

## 八、课程后续路线图

学完曲线曲率这个 toy case 后，课程将进入**网格几何**和**外微积分**两条主线：

1. **组合曲面（Combinatorial Surfaces）**
   - 即网格（Meshes）。
   - 关注**连接性（Connectivity）**与编码它的**数据结构**。

2. **外微积分（Exterior Calculus）/ 微分形式**
   作为后续微分几何的**基础语言**，分三层递进：
   - **外代数（Exterior Algebra）**：线性代数的自然延伸，引入楔积 $\wedge$ 和高阶对象（有向面元、体元）。
   - **微分形式（Differential Forms）**：1-form / 2-form / 3-form，作为流形上的被积量。
   - **外微分与积分**：外微分 $d$ 统一了梯度/旋度/散度，满足 $d^2 = 0$；Stokes 定理 $\int_\Omega d\omega = \int_{\partial\Omega} \omega$。

3. **离散外微积分（DEC）**
   把上述语言落到网格上的最终目标。它是一台"翻译机"，能把光滑几何概念（曲率、Laplacian、测地线）系统地转成可实现的网格算法。

4. **后续应用主题**
   - 曲面曲率、**Laplace-Beltrami 算子 $\Delta$**（统治热扩散、振动、几何流等）
   - 测地线（曲面上的最短路径）
   - 共形几何（保角变换）
   - 同调 / 上同调（拓扑不变量、孔洞分类）

> **学习预期**：基础只需线性代数 + 向量微积分；通过大量可视化和动手练习理解抽象概念，最终能灵活把这些工具用到图形学算法中。

## 九、关键术语速查

| 术语 | 含义 |
| :--- | :--- |
| 微分几何 | 用微积分讨论形状局部性质并连接全局的语言 |
| DDG（离散微分几何） | 用有限长度、角度描述几何性质的离散版本 |
| 流形（Manifold） | 局部像欧氏空间的几何对象；任何 $f(x)=0$ 的解空间 |
| 参数化曲线 | 映射 $\gamma: [0,L] \to \mathbb{R}^n$ |
| 弧长参数化 | $\|\gamma'(s)\| = 1$，使切向量直接等于导数 |
| 切向量 $T$ | 单位化的 $\gamma'$，反映方向 |
| 法向量 $N$ | 切向量旋转 90° 所得的正交单位向量 |
| 曲率 $\kappa$ | 弯曲程度；$\kappa = \langle N, \gamma''\rangle$ 或 $d\phi/ds$ |
| 外角 $\theta_i$ | 折线相邻两边之间的有向夹角 |
| 转角法 | $\kappa_i = \theta_i$，离散曲率的标准定义 |
| 密切圆 | 在某点最佳逼近曲线的圆，半径倒数为曲率 |
| Steiner 公式 | 法向偏移后长度变化与曲率的积分关系 |
| Gauss-Bonnet（离散版） | $\sum \theta_i = 2\pi$ 对闭合简单折线 |
| 曲率流 | $\partial\gamma/\partial t = \kappa N$，几何平滑演化 |
| No Free Lunch | 不存在同时满足所有光滑性质的离散定义 |
| 外代数 / 微分形式 / DEC | 后续课程的几何语言基础 |
