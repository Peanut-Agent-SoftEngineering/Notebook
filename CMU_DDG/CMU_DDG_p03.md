# CMU_DDG_p03

> 主题：组合曲面 —— 单纯形、单纯复形、定向，及其在拓扑数据分析中的应用

## 一、为什么需要单纯复形

- **目标**：DDG 不处理连续曲面，而是处理**离散网格**。**单纯形**（simplex）是构建网格的"原子积木"，**单纯复形**（simplicial complex）则是把这些积木有规则地组合起来。
- **两个理解维度**：
  - **几何单纯复形**：在空间中具体指定元素的位置。
  - **抽象单纯复形**：仅描述元素之间的**连接关系（connectivity）**，是**拓扑空间的一个离散模拟**。
- **关键洞察**：单纯复形是**连接连续数学（凸集、几何）和离散计算（网格、算法）的桥梁**。后续的研究内容包括：定向、流形性质、数据结构（邻接表 / 关联矩阵 / 半边网格）、数据分析应用、胞腔复形与庞加莱对偶。

## 二、几何前置：凸集与凸包

### 1. 凸集（Convex Set）

- **定义**：集合 $S \subseteq \mathbb{R}^n$ 是凸集，当且仅当对任意 $p, q \in S$，连接它们的线段完全在 $S$ 内：
  $$\forall p, q \in S, \{(1-t)p + tq \mid t \in [0,1]\} \subseteq S$$
- **判定**：
  - **判非凸**容易：找到一对点其连线穿出即可（如星形、瓶子、被切口的 blob）。
  - **判凸**需验证所有点对（如三角形、十二面体、完整 blob）。

### 2. 凸包（Convex Hull）

- **直观定义**：包含 $S$ 的**最小**凸集。"用橡皮筋紧紧包裹形状"得到的形状。
- **等价定义**：所有包含 $S$ 的凸集的**交集**。
- **数学公式**：
  $$\mathrm{Conv}(S) = \left\{\sum_i \lambda_i p_i \mid p_i \in S,\ \lambda_i \ge 0,\ \sum_i \lambda_i = 1\right\}$$
  即所有**凸组合**（非负权重、权重和为 1 的线性组合）的集合。
- **示例**：8 个立方体顶点 $\{(\pm 1, \pm 1, \pm 1)\}$ 的凸包是**实心立方体**——逐步连接顶点 → 棱 → 面 → 内部，得到的就是立方体本身（不是外接球，凸包必须"恰好包裹"）。

## 三、单纯形（Simplex）

### 1. 仿射独立性（Affine Independence）

> 仿射独立性是**点**的概念，类比线性代数中**向量**的线性独立性。

- **定义**：点集 $\{p_0, p_1, \dots, p_k\}$ 仿射独立，当且仅当向量集合 $\{p_1 - p_0, p_2 - p_0, \dots, p_k - p_0\}$ 线性独立。
- **直观**：
  - **仿射独立** = 点处于"一般位置"，不退化。
  - **仿射相关** = 点位于一条直线/低维子空间中。
- **退化判据**：
  - 2 个点：不能重合
  - 3 个点：不共线
  - 4 个点：不共面

### 2. 几何定义

> 一个 **k-单纯形** 是 $k+1$ 个仿射独立点的**凸包**，这 $k+1$ 个点称为它的**顶点**。

| 维度 | 顶点数 | 形状 | 退化判据 |
| :--- | :---: | :--- | :--- |
| 0-单纯形 | 1 | 点 | — |
| 1-单纯形 | 2 | 线段 | 两点不重合 |
| 2-单纯形 | 3 | 三角形 | 三点不共线 |
| 3-单纯形 | 4 | 四面体 | 四点不共面 |

- **数学表达**：
  $$\sigma^k = \left\{\sum_{i=0}^k \lambda_i v_i \mid \lambda_i \ge 0,\ \sum_{i=0}^k \lambda_i = 1\right\}$$
- **维度限制**：$d$ 维空间中最高存在 $d$-单纯形（四面体不能嵌入 2D）。

### 3. 升维构造直觉

1. 从 0-单纯形（点 $A$）开始
2. 沿不与 $A$ 重合的方向拉伸 → 1-单纯形（线段 $AB$）
3. 沿不共线方向拉伸 → 2-单纯形（三角形 $ABC$）
4. 沿不共面方向拉伸 → 3-单纯形（四面体 $ABCD$）

每升一维，加入一个**位于现有维度之外**的点。

### 4. 重心坐标（Barycentric Coordinates）

单纯形内任一点 $p$ 都可**唯一**表示为顶点的**凸组合**：
$$p = \sum_{i=0}^{k} t_i\, v_i, \qquad t_i \ge 0,\ \sum_i t_i = 1$$

- 权重 $t_i$ 即 $p$ 的**重心坐标**。
- **示例**：三角形内一点 $p = 0.2\, p_1 + 0.4\, p_2 + 0.4\, p_3$，由于 $p_2, p_3$ 权重大，$p$ 离它们更近。
- **凸包的等价定义**：点集 $S$ 的凸包就是 $S$ 中所有点的**所有凸组合**。
- **应用**：插值、纹理映射、物理模拟。

### 5. 标准 n-单纯形与概率单纯形

- **定义**：在 $\mathbb{R}^{n+1}$ 中
  $$\Delta^n = \left\{(x_0, \dots, x_n) \mid x_i \ge 0,\ \sum_i x_i = 1\right\}$$
- **几何**：顶点位于各坐标轴单位向量处。
- **别名**：**概率单纯形（Probability Simplex）**——非负 + 和为 1 恰好就是概率分布的两条公理。

## 四、单纯复形（Simplicial Complex）

### 1. 面（Face）

- **定义**：单纯形 $\sigma$ 的**面**是由 $\sigma$ 顶点的子集构成的任何单纯形。
- **示例（三角形 $\{P_0, P_1, P_2\}$ 的所有面）**：
  - 顶点（0-面）：$\{P_0\}, \{P_1\}, \{P_2\}$
  - 边（1-面）：$\{P_0, P_1\}, \{P_1, P_2\}, \{P_0, P_2\}$
  - 自身（2-面）：$\{P_0, P_1, P_2\}$
- **技术细节**：空集 $\emptyset$ 形式上是 $-1$ 维单纯形，纯技术性概念，理解上可忽略。

### 2. 几何单纯复形：两条规则

集合中的每个元素是几何单纯形，且必须满足：

1. **规则一（交的封闭性）**：任意两个单纯形的交，必须是它们共同的一个**整面**（或为空）。
   - 不允许两个三角形仅相交于半条边、或一段非完整边。
2. **规则二（面的完整性 / 闭包性质）**：若 $\sigma$ 在复形中，则 $\sigma$ 的**所有面**也在复形中。
   - 包含一个四面体 → 必须包含其 4 个三角形面、6 条边、4 个顶点。

### 3. 抽象单纯复形（Abstract Simplicial Complex）

剥离几何，只保留组合结构：

> $S$ 是一个集合的集合。若对每个 $\sigma \in S$，$\sigma$ 的所有子集也都在 $S$ 中（**向下封闭**），则 $S$ 是抽象单纯复形。
> $$\forall \sigma \in S,\ \tau \subseteq \sigma \implies \tau \in S$$

- **抽象 k-单纯形**：大小为 $k+1$ 的集合 $\sigma$。
- **维度公式**：$\dim(\sigma) = |\sigma| - 1$
- **几何 vs 抽象**：每个几何单纯复形对应一个抽象单纯复形；但同一个抽象单纯复形可有多种几何实现。

### 4. 单纯复形 vs 拓扑空间

| 不包含的信息 | 包含的信息 |
| :--- | :--- |
| 顶点坐标 | 顶点间连接关系 |
| 边长 / 面积 / 角度 | 哪些顶点构成边/三角形/四面体 |

抽象单纯复形提供了一张**"连接蓝图"**，不依赖几何形状或嵌入。

### 5. 示例

**示例 1（图 = 1 维抽象单纯复形）**：无向图 $G = (V, E)$，0-单纯形 = 顶点，1-单纯形 = 边。

**示例 2（符号集合验证）**：
$$S = \{\emptyset,\ \{1\}, \{2\}, \{\heartsuit\}, \{\smiley\},\ \{1,2\}, \{2,\heartsuit\}, \{\heartsuit,1\}, \{1,\smiley\}, \{2,\smiley\}, \{\heartsuit,\smiley\},\ \{1,2,\heartsuit\}\}$$
检查最高维元素 $\{1,2,\heartsuit\}$ 的所有子集都在 $S$ 中 → $S$ 是有效的抽象单纯复形。可绘制为 $\{1,2,\heartsuit\}$ 构成的三角形 + $\smiley$ 通过边与其他三个顶点相连。

## 五、解剖操作：Closure / Star / Link

为了精确描述网格的局部结构，定义三种基本集合操作：

### 1. 闭包（Closure）

- **定义**：包含给定集合的**最小单纯复形**。
- **作用**：把所有缺失的低维面补齐。
- **例**：给定一个三角形 + 一条不相连的线段，闭包要补全三角形的 3 条边、3 个顶点，以及线段的 2 个顶点。
- $\text{Cl}(S)$：找包含 $S$ 的最小复形。

### 2. 星（Star）

- **定义**：所有**包含**给定子集的单纯形的集合。
  $$\text{St}(A) = \{\tau \in K \mid A \subseteq \tau\}$$
- **顶点 $v$ 的星**：包含 $v$ 的所有边、所有三角形等。
- **注意**：星**不包含**仅与 $v$ 相邻、但本身不含 $v$ 的边（即外圈边环）。

### 3. 链环（Link）

- **定义**：星的闭包减去星本身——剥去填充，留下"边界一圈"。
  $$\text{Lk}(A) = \text{Cl}(\text{St}(A)) \setminus \text{St}(A)$$
- **顶点 $v$ 的 link**：环绕 $v$ 的一圈边构成的环（不含 $v$ 自己）。
- **用途**：判断 $v$ 是否处于盘的内部还是边界，是后续判定流形条件的核心工具。

> **应用场景**：想说"与某顶点相邻的所有三角形"时说"该顶点的 star"；想说"环绕该顶点的边环"时说"该顶点的 link"。

## 六、网格的标准记号

### 1. 三角网格符号

$$K = (V, E, F)$$
- $V$ = 顶点集合（Vertices）
- $E$ = 边集合（Edges）
- $F$ = 面集合（Faces）

### 2. 术语警告

- **通用单纯形理论中**："face" 指任何低维子单纯形（包括边、顶点）。
- **网格上下文中**：$F$ 特指**最高维面**（三角形本身）。

### 3. 命名来源

- $K$ 来自德文 **Komplex**（复形）。

## 七、定向（Orientation）

### 1. 动机

- 仅有顶点/边/面的拓扑结构还不够。当在网格上做几何计算（法向、曲率、积分）时，需要给每个单纯形赋予**方向**。
- **生活类比**：单行道（边的定向）、有方向的盒子（面的定向）。
- **物理类比**：液晶中的小粒子有方向但没"优势方向"。
- **关键性**：**积分符号依赖定向**——边从 $a$ 到 $b$ 的积分与从 $b$ 到 $a$ 的积分符号相反。

### 2. 单纯形的定向

| 维度 | 定向方式 | 等价类 |
| :--- | :--- | :--- |
| 0-单纯形 | 唯一排列 | **永远正定向**（无正负之分） |
| 1-单纯形 | 有序对 $(a,b)$ vs $(b,a)$ | 两种 |
| 2-单纯形 | 顶点环绕顺序 | 两种（顺时针/逆时针） |
| k-单纯形 | 有序 $(k+1)$-元组 | 两种 |

**通用规则**：

- 一个**有向 k-单纯形**由一个有序 $(k+1)$-元组 $(v_0, v_1, \dots, v_k)$ 给出。
- 两个元组**等价**（同定向）当且仅当通过**偶置换**互相转换。
- 通过**奇置换**转换则定向**相反**。
- **循环移位**是偶置换的特例：$(a,b,c) \sim (b,c,a) \sim (c,a,b)$（同向）；$(a,c,b) \sim (c,b,a) \sim (b,a,c)$（反向）。

**置换数量关系**：
$$(k+1)!\ \text{种顶点排列} \implies \frac{(k+1)!}{2}\ \text{种元组表示同一定向}\quad (k \ge 1)$$

**四面体（3-单纯形）示例**：顶点 1,2,3,4
- 正定向：$[1,2,3,4],\ [1,3,4,2],\ [1,4,2,3]$ 等
- 负定向：$[1,2,4,3],\ [3,1,4,2],\ [1,3,2,4]$ 等
- 简记：$+\sigma = [1,2,3,4]$，$-\sigma = [1,2,4,3]$

### 3. 定向单纯复形

- **定义**：单纯复形中每个单纯形都被赋予一个定向。
- **关键性质**：方向选择**完全任意**——一个三角形是顺时针，并不要求它的边也顺时针；每个单纯形独立选择。

### 4. 相对定向（Relative Orientation）

> 两个有向单纯形如果它们交集中的**最大维面**具有**相反方向**，则它们具有**相同的相对定向**。

- **判定标准**：对两个共享一条边的三角形，**检查这条共享边在两个三角形中的方向是否相反**：
  - **方向相反** → 相同的相对定向（"流入一个，流出另一个"，连续）
  - **方向相同** → 不同的相对定向

**示例**：

- 三角形 $(a,b,c)$：边 $bc$ 的方向是 $b \to c$
- 三角形 $(d,c,b)$：边 $bc$ 的方向是 $c \to b$
- 两者方向相反 → **相同的相对定向**（局部都是同向旋转）

### 5. 可定向性（Orientability）

**关键问题**：能否给整个复形的**所有**面分配定向，使任意两个共享边的相邻面都具有相同的相对定向？

| 形状 | 是否可定向 |
| :--- | :--- |
| 球面、圆柱体 | ✅ 可定向 |
| 莫比乌斯带 | ❌ 不可定向（拓扑扭曲，全局会矛盾） |

> **重要启示**：相对定向这种**局部信息**实际上揭示了形状的**全局拓扑性质**——这是 DDG 的核心洞察之一。

> **后续展望**：相对定向是**离散外微积分（DEC）** 的基础，它会决定边界算子的符号：
> $$\partial [v_0, \dots, v_k] = \sum_{i=0}^{k} (-1)^i [v_0, \dots, \hat{v_i}, \dots, v_k]$$

## 八、应用案例：拓扑数据分析（TDA）

### 1. 核心思想：从点云到拓扑

- **目标**：仅通过**连通性**（不依赖几何信息），分析点云数据的内部结构。
- **关键工具**：用单纯复形将离散数据转化为可分析的拓扑结构。

### 2. 持久同调（Persistent Homology）

**算法（球膨胀过程，又称 Vietoris–Rips 复形）**：

```
输入：点集 P = {p_1, ..., p_n}
对半径 r = r_0, r_1, ..., r_k（递增）：
  - 两球重叠（距离 ≤ 2r） → 添加 1-单纯形（边）
  - 三球两两重叠 → 添加 2-单纯形（三角形）
  - 更多球重叠 → 添加更高维 d-单纯形
```

**演化现象**：

- 半径很小：所有点孤立，多个连通分量。
- 半径中等：内部正确连接出现。
- 半径很大：错误连接出现，孔洞被填满，不同字母被连一起。

> **结论**：没有哪个单一半径"正确"——必须**跟踪不同特征的"出生"与"死亡"**。

### 3. 持久性图（Persistence Diagram）

- 每个拓扑特征（连通分量、环）作为一个点：
  - **X 轴**：出生半径
  - **Y 轴**：死亡半径
- **性质**：所有点在对角线 $y = x$ 上方（先生才能死）。
- **解读**：
  - **远离对角线** → 寿命长 → 真实信号
  - **靠近对角线** → 寿命短 → 噪声

**核心公式**：

$$\text{持久性（Persistence）} = \text{死亡时间} - \text{出生时间}$$

### 4. 应用全景

| 应用领域 | 输入 | 核心洞察 | 实际意义 |
| :--- | :--- | :--- | :--- |
| **材料科学（玻璃）** | 原子坐标（3D 点云） | 持久条形图中出现非常显著的拓扑特征，在随机排列中缺失 | 量化定义"玻璃态"的数学工具 |
| **几何识别（指尖）** | 曲面上一点的局部邻域 | 球膨胀过程中遇到的拓扑事件序列（凹陷、自切、覆盖）形成局部签名 | 多尺度局部唯一签名，用于点对点匹配 |
| **生物网络（脑神经）** | 神经元连接图 | 不同类别（健康 vs 病变）的条形码在长度和复杂度上有差异 | 区分病理状态的非监督方法（阿尔茨海默症、癫痫等） |
| **其他** | — | — | 社交网络分析、癌症识别、运动员分类、图像压缩 |

> **一句话总结**：持续同调将任意维度的点云数据通过拓扑学的"镜头"，转化为**抗噪声、可比较的"指纹"**。
