# CMU_DDG_p20

> 主题：离散曲率的"变分观"——通过对几何量求导建立标量曲率与向量曲率法线之间的桥梁，并把这套机器用于曲率流（平均曲率流、Willmore 流等）。

---

## 一、整体观与变分观的桥梁

### 1. 两种视角的对立统一

| 视角 | 方法 |
| :--- | :--- |
| **整体观（前节课）** | 对偶单元上积分曲率法线，得到向量曲率量 |
| **变分观（本节）** | 对曲面求导（变分），得到曲率 |

**核心发现**：**标量曲率的变分（梯度）等于向量曲率法线**——两种视角完全相通。

### 2. 标量量与向量量的对照

#### 标量曲率量
| 量 | 定义 | 关联对象 |
| :--- | :--- | :--- |
| 顶点高斯曲率 $K_i$ | $2\pi-\sum\theta$（角度缺陷） | 顶点 |
| 边平均曲率 $H_e$ | $\tfrac{1}{2}l_{ij}\phi_{ij}$ | 边 |
| 三角形面积 $A_t$ | 标准面积 | 三角形 |
| 四面体体积 $V_t$ | 顶点+三角形构成的四面体 | 体积分解单元 |

#### 向量曲率量
| 量 | 离散形式 | 权重 |
| :--- | :--- | :--- |
| 面积矢量 $\mathbf{A}$ | $\sum f_i\times f_j$（顶点星位置叉积之和） | — |
| 平均曲率法线 $\mathbf{H}$ | $\tfrac{1}{2}\sum(\cot\alpha+\cot\beta)(f_i-f_j)$ | 余切权重 |
| 高斯曲率法线 $\mathbf{K}$ | $\sum(\phi_{ij}/l_{ij})(f_j-f_i)$ | 二面角/边长 |

## 二、主曲率与"位置不一致"的处理

### 1. 光滑公式

由 $H = (\kappa_1+\kappa_2)/2$、$K = \kappa_1\kappa_2$：
$$\kappa_{1,2} = H \pm \sqrt{H^2 - K}$$

### 2. 离散设置的难题

| 问题 | 影响 |
| :--- | :--- |
| $K$ 定义在**顶点**上 | 顶点级数据 |
| $H_e$ 定义在**边**上 | 边级数据 |
| 量纲不一致 | 无法直接代入光滑公式 |

### 3. 顶点平均曲率：从"边"转到"点"

通过对偶胞元 $C_i$ 上积分：
$$H_i = \frac{1}{4}\sum_j l_{ij}\phi_{ij}$$

- 这是顶点邻域内**总平均曲率**（积分量），非"点"上的平均曲率。

### 4. 归一化：标量值 vs 积分量

- $H_i$ 与 $K_i$ 都是**积分量**（2-form 在区域上的积分结果）。
- 经典公式 $\kappa_{1,2} = H\pm\sqrt{H^2-K}$ 中的 $H, K$ 是**点值**（0-form）。
- **必须先除以对偶面积** $A_i$：
  $$\hat{H}_i = H_i/A_i,\quad \hat{K}_i = K_i/A_i$$
  再代入光滑公式得到主曲率。

## 三、几何求导的方法论

### 1. 黄金法则：方向 + 幅度

面对几何量对顶点位置的导数，**不要**机械求偏导。改为两步：
1. **方向**：该量变化最快的方向（梯度方向）。
2. **速率**：在该方向上单位移动的变化率（梯度大小）。

**优势**：
- 计算简洁，避开冗长代数。
- 暴露问题的几何本质。
- 结果数值稳定。

### 2. 案例 A：角度对端点的梯度

求 $\angle BAC = \theta$ 关于顶点 $A$ 的梯度。

#### 方向
- 沿 $AB$ 方向移动 $A$ 不改变夹角。
- 只有**垂直于 $AB$** 的移动才改变夹角。
- 方向 $\mathbf{u} = \mathbf{d}^{\perp}$，其中 $\mathbf{d} = (B-A)/\|B-A\|$。

#### 幅度
- 沿垂直方向移动相当于在以 $B$ 为中心、半径 $\|B-A\|$ 的圆上运动。
- 角速度 = $1/r = 1/\|B-A\|$。

#### 公式
$$\nabla_A\theta = \frac{1}{\|B-A\|}\left(\frac{B-A}{\|B-A\|}\right)^\perp$$

数值稳定，物理意义清晰。

### 3. 案例 B：三角形面积对顶点的梯度

求三角形面积关于顶点 $P$ 的梯度。

#### 方向
- 沿底边方向移动 $P$ 不改变高，面积不变。
- 沿底边法线方向移动改变高度。
- 在 3D 中，垂直于三角形平面的移动只是旋转，**不改变面积**——故梯度位于三角形平面内。

#### 幅度
- 高度改变 1，面积变化 $\tfrac{1}{2}\times\text{底长}$。

#### 优雅向量公式
$$\nabla_P A = \tfrac{1}{2}(\mathbf{n}\times\mathbf{e})$$

- $\mathbf{n}$：三角形单位法向量。
- $\mathbf{e}$：与 $P$ 相对的底边向量。

### 4. 案例 C：边长对端点的梯度

$$\nabla_{f_i}l_{ij} = \frac{f_i - f_j}{l_{ij}} = \hat{u}_{ij}$$

- **方向**：从 $f_j$ 指向 $f_i$ 的单位向量。
- **幅度**：1（沿连接方向移动单位距离，边长变化单位距离）。

### 5. 求导策略对比

| 策略 | 方法 | 精度 | 速度 | 灵活性 | 适用 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 手工求导 | 手动推导并编码 | 极高 | 极快 | 极低 | 固定热点函数 |
| 数值微分 | $\Delta f / \Delta x$ | 低 | 慢 | 极高 | 黑盒函数临时方案 |
| 自动微分 | 代码转换 + 链式法则 | 高 | 高 | 中 | 频繁修改的复杂函数 |

#### 数值微分的陷阱
- $\epsilon$ 太大→近似误差；太小→浮点精度问题，输出无变化导致导数为零。
- 数值不稳定。

#### 自动微分的局限
- 需要专门的库或类型，无法直接用于黑盒函数。
- **不知道领域知识**：例如不知道边向量之和为零（闭合性），或某些总和恒等于 $2\pi\chi$，会冗余计算导数最终为零的项。**手动推导利用领域知识可极大简化**。

## 四、几何量梯度的"链式关系"

### 1. 光滑曲面下的导数链

| 标量量 | 梯度 | 几何含义 |
| :--- | :--- | :--- |
| 体积 $V$ | $n\,dA$（沿法向，常数权重） | 沿法向膨胀最快 |
| 表面积 $A$ | $2H\,n\,dA$（与平均曲率成正比） | 沿法向，速度正比 $H$ |
| 总平均曲率 $\int H\,dA$ | $K\,n\,dA$（与高斯曲率成正比） | 沿法向，速度正比 $K$ |
| 总高斯曲率 $\int K\,dA$ | $\mathbf{0}$ | 拓扑不变量，不可变 |

链：
$$V \xrightarrow{\nabla\propto 1} A \xrightarrow{\nabla\propto H} \int H \xrightarrow{\nabla\propto K} \int K \xrightarrow{\nabla = 0}$$

最后一项归零，是 Gauss-Bonnet（**Theorema Egregium** 的整体形式）的体现。

### 2. 体积公式推导

#### 切片为无穷小金字塔
- 顶点：任选 $p$；底面：曲面上的面元 $da$。
- 高度 $h = (f-p)\cdot n$。
- 金字塔体积 = $\tfrac{1}{3}bh = \tfrac{1}{3}(f-p)\cdot n\,da$。

#### 总积分
$$V = \tfrac{1}{3}\int(f-p)\cdot n\,da = \tfrac{1}{3}\int f\cdot n\,da - \tfrac{p}{3}\cdot\int n\,da$$

闭合曲面上 $\int n\,da = 0$（Stokes），故：
$$V = \tfrac{1}{3}\int f\cdot n\,da$$

与参考点 $p$ 无关。

#### 离散版本
$$V_{\text{离散}} = \tfrac{1}{6}\sum_{\triangle ijk}f_i\cdot(f_j\times f_k)$$

仅依赖位置和法线，无导数项——**安全的离散化**（不像曲率定义那样需要导数信息）。

### 3. 体积梯度：直接给出顶点法线

#### 离散公式
$$\nabla_{f_i}V = \sum_{\triangle ijk\ni i}\tfrac{1}{3}f_j\times f_k$$

- 这恰好等于**顶点星周围的面积矢量**（前节课从对偶胞元积分得到的结果）。

#### 等价推导：四面体求和
- 离散体积 = 有符号四面体体积之和（每个 = 一个三角形 + 参考点）。
- 顶点对单个四面体的梯度方向 = 使该四面体体积增加最快的方向。
- 求和后与上面的公式一致。

### 4. 表面积梯度 → 余切公式

#### 单个三角形
$$\nabla_{f_i}A_T = \tfrac{1}{2}\mathbf{n}\times(f_j - f_k)$$

#### 全局求和并化简
对所有相邻三角形求和后用余切重写：
$$\boxed{\nabla_{f_i}A = \tfrac{1}{2}\sum_{j\in N(i)}(\cot\alpha_{ij}+\cot\beta_{ij})(f_i - f_j)}$$

这正是**离散平均曲率法线在对偶胞元上的积分**——余切公式。

### 5. 总平均曲率梯度 → 高斯曲率法线

#### Steiner 多项式视角
$$H_{\text{tot}} = \sum_{\text{边}}l_{ij}\phi_{ij}$$

#### Schläfli 公式（关键工具）
对封闭多面体的任意变形：
$$\sum_{\text{边}}l_{ij}\,\delta\phi_{ij} = 0$$

它是离散 Gauss-Bonnet 的微分版本。

#### 推导

$$\nabla_{f_i}H_{\text{tot}} = \sum_{j\in N(i)}\big[(\nabla_{f_i}l_{ij})\phi_{ij} + l_{ij}(\nabla_{f_i}\phi_{ij})\big]$$

由 Schläfli，第二项在所有接触 $i$ 的边上求和为零。剩下：
$$\nabla_{f_i}H_{\text{tot}} = \sum_{j\in N(i)}\phi_{ij}\,\hat{u}_{ij}$$

代入 $\hat{u}_{ij} = (f_i - f_j)/l_{ij}$：
$$\boxed{\nabla_{f_i}H_{\text{tot}} = \sum_{j\in N(i)}\frac{\phi_{ij}}{l_{ij}}(f_i - f_j)}$$

这正是**离散高斯曲率法线**。

### 6. 总高斯曲率梯度恒为零

$$\sum_i\Omega_i = 2\pi\chi$$

是拓扑不变量，故对任何顶点位置：
$$\nabla_{f_i}\!\left(\sum\Omega_j\right) = 0$$

### 7. 离散梯度链总结

| 标量量 | 梯度（向量曲率量） |
| :--- | :--- |
| 封闭体积 $V$ | 面积矢量 $\mathbf{A}$ |
| 总表面积 $A$ | 平均曲率法线 $\mathbf{H}$（余切公式） |
| 总平均曲率 $H_{\text{tot}}$ | 高斯曲率法线 $\mathbf{K}$（二面角/边长） |
| 总高斯曲率 $\sum\Omega_i$ | $\mathbf{0}$ |

链结构与光滑情况完全一致；权重从余切换成"二面角/边长"反映高斯曲率与平均曲率的角色差异。

### 8. 离散曲率框架的全景

#### 离散曲率的两种来源

**标量曲率**：通过"球软化多面体" → 积分得 $K$、$H$ 等。

**向量曲率**：积分两种基本形式：
- 面积形式：$df\wedge df$。
- 曲率形式：$df\wedge dn$（$H$）、$dn\wedge dn$（$K$）。

#### 三大支撑定理
- 离散 Gauss-Bonnet：$\sum\Omega_i = 2\pi\chi$。
- Schläfli 多面体公式：$\sum l_{ij}\delta\phi_{ij} = 0$。
- 微分 Steiner 多项式：$V(\epsilon) = V_0 + A_0\epsilon + \tfrac{1}{2}H\epsilon^2 + \tfrac{1}{3}K\epsilon^3$。

#### 通用性
- 自然推广到 $n$ 维流形。
- 处理分段光滑、含折痕、退化情况。
- 计算简单（仅三角形网格上的线性公式）——适合实时图形管线。

## 五、曲率流：能量最小化的应用

### 1. 通用框架：变分梯度下降

#### 连续形式
$$\frac{\partial f}{\partial t} = -\frac{\delta E}{\delta f}$$

#### 离散迭代
$$f_i^{k+1} = f_i^k - \tau\,\nabla E(f_i^k)$$

$\tau$ 为时间步长。

### 2. 法向流（基于体积能量）

- 能量：$E = V$。
- 梯度：面积矢量 $\mathbf{A}_i$。
- 迭代：$f_i \leftarrow f_i \pm \tau\,\mathbf{A}_i$。

#### 效果
- 减号（收缩流）：曲面**收缩并皱缩**——不光滑。
- 加号（膨胀流）：曲面像吹气球。
- 用途：建模膨胀/收缩，但**不适合平滑**。

### 3. 平均曲率流（基于面积能量）

- 能量：$E = \int dA$。
- 梯度：平均曲率法线 $\mathbf{H}$（余切公式）。
- 迭代：
  $$f_i^{k+1} = f_i^k - \tau\,\mathbf{H}_i$$

#### 效果
- 标准平滑流：去除噪声，趋向极小曲面（如肥皂泡）。
- **奇异退化**：长时间演化下细窄区域被捏断，形成骨架状结构；网格的数值稳定性允许继续演化（光滑设定下流会停止定义）。
- 最终面积趋向于零。

#### 应用
1. **3D 模型骨架提取**：利用退化骨架特性。
2. **平台问题（Plateau Problem）**：求边界给定的最小面积曲面（皂膜形状）。
   - 实验观察：两圆柱形电线浸入皂液，肥皂膜会自动找到最小面积配置；某些情况下分裂成两个不相交圆盘（总面积更小）。

### 4. 高斯曲率流（基于总平均曲率能量）

- 能量：$E = H_{\text{tot}}$。
- 梯度：高斯曲率法线 $\kappa\mathbf{n}$（公式 $\phi_{ij}/l_{ij}\cdot e_{ij}$）。
- 物理直觉："滚石下山"——尖角（高斯曲率大处）被先磨掉，凸面趋向球形。
- **局限**：仅对凸面有效；非凸时 $K$ 可能为负，无法实现通用平滑。

### 5. 平均曲率流的陷阱

#### 问题：平均曲率有符号
- 最小化"总平均曲率" 不是平滑，而是让 $H$ 趋向 $-\infty$。
- 凹面区域比平面更能"降低"总和。
- 结果：曲率绝对值反而增大，曲面"爆炸"。

#### 关键辨析
不是离散化缺陷，是**数学模型本身**的问题——光滑曲面也会扭曲而非平滑。

### 6. Willmore 能量：真正的平滑

#### 定义
$$\mathcal{W} = \int_M H^2\,dA$$

或等价的（差一个拓扑常数）：
$$\mathcal{W} = \int_M(H^2 - K)\,dA = \int H^2\,dA - 2\pi\chi$$

#### 关键改进
平方使**正曲率与负曲率都被惩罚**——模型真正"不喜欢"任何弯曲，推动曲面趋于平坦或最小曲面。

#### 离散化
$$\mathcal{W}_{\text{离散}} \approx \sum_v\frac{\|\mathbf{H}_v\|^2}{A_v}$$

**为什么除以面积？** 平均曲率向量是对偶胞元上的积分量 $\sim H\cdot A$，故 $\|\mathbf{H}_v\|^2 \sim H^2 A^2$；除以 $A_v$ 抵消一个面积因子，恢复对 $dA$ 的积分。

#### 演化效果
1. 抹平细节：高频细节被快速消除。
2. 保留体积：粗糙形状基本保持。
3. 最终极限：任意拓扑曲面收敛到圆球。

#### 物理对应
- **表面张力作用下的液滴**：水滴受表面张力收缩为完美球形。
- **生物膜**：细胞膜、胆脂膜的弯曲能。
- **弹性板**：扁平弹性板的形变。

#### 研究方向
- 处理平均曲率流奇异性（拓扑变化、收缩成点、分裂、合并）。
- 用平均曲率法线构造离散 Willmore 能量。
- 改进版离散 Willmore 能量更好捕捉网格几何特征、避免退化。
