# CMU_DDG_p10

> 主题：把外微积分翻译到网格上 —— 离散微分形式（链与上链）、Whitney 基函数与离散外微积分（DEC）的核心框架。

## 一、为什么需要离散：从 $d\alpha = \omega$ 谈起

### 1. 一个看似简单的逆问题

在 $\mathbb{R}^2$ 中给定常值 2-形式 $\omega = dx\wedge dy$，求 1-形式 $\alpha$ 使 $d\alpha = \omega$。

**推导**：设 $\alpha = u\,dx + v\,dy$。

$$d\alpha = du\wedge dx + dv\wedge dy = \left(\frac{\partial u}{\partial y}\,dy\wedge dx\right) + \left(\frac{\partial v}{\partial x}\,dx\wedge dy\right) = \left(\frac{\partial v}{\partial x} - \frac{\partial u}{\partial y}\right) dx\wedge dy$$

（用到 $d(dx)=d(dy)=0$，$dx\wedge dx=0$，$dy\wedge dx = -dx\wedge dy$。）

令其等于 $\omega$ 得：

$$\frac{\partial v}{\partial x} - \frac{\partial u}{\partial y} = 1$$

一个简单解：$u = -\tfrac{1}{2}y$，$v = \tfrac{1}{2}x$，所以

$$\alpha = \tfrac{1}{2}x\,dy - \tfrac{1}{2}y\,dx$$

### 2. 反直觉的发现

- 输入是**处处均匀**的常值面积元，输出却是**绕原点的旋转向量场**（涡）。
- 解不唯一：通解形如 $\alpha = (\tfrac{1}{2}x+a)\,dy - (\tfrac{1}{2}y+b)\,dx$，涡心可平移。
- **教训**：即使这种最简单的微分方程也已暗藏几何结构——外微分 $d$ 不是可逆的、直观的算子。

### 3. 为何必须走向计算

- 一旦 $\omega$ 不是常值，或换成更复杂的算子，闭式求解立即失效。
- 当域从 $\mathbb{R}^2$ 变成"bunny mesh"等任意几何体时，解析解不存在。
- 任何物理仿真、几何处理都需要在**离散网格**上近似求解 PDE。
- **结论**：必须开发能在网格上执行的算法 —— 这就是 DEC 的舞台。

## 二、DEC 的核心思路：连续到离散的三大替换

| 连续对象 | 离散对应物 | 说明 |
| :--- | :--- | :--- |
| 域 (Domain) | **网格 / 定向单纯复形** | 例如把 bunny 替换为四面体网格 |
| 微分形式 (Differential Form) | **离散微分形式** | $k$-形式存为定向 $k$-单纯形上的数值 |
| 微分算子 (Exterior Derivative, Hodge $\star$, ...) | **稀疏矩阵** | 例：外微分 = 带符号关联矩阵 |

### 三大基础矩阵

1. **对角矩阵**：存几何量（面积、长度、长度比）。
2. **带符号关联矩阵**（signed incidence matrix）：编码顶点-边、边-面的拓扑邻接，元素为 $\pm 1$ 表示定向。
3. **组合矩阵**：通过乘法得到所有复杂算子。

### 微分算子的统一分解

借助标准向量微积分等价：
- **梯度** = $d$ 作用于 0-形式
- **旋度** = $\star \circ d$ 作用于 1-形式
- **散度**、**Laplacian** 也是类似分解

即使在弯曲流形上，Laplacian 这类原本复杂的算子，最终也只是几个稀疏矩阵的乘积。

### 标准工作流

1. 加载网格
2. 构建基础矩阵（面积/长度对角阵 + 带符号关联矩阵）
3. 矩阵相乘组合出目标算子
4. 求解线性系统
5. 处理 / 可视化结果

### 数学根基：链与上链

- **链 (Chain)**：网格中 $k$-单纯形的形式线性组合，代表"被测量"的区域。
- **上链 (Cochain)**：定义在链上的线性函数，代表"测量者"——即离散微分形式。
- 这正是**单纯同调**的标准框架。

### 离散化 vs 插值

- **离散化（连续 → 离散）**：从连续对象采样有限测量值。会丢失信息。
- **插值（离散 → 连续）**：根据采样重建一个与样本一致的连续对象。是一种**假设**。

> **务实视角**：插值出的连续对象不必等于原始对象，但**必须与采样测量一致**。

## 三、离散微分 $k$-形式：积分而非采样

### 1. 朴素点采样为何失败

对连续 $k$-形式只在顶点处取值，得到一堆孤立数据，**没有结构**：无法定义外微分（缺乏点之间的连接关系），不能求解 PDE。

### 2. 正确做法：De Rham 映射

**离散微分 $k$-形式**就是一个把每个**定向 $k$-单纯形 $\sigma$** 映射到一个实数的函数：

$$\boxed{\hat{\omega}(\sigma) = \int_{\sigma} \omega}$$

形式的次数必须与单纯形的维度严格匹配：

| 形式阶 | 积分对象 | 几何意义 |
| :---: | :--- | :--- |
| 0-形式 | 顶点（0-单形） | 顶点处取值（采样退化） |
| 1-形式 | 边（1-单形） | 沿边的环量 |
| 2-形式 | 面（2-单形） | 穿过面的有向通量 |
| 3-形式 | 体（3-单形） | 体积分 |

### 3. 优点对比

| 方法 | 数据单元 | 优势 | 劣势 |
| :--- | :--- | :--- | :--- |
| 点采样 | 顶点 | 简单 | 无结构，无法定义算子 |
| **基于积分** | $k$-单形 | 保留积分结构；可定义 $d$、$\star$；可求解 PDE | 需要网格；信息有限 |

## 四、离散积分的实战

### 1. 0-形式：顶点取值

$\phi(x,y) = x^2 + y^2 + \cos(4x+y)$，顶点 $v=(1,-1)$：

$$\hat{\phi}(v) = \phi(1,-1) = 1+1+\cos 0 = 3$$

### 2. 1-形式沿一条边的积分（流程）

被积形式 $\alpha$，边 $e$ 端点 $P_0,\,P_1$。

1. **边长**：$L = \|P_1 - P_0\|$。
2. **单位切向量**：$\mathbf{T} = (P_1-P_0)/L$。
3. **$\alpha$ 作用于 $\mathbf{T}$**：得到沿边的标量函数 $\alpha(\mathbf{T})$。
4. **数值积分**（求积法则：中点 / 梯形 / 高斯）：

$$\int_e \alpha \approx L \cdot \overline{\alpha(\mathbf{T})}$$

> 实际工程几乎从不做闭式积分，都用数值采样平均后乘以 $L$。

### 3. 完整算例：$\alpha = xy\,dx - x^2\,dy$，$P_0=(-1,2),\ P_1=(3,1)$

- $P_1-P_0 = (4,-1)$，$L=\sqrt{17}$，$\mathbf{T}=(4/\sqrt{17},\,-1/\sqrt{17})$。
- $\alpha(\mathbf{T}) = xy\cdot\tfrac{4}{\sqrt{17}} + (-x^2)\cdot\tfrac{-1}{\sqrt{17}} = \tfrac{4xy + x^2}{\sqrt{17}}$。
- 弧长参数化 $p(s) = P_0 + s\mathbf{T}$，$s\in[0,L]$，代入并整理为标准黎曼积分。
- 端点采样平均法：$\alpha(\mathbf{T})|_{P_0} = -7/\sqrt{17}$，$\alpha(\mathbf{T})|_{P_1} = 21/\sqrt{17}$，平均 $7/\sqrt{17}$，乘 $L=\sqrt{17}$ 得 $\boxed{7}$。

### 4. 更小的例：$\omega = 2\,dx$ 在单位正方形上

| 边 | 切向量 $T$ | $\omega(T)$ | 积分 |
| :--- | :--- | :--- | :--- |
| 底边 $e_1$（水平） | $\partial/\partial x$ | $2$ | $2$ |
| 左边 $e_2$（竖直） | $\partial/\partial y$ | $0$ | $0$ |
| 对角线 $e_3$ | $(1/\sqrt 2)(\partial_x+\partial_y)$ | $\sqrt{2}$ | $\sqrt{2}\cdot\sqrt{2} = 2$ |

**洞察**：$e_1$ 和 $e_3$ 长度不同却同值——$\omega = 2\,dx$ 测量的是**水平跨度**而非路径长度。两条边都覆盖了 $x\in[0,1]$。

## 五、定向：符号即信息

### 1. 反转方向 ⇒ 符号取反

把 $\mathbf{T}$ 换成 $-\mathbf{T}$，上面的算例就从 $+7$ 变成 $-7$。原因：微分形式是**反对称**的：

$$\alpha(-\mathbf{T}) = -\alpha(\mathbf{T})$$

类比微积分基本定理：

$$\int_a^b df = \phi(b)-\phi(a) = -\int_b^a df$$

### 2. 物理与算法意义

- **流体仿真的通量**：定向告诉你测的是流入还是流出。
- **离散版守恒律**：网格中相邻单元在共享边/面上方向相反，全局求和时正确**抵消**——这正是离散斯托克斯成立的前提。
- **代码注意**：必须**全局一致地**为边、面分配定向（如"小索引顶点指向大索引"）。忽略定向 → 全是符号错误。

### 3. 反对称与置换性质

对离散 $k$-形式 $\beta$，定义在定向 $k$-单纯形 $[v_{i_0}, \ldots, v_{i_k}]$ 上：

- **奇置换 ⇒ 取反**；**偶置换 ⇒ 不变**。

具体例：

| 形式 | 关系 | 备注 |
| :--- | :--- | :--- |
| 1-形式 | $\alpha_{ij} = -\alpha_{ji}$ | 边方向反转，值取反 |
| 2-形式 | $\beta_{ijk} = \beta_{jki} = \beta_{kij}$ | 偶置换 |
| 2-形式 | $\beta_{jik} = \beta_{kji} = \beta_{ikj} = -\beta_{ijk}$ | 奇置换 |

这与定向单纯形的代数性质完全一致。

## 六、2-形式在三角形上的积分

### 三步配方

1. **构建正交基**：在三角形 $t$ 所在平面取一对正交向量 $\mathbf{t}_1, \mathbf{t}_2$。
2. **应用形式**：$f(p) = \omega_p(\mathbf{t}_1, \mathbf{t}_2)$ 是一个标量函数。
3. **标量积分**：$\int_t \omega = \int_t f(p)\,dA$。

### 数值实现（标准做法）

$$\int_t \omega \approx \text{Area}(t) \cdot \frac{1}{N}\sum_{i=1}^N \omega_{p_i}(\mathbf{t}_1, \mathbf{t}_2)$$

随机采样 $N$ 个点 → 取均值 → 乘以三角面积。

### 平均对齐度

$\dfrac{\int_t \omega}{\text{Area}(t)}$ 给出三角形与 $\omega$ 的"平均对齐度"。

### 定向的几何来源

交换 $\mathbf{t}_1, \mathbf{t}_2$ 的顺序 ⇒ 由它们张成的平行四边形的有向面积反号 ⇒ $\omega(\mathbf{t}_2,\mathbf{t}_1) = -\omega(\mathbf{t}_1,\mathbf{t}_2)$。这与三角形顶点排列的奇/偶置换完全一致。

## 七、用列向量编码离散形式

| 形式 | 存储位置 | 向量维度 | 符号规则 |
| :--- | :--- | :--- | :--- |
| 0-形式 | 顶点 | $N_v$ | — |
| 1-形式 | 定向边 | $N_e$ | 反转边方向 ⇒ 取反 |
| 2-形式 | 定向面 | $N_f$ | 奇置换 ⇒ 取反 |

实现细节：
- 为每个单纯形分配整数索引。
- 计算中维护"单纯形 → 数组索引"映射。
- 一旦定下定向约定，整个程序中**保持一致**。

> **关键提醒**：编程语言索引通常从 0 起，但符号规则与索引无关，只与"是否对调过定向"有关。

## 八、链与上链：离散世界的对偶语言

### 1. 单纯链 (Chain)

$k$-单形的整数线性组合：

$$c = \sum_i a_i \sigma_i,\quad a_i \in \mathbb{Z},\ \sigma_i \text{ 为定向 }k\text{-单纯形}$$

- 系数 $-1$：**反转该单纯形的定向**。
- 系数 $0/1$：选 / 不选。
- 多重系数：副本数。

**链群** $C_k$：所有 $k$-链构成的自由阿贝尔群。

### 2. 链的代数：边可抵消

两条带符号边和：

$$c_1 = e_3 - e_{12} + e_{18} - e_{15} + e_6 - e_1$$

两个环 $c_1+c_2$ 相加，公共边方向相反时**自动抵消**，融合为更大的单环。这把几何 / 拓扑问题转成线性代数。

### 3. 边界算子 $\partial$

定义在定向 $k$-单纯形 $[v_{i_0},\ldots,v_{i_k}]$ 上：

$$\partial[v_{i_0},\ldots,v_{i_k}] = \sum_{p=0}^{k} (-1)^p [v_{i_0},\ldots,\hat{v}_{i_p},\ldots,v_{i_k}]$$

具体例：

| 单形 | 边界 |
| :--- | :--- |
| 三角形 $[v_0,v_1,v_2]$ | $[v_0,v_1] + [v_1,v_2] + [v_2,v_0]$（闭合环） |
| 边 $[v_0,v_1]$ | $v_1 - v_0$（终减始） |
| 顶点 $v_1$ | $0$（点没有边界） |

**线性性**：$\partial(\sum a_i\sigma_i) = \sum a_i\,\partial\sigma_i$。

### 4. 边界的边界为零（代数版）

两个三角形 $\sigma_1 + \sigma_2$ 共享边 $e_4$，两侧定向相反：

$$\partial(\sigma_1+\sigma_2) = (e_1+e_4+e_3) + (e_6+e_7-e_4) = e_1+e_3+e_6+e_7$$

$e_4$ 抵消，剩外轮廓。再取一次边界：

$$\partial(e_1+e_3+e_6+e_7) = (v_1-v_0)+(v_4-v_1)+(v_3-v_4)+(v_0-v_3) = 0$$

**结论**：

$$\boxed{\partial\circ\partial = 0}$$

这是连续几何"环路没有端点"的代数化精确表述，也是 $d^2 = 0$ 的对偶根源。

### 5. 余边界 (Coboundary) $\delta$

$k$-单形 $\sigma$ 的余边界 = **所有以一致定向包含 $\sigma$ 的 $(k+1)$-单形**。

- **顶点 $v$ 的余边界**：所有以 $v$ 为终点的边（因为 $\partial e = v_{\text{head}} - v_{\text{tail}}$，要在 $v$ 处出现 $+1$ 系数，须以 $v$ 为终点）。
- **边 $e$ 的余边界**：与其相邻的两个三角形，**带相反符号**（一个 $+\sigma_1$ 一个 $-\sigma_2$，因为它们相对边 $e$ 的局部走向相反）。

**对比**：$\partial$ 降维（面→边→顶点）；$\delta$ 升维（顶点→边→面）。

**与"单纯形星"的区别**：余边界 (a) 限制在恰好高一维；(b) 考虑相对定向。

### 6. 单纯上链 (Cochain) $C^k$

定义：从 $C_k$ 到 $\mathbb{R}$ 的线性映射。

$$\alpha\colon C_k \to \mathbb{R},\quad \alpha\!\left(\sum_i c_i\sigma_i\right) = \sum_i c_i\,\alpha(\sigma_i)$$

- 用**行向量**存储；链用**列向量**存储；配对 $\langle\alpha,c\rangle = \alpha\cdot c$ 是矩阵乘法。
- 形式定义：$C^k = \mathrm{Hom}(C_k,\mathbb{R})$。
- **几何含义**：上链就是离散微分形式——给每个 $k$-单纯形赋一个数（即"局部测量结果"）。
- **配对的几何意义**：$\alpha(c) = \int_{c} \omega$（连续语言中沿区域 $c$ 的积分）。

> 由于上链已存有每个微小单纯形上的积分结果，**任何更大区域的积分只需把这些值线性相加，无近似、无误差**。

## 九、Whitney 基函数：离散到连续的标准插值

### 1. 为何需要插值

我们仅有顶点 / 边 / 面上的离散数值。要在网格内任意点求值（可视化、查询、后处理），需要重建一个连续形式。

### 2. 帽函数（Lagrange 基）：插值 0-形式

每个顶点 $i$ 关联一个分段线性的 $\phi_i$，满足：

$$\phi_i(V_j) = \delta_{ij}$$

它在每个三角形内部就是顶点 $i$ 对应的**重心坐标**。

**插值公式**（标量场 / 0-形式）：

$$u(x) = \sum_i u_i\,\phi_i(x)$$

性质：
- 在顶点处精确等于离散值。
- 像素着色器 / 光栅化阶段的"逐顶点属性插值"就是这套帽函数的硬件实现。

### 3. Whitney 1-形式：插值边数据

对每条有向边 $ij$，定义：

$$\boxed{\phi_{ij} = \phi_i\,d\phi_j - \phi_j\,d\phi_i}$$

性质：
- **方向反转 ⇒ 取反**：$\phi_{ji} = -\phi_{ij}$，与离散 1-形式的符号规则吻合。
- 几何上沿边 $ij$ 主要"指向"该边方向，远离时衰减为零。
- 插值公式：$u = \sum_{ij} u_{ij}\,\phi_{ij}$。

### 4. 通用 Whitney $k$-形式

对定向 $k$-单形 $\sigma = [i_0,\ldots,i_k]$：

$$\phi_\sigma = k! \sum_{j=0}^{k} (-1)^j \phi_{i_j}\, d\phi_{i_0}\wedge\cdots\wedge\widehat{d\phi_{i_j}}\wedge\cdots\wedge d\phi_{i_k}$$

（$\widehat{\cdot}$ 表示省略该项。）

### 5. 与离散化算子的对偶恒等式

**关键性质**：

$$\int_\tau \phi_\sigma = \delta_{\sigma\tau}$$

由此得到**插值 → 离散化恒等映射**：
1. 离散数据 $\hat\omega$ → Whitney 插值得连续 $\omega$ → 沿 $\sigma$ 积分 → 还原为 $\hat\omega_\sigma$。
2. **连续 → 离散 → 连续**则**不一定**回到原形式（离散化是非单射，丢信息）。这是良好的自测题。

### 6. 数据流总览

| 数据类型 | 离散形式 | 基函数 | 连续形式 | 插值公式 |
| :--- | :--- | :--- | :--- | :--- |
| 标量场 | 顶点值 $u_i$ | 帽函数 $\phi_i$ | 0-形式 | $u(x) = \sum_i u_i\,\phi_i$ |
| 边量 / 1-形式 | 边值 $u_{ij}$ | $\phi_{ij} = \phi_i\,d\phi_j - \phi_j\,d\phi_i$ | 1-形式 | $u = \sum_{ij} u_{ij}\,\phi_{ij}$ |
| 面量 / 2-形式 | 面值 $u_{ijk}$ | Whitney 2-形式 | 2-形式 | $u = \sum u_{ijk}\,\phi_{ijk}$ |

## 十、实践中的离散值来源

| 场景 | 离散值的来源 |
| :--- | :--- |
| 网格几何 | 顶点坐标直接给出（0-形式） |
| 函数采样 | 顶点上对标量场采样（高度图、温度场） |
| 物理输入 | 每条边/面上的实测通量（磁通、流量） |
| 数值仿真 | 由离散外微分、离散 Hodge 星算子等**组合现有离散值**生成 |

> **重要观察**：离散形式**很少**通过对一个高分辨率连续场严格积分得到。常见路径是：先有顶点采样 → 通过离散算子推出更高阶的离散形式。

## 十一、对游戏 / 渲染工程师的价值

- **网格属性 ↔ 离散形式**：顶点颜色 / 法线 / UV 是 0-形式；边/面上的物理量是 1/2-形式。
- **着色器与光栅化**：硬件做的逐顶点属性插值就是 Lagrange 帽函数。理解这点便于实现高阶或自定义插值（高次、非线性、各向异性）。
- **数值稳定性**：理解 $d$、$\star$ 的守恒性 / 单位性，可设计低误差的物理模拟（流体、布料）。
- **调试利器**：$d^2 = 0$、$\partial^2 = 0$ 可作严格的离散一致性检查，验证模拟的物理正确性。
- **设计统一性**：把所有微分算子表达为"对角矩阵 × 关联矩阵"的乘积，使代码结构简洁、易于复用。

| 定向 $k$-单纯形 | 带顶点排列定向的 $k$ 维单纯形；奇置换取反 |
| 离散微分 $k$-形式 | 把每个定向 $k$-单纯形映射到一个实数的函数 |
| De Rham 映射 | 通过沿单纯形积分把连续 $k$-形式离散化的线性映射 |
| 链 $C_k$ | 定向 $k$-单纯形的整数线性组合 |
| 边界 $\partial$ | $C_k\to C_{k-1}$ 的线性算子；$\partial^2 = 0$ |
| 余边界 $\delta$ | $C_k\to C_{k+1}$ 的对偶算子；考虑相对定向 |
| 上链 $C^k$ | 链上的线性映射，等价于离散微分形式 |
| 配对 $\langle\alpha, c\rangle$ | 上链作用于链；几何上即区域积分 |
| 关联矩阵 (Incidence Matrix) | 编码顶点-边、边-面邻接的带符号矩阵 |
| Hodge 星（离散） | 借助对偶网格定义的算子，$k$-形式 ↔ $(n\!-\!k)$-形式 |
| 离散外微分 (Discrete $d$) | 用带符号关联矩阵实现的算子，满足 $d^2=0$ |
| Whitney 基函数 | 把离散形式插值回连续形式的标准基；与 De Rham 映射互为恒等 |
| 帽函数 / Lagrange 基 $\phi_i$ | 顶点处的分段线性插值基函数；即重心坐标 |
| 重心坐标 | 在 $k$-单纯形内点的非负权重表示，权和为 1 |
| 楔积 $\wedge$ | 形式之间的反对称积，满足 $\alpha\wedge\beta = (-1)^{pq}\beta\wedge\alpha$ |
