# CMU_DDG_p11

> 主题：离散外微积分（Discrete Exterior Calculus, DEC）—— 离散微分形式、离散外导数 $d$、对偶网格、离散 Hodge 星算子，以及由它们组合得到的经典微分算子

## 一、DEC 总览：拓扑与几何的分离

DEC 把光滑微分几何的三大算子（外导数 $d$、积分 $\int$、Hodge 星 $\star$）系统地搬到网格上。它的核心优雅之处在于**清晰分离了两种信息**：

| 算子 | 离散实现方式 | 依赖什么 |
| :--- | :--- | :--- |
| **外导数** $d$ | 通过斯托克斯定理 → 关联矩阵 | **仅依赖网格拓扑连接性** |
| **积分** $\int$ | 直接对离散值求和 | 已"预积分"在离散形式中 |
| **Hodge 星** $\star$ | 近似对偶胞腔积分 | **捕获所有几何信息**（长度、面积、体积、角度） |

> 这种分离在工程上极为重要：当你修改网格几何时，能精准地知道哪些方程会受影响（含 $\star$ 的部分），哪些不变（仅含 $d$ 的部分）。

## 二、离散微分形式：在 $k$-单形上"存数"

### 1. 定义

**离散 $k$-形式** = 在每个 $k$ 维单形（顶点/边/面/四面体…）上存储一个数值。

| 离散形式 | 存储位置 |
| :--- | :--- |
| 离散 0-形式 | 顶点（0-单形） |
| 离散 1-形式 | 边（1-单形） |
| 离散 2-形式 | 三角形（2-单形） |
| 离散 3-形式 | 四面体（3-单形） |

### 2. 关键解释：值 = 积分，不是采样

每个离散值的含义是**对应连续 $k$-形式在该定向 $k$-单形上的积分**：
$$\hat\omega(\sigma) := \int_\sigma \omega$$

这与"点采样"完全不同——**这是 DDG 保结构的关键**。两个直接推论：

- **定向相关**：翻转单形定向 $\Rightarrow$ 数值变号，$\int_{-\sigma}\omega = -\int_\sigma \omega$。
- **区域积分天然等于求和**：$\int_\Omega \omega \approx \sum_{\sigma\in\Omega}\hat\omega(\sigma)$。

### 3. 反向重建：Whitney 基函数

可以从离散值出发，通过 **Whitney 基函数** 插值回连续 $k$-形式（对原始网格简单且优雅，对偶网格则非常复杂——见第八节）。

## 三、离散外导数 $d$：用斯托克斯定理"反推"

### 1. 核心策略

不是用"有限差分"近似导数，而是问：**"如果先做光滑外微分 $d\omega$，再在 $(k+1)$-单形上积分，结果是什么？"**——这就直接定义离散值。

### 2. 斯托克斯定理是基石

$$\int_\sigma d\omega \;=\; \int_{\partial \sigma} \omega$$

把"在区域内部对导数积分"转化为"在边界上对原函数积分"。**离散化时这个等式不再是近似——它精确成立**。

### 3. 通用公式：边界上的带符号求和

$$\boxed{\;(\hat d\hat\alpha)(\sigma) \;=\; \sum_{\tau \in \partial \sigma} \mathrm{sgn}(\tau,\sigma)\,\hat\alpha(\tau)\;}$$

其中 $\mathrm{sgn}(\tau,\sigma)=\pm 1$ 取决于边界 $k$-单形 $\tau$ 的定向是否与 $\sigma$ 一致。

> **记忆口诀**：外微分 = 绕边界，求和差分即得到。

### 4. 实例：0-形式 → 1-形式（梯度）

设边 $e$ 从 $v_1$ 指向 $v_2$，则
$$\widehat{d\phi}(e) = \int_e d\phi = \int_{\partial e}\phi = \phi(v_2)-\phi(v_1)$$

- **本质**：一维斯托克斯就是微积分基本定理。
- **重要**：**无需知道函数沿边的内部形状**——即使边很长，只要端点值已知，结果**精确**。

### 5. 实例：1-形式 → 2-形式（旋度）

对定向三角形 $\sigma = e_1+e_2+e_3$（边界由三条定向边代数和构成）：
$$\widehat{d\alpha}(\sigma) = \sum_{i=1}^3 \mathrm{sgn}(e_i,\sigma)\,\hat\alpha(e_i)$$

具体例子（左上三角形，逆时针定向，三边值为 $3,9,7$，其中后两条边定向与三角形相反）：
$$3-9-7=-13$$

### 6. 通用计算模式

| 维度 | 公式 | 实质 |
| :--- | :--- | :--- |
| 1（边） | $\widehat{d\phi}(e) = \phi(v_2)-\phi(v_1)$ | 端点值差 |
| 2（面） | $\widehat{d\alpha}(\sigma)=\sum_i \pm\hat\alpha(e_i)$ | 沿边界循环带符号求和 |
| 一般 | $\widehat{d\hat\alpha}(\sigma)=\sum_{\tau\in\partial\sigma}\pm\hat\alpha(\tau)$ | 共边界算子（co-boundary） |

## 四、矩阵化表示：稀疏关联矩阵

### 1. 为什么要用矩阵？

大规模网格上，把离散外导数写成矩阵 $D_k$ 后，所有运算都变成稀疏线性代数，便于实现、优化和求解线性系统。

### 2. $d_0$：顶点 → 边（梯度）

- **维度**：$E\times V$（$E$ 条边，$V$ 个顶点）。
- **每行恰好两个非零元**：起点列为 $-1$，终点列为 $+1$，其余为 $0$。
- **行向量的物理意义**：编码"该边连接哪两个顶点 + 边的方向"。
- **矩阵-向量乘法**：$(d_0\boldsymbol\phi)_e = \phi(v_2)-\phi(v_1)$。

### 3. $d_1$：边 → 三角形（旋度）

- **维度**：$F\times E$（$F$ 个面，$E$ 条边）。
- **每行恰好三个非零元**（三角形面）：取值 $\pm 1$，由"边定向 vs 面环绕方向"决定。
  - 一致 → $+1$；相反 → $-1$。
- **矩阵-向量乘法**：$(d_1\boldsymbol\alpha)_f = \sum_{e\in\partial f}\pm\alpha_e$。

### 4. 一般 $d_k$

从 $k$-形式到 $(k+1)$-形式的线性映射，矩阵元素 $\in\{-1,0,+1\}$，仅由**网格拓扑**（关联关系）和**定向**决定，**不含任何度量**。

### 5. 重要提醒：矩阵不是算子本身

同一个离散算子 $d_k$ 可以有无数种矩阵表示——重排顶点/边的索引只会得到原矩阵的一个排列。**关键的是它执行的运算（边界差分），不是具体数值表**。

### 6. 实际编程：必须用稀疏数据结构

这些矩阵极度稀疏，每行只有 $O(1)$ 个非零元；用稠密矩阵会让性能崩溃。

## 五、离散外导数的两大基石性质

### 1. 与离散化交换：$\widehat{d\alpha} = D\,\hat\alpha$

光滑外微分 $d$ 与"离散化（对单形积分）"两个操作可以**交换顺序**：

$$\underbrace{\widehat{d\alpha}}_{\text{先光滑微分再离散积分}} \;=\; \underbrace{D\,\hat\alpha}_{\text{先离散化再用矩阵}D}$$

这是离散外导数**精确**的根源——它直接源于斯托克斯定理。

### 2. 莱布尼茨法则（分级乘积法则）

外微分对楔积满足分级版的 Leibniz 法则：

$$d(\alpha\wedge\beta) = d\alpha\wedge\beta + (-1)^k\,\alpha\wedge d\beta$$

其中 $k$ 是 $\alpha$ 的阶数，符号 $(-1)^k$ 来自把 $d$ "穿过" $\alpha$ 时的反对称代价。这条性质是 $d$ 与楔积兼容的充要条件，也是它能统一推广 grad/curl/div 乘积法则的根本原因。

### 3. 幂零性：$d\circ d = 0$

$$\boxed{\;D_{k+1}\,D_k = 0\;}$$

- **几何理由**：连续世界 $\partial\partial\Omega = \varnothing$（边界的边界为空）。
- **离散世界**：一个三角形的"边的边的端点"在带符号求和时恰好成对抵消。
- **物理意义**：保证"梯度场无旋"、"旋度场无散"等恒等式在离散层面**精确**成立，没有数值误差。

> 在实现中可以直接计算 $D_1\,D_0$ 验证为零矩阵。这是离散庞加莱引理的核心。

## 六、对偶网格与 Poincaré 对偶

### 1. 动机：为 Hodge 星准备对偶元素

光滑情形 $\star$ 把 $k$-形式映为 $(n-k)$-形式。在网格上要实现这一映射，需要在**每个原始 $k$-单形**和**某个 $(n-k)$ 维对偶元素**之间建立一一对应。

### 2. 二维情形对应表

| 原始单形 | 维度 | $\leftrightarrow$ | 对偶单元 | 维度 |
| :--- | :---: | :---: | :--- | :---: |
| 顶点（黑色） | 0 | $\leftrightarrow$ | 对偶 2-胞腔（围绕该顶点的区域） | 2 |
| 边 | 1 | $\leftrightarrow$ | 对偶边（连接两相邻三角形外心，与原始边垂直） | 1 |
| 三角形 | 2 | $\leftrightarrow$ | 对偶顶点（位于三角形内的某个特征点） | 0 |

### 3. 外心对偶（Circumcentric Dual）：标准方案

把每个 $k$-单形的对偶顶点取为它的**外心**（外接球/圆的中心）：

| 单形维度 | 外心位置 |
| :--- | :--- |
| 0（顶点） | 自身 |
| 1（边） | 中点 |
| 2（三角形） | 外接圆圆心 |
| 3（四面体） | 外接球球心 |

**核心优良性质：原始单形 $\sigma^k$ 与对偶单元 $\star\sigma^{n-k}$ 在交点处互相正交**。

### 4. 警告：负的"有符号"度量

外心**不一定在单形内部**——例如**钝角三角形**的外心在三角形外。
后果：相关对偶边/对偶面会出现**负的有符号长度/面积**，在某些数值方法（流体仿真等）中引发不稳定。

## 七、对偶离散微分形式

### 1. 定义

把数值存储在对偶网格的元素上：

| 形式 | 二维存储位置 |
| :--- | :--- |
| 对偶 0-形式 | 对偶顶点（即每个原始三角形上一个值） |
| 对偶 1-形式 | 对偶边（每条原始边的对偶上一个值） |
| 对偶 2-形式 | 对偶 2-胞腔（每个原始顶点周围一个区域上的值） |

### 2. 自由度通常不匹配

光滑情形下原始 $k$-形式与对偶 $(n-k)$-形式属于同一空间（维数相同）；但**离散情形下两者维数一般不相等**。

例如二维三角网格：通常 $\#\text{三角形} \approx 2\,\#\text{顶点}$，所以原始 0-形式（每个顶点一个值）与对偶 0-形式（每个三角形一个值）的向量维度不同。**特例**：正四面体表面 4 顶点 = 4 三角形，恰好相等。

### 3. 对偶外微分：仍然只看拓扑

对偶网格上的 $d$ 与原始网格定义方式完全一致——对偶 $(k+1)$-胞腔的值 = 其对偶边界上对偶 $k$-胞腔值的**带符号求和**，符号只由相对定向决定，**不含任何几何量**。

事实上，**对偶外微分恰好等于原始外微分的转置**：
$$D_k^{\text{dual}} = D_{n-k-1}^{\;T}$$

维度上立刻自洽：原始 $D_0$ 是 $E\times V$ → 对偶 $E\times V \to F$，转置 $V\times E$ 正是对偶 $d_0$（对偶 0-胞腔到对偶 1-胞腔，即 $F$ 个 → $E$ 个 反过来）。

## 八、对偶网格的插值难题

### 1. 原始网格：用 Whitney 基函数（简单）

- 一个 $k$-单形 = $k+1$ 个仿射无关顶点的凸包，几何上**简单、线性**。
- **Whitney 形式** 提供了一组优雅的基函数：在指定顶点为 1、其他顶点为 0，自然适配单形几何。

### 2. 对偶网格：通用插值困难

对偶胞腔（如 Voronoi 胞腔）是**任意多边形/多面体**，可能：
- 顶点数不定；
- 顶点不在同一 $k$ 维线性子空间内；
- **不一定是凸的**。

**关键论证**：在一个 7 边形对偶胞腔上，无法找到一个**线性函数**同时通过 6 个零顶点（要求函数平坦）和 1 个值为 1 的顶点（要求函数倾斜）——线性插值彻底失效。

### 3. 退路：广义重心坐标

适用于一般多边形/多面体的基函数，理想性质包括：

1. 非负性
2. 插值性（顶点处取给定值）
3. 线性精确性（能精确再现线性函数）
4. 内部光滑

具体实现如 **Wachspress 坐标** 适用于**凸**多边形，但**对凹形会失效**（产生负值或奇异）。**No Free Lunch**：一般多边形的插值不存在同时满足所有理想性质的方案。

### 4. 实践含义

- 工程上往往**把数据存储在原始网格上**，借助 Hodge 星按需把量"翻译"到对偶侧。
- 对偶值仍便于代数操作（求和、矩阵乘法），只是**插值困难**而已。

## 九、离散 Hodge 星算子 $\star$

### 1. 几何直觉：互补描述

光滑 $\star$ 在 $k$-形式与 $(n-k)$-形式间转换，形如"线性子空间 ↔ 正交补"，但与子空间对偶不同的是：**$k$-形式额外携带大小和定向**。

物理上：
- 沿原始边的**环量**（circulation, $\hat\alpha$）
- ↔ 穿过对偶边的**通量**（flux, $\widehat{\star\alpha}$）

> 想象沿一根吸管的水流量 vs 穿过该吸管对应小环面的水流量——两者描述同一物理场的两种方式。

### 2. 关键近似：体积比

如果连续形式 $\alpha$ 局部近似为常数（光滑、网格细密），则原始积分与对偶积分的比值约等于体积比：
$$\boxed{\;(\star\hat\alpha)_{\sigma^\star} \;\approx\; \frac{|\sigma^\star|}{|\sigma|}\,\hat\alpha_\sigma\;}$$

| 量 | 定义 |
| :--- | :--- |
| $\sigma$ | 原始 $k$-单形 |
| $\sigma^\star$ | 对应的对偶 $(n-k)$-胞腔 |
| $\|\sigma\|, \|\sigma^\star\|$ | 体积（长度/面积/体积） |

### 3. 与外导数的本质区别

| 算子 | 精确性 | 依据 |
| :--- | :--- | :--- |
| 离散外导数 $d$ | **精确**（在离散层面） | 斯托克斯定理 |
| 离散 Hodge 星 $\star$ | **近似** | 几何"体积比"假设 + 局部常值近似 |

近似精度由**网格细度**和**形式光滑性**共同决定。

### 4. 二维三角网格：关键公式

#### 1-形式的 $\star$（边 ↔ 对偶边）
$$\star\hat\alpha = \frac{|e^\star|}{|e|}\,\hat\alpha$$

对外心对偶，**$\frac{|e^\star|}{|e|}$ 化为著名的 cotan 公式**：
$$\frac{|e^\star|}{|e|} = \tfrac{1}{2}\bigl(\cot\alpha_{ij} + \cot\beta_{ij}\bigr)$$
其中 $\alpha_{ij},\beta_{ij}$ 是该边对面的两个角。

#### 0-形式的 $\star$（顶点 ↔ 对偶 2-胞腔）
$$\star\hat\phi = |\sigma^\star_{\text{vertex}}|\cdot\hat\phi$$
（原始顶点体积 $|\sigma|=1$；这里取顶点周围对偶 2-胞腔的面积，**而不是字面上的"零体积"**——避免除零陷阱。）

#### 2-形式的 $\star$（三角形 ↔ 对偶顶点）
$$\star\hat\omega = \frac{1}{|\text{三角形}|}\,\hat\omega$$

三角形面积可用 Heron 公式：$A=\sqrt{s(s-a)(s-b)(s-c)},\;s=(a+b+c)/2$。

#### 三维例（2-形式）
两个四面体共享三角面 $ijk$，$\star\hat\omega = \dfrac{|e^\star_{ab}|}{A_{ijk}}\hat\omega$。

### 5. 矩阵形式：对角 Hodge 星

由于每个体积比只涉及一对原始/对偶单元，对角 Hodge 星就是一个**对角矩阵**：
$$[\star_k] = \mathrm{diag}\Bigl(\frac{|\sigma_1^\star|}{|\sigma_1|},\frac{|\sigma_2^\star|}{|\sigma_2|},\ldots\Bigr)$$

**实际构建技巧**：只需边长、面积、cotan 等基本几何量；**无需显式求出对偶顶点坐标**。

### 6. 重心对偶 vs 外心对偶 vs Galerkin

| 方案 | 对偶顶点位置 | 体积是否为正 | 正交性 | 矩阵形式 | 精度 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **外心对偶** | 外心 | 钝角时可能为负 | ✅ | 对角 | 中-高 |
| **重心对偶** | 顶点平均（如四面体的 $(v_1+v_2+v_3+v_4)/4$） | **始终为正** | ❌ 缺失 | 对角 | 较低 |
| **Galerkin（有限元）** | — | 正 | — | **稀疏但非对角** | 较高 |

**质量集中（mass lumping）**是连接它们的桥梁：对 Galerkin 得到的非对角矩阵做 mass lumping（把非对角元压到对角线），即可得到外心对偶的对角 Hodge 星。

## 十、DEC 算子连接图（导航地图）

写代码、推导算法时随时回头看的"救命图"。

### 1. 符号定义

- $\Omega^k$：**原始 $k$-形式空间**（值在原始 $k$-单形上）
- $\Omega^\star_k$：**对偶 $k$-形式空间**（值在对偶 $k$-维单元上）

### 2. 横向（外导数）/纵向（Hodge 星）布局（二维 $n=2$）

```
原始侧:    Ω⁰  --d₀-->  Ω¹  --d₁-->  Ω²
            |           |            |
           ⋆₀          ⋆₁           ⋆₂
            ↓           ↓            ↓
对偶侧:   Ω⋆₂  <-d₀ᵀ-- Ω⋆₁  <-d₁ᵀ-- Ω⋆₀
```

### 3. 关键映射

| 算子 | 起点 → 终点 | 维度 |
| :--- | :--- | :--- |
| $d_0$ | $\Omega^0\to\Omega^1$ | $E\times V$ |
| $d_1$ | $\Omega^1\to\Omega^2$ | $F\times E$ |
| $\star_0$ | $\Omega^0\to\Omega^\star_2$ | $V\times V$（对角） |
| $\star_1$ | $\Omega^1\to\Omega^\star_1$ | $E\times E$（对角） |
| $\star_2$ | $\Omega^2\to\Omega^\star_0$ | $F\times F$（对角） |
| 对偶 $d_0$ | $\Omega^\star_0\to\Omega^\star_1$ | $= D_1^T$ |
| 对偶 $d_1$ | $\Omega^\star_1\to\Omega^\star_2$ | $= D_0^T$ |

> **维度匹配是检查算子合法性的最可靠手段**：发现矩阵维度不对，必有概念出错。

## 十一、组合矩阵构建经典算子

DEC 的强大之处：把上图中的箭头**串起来**，就构建出向量微积分中所有经典算子（在曲面上也成立！）。

### 1. 基本"配方"

| 经典算子 | DEC 表达式 | 路径 |
| :--- | :--- | :--- |
| **梯度** | $D_0$ | $\Omega^0\to\Omega^1$ |
| **旋度** | $\star_2 D_1$ | $\Omega^1\to\Omega^2\to\Omega^\star_0$ |
| **散度** | $\star_0^{-1} D_0^T \star_1$ | $\Omega^1\to\Omega^\star_1\to\Omega^\star_2\to\Omega^0$ |
| **拉普拉斯**（标量场） | $\Delta = \star_0^{-1} D_0^T \star_1 D_0$ | $\Omega^0\to\Omega^1\to\Omega^\star_1\to\Omega^\star_2\to\Omega^0$ |

> 标量拉普拉斯 = 散度 ∘ 梯度，DEC 自动落到著名的 **cotan-Laplace** 矩阵。该矩阵的对称正定性直接由路径结构保证。

### 2. 通用工作流：从网格到算法

1. **加载网格**（顶点坐标 + 三角形/四面体连接关系）
2. **构建基础稀疏矩阵**：
   - 关联矩阵 $D_0, D_1$（仅拓扑）
   - 对角 Hodge 星 $\star_0,\star_1,\star_2$（基本几何量）
3. **组合**目标算子（梯度/散度/旋度/Laplace…）
4. **求解**线性方程组（如 Poisson、热扩散）或优化问题

这个流程对**平面**、**任意拓扑曲面**、**高维流形**统一适用。

### 3. 离散楔积 $\wedge$

基本算子家族里还有楔积，用于不同阶离散形式之间的乘法（如两个离散 1-形式 → 一个离散 2-形式）。它不是简单的矩阵乘法，而是基于**交错和**（alternating sum），与连续楔积一致。

### 4. 应用全景

| 领域 | DEC 用法 |
| :--- | :--- |
| 几何处理 | 网格降噪、参数化、形变分析 |
| 物理仿真 | 流体（涡度）、弹性体 |
| 图像处理 | 基于扩散/泊松的算法 |
| 可视化 | 矢量场分解、流线提取 |
| 工业 | 有限元方法（FEM）的基础工具 |

## 十二、设计哲学：No Free Lunch

### 1. 离散外微积分的根本取舍

光滑算子之间存在丰富的相互关系（结合律、可逆性、与度量的相容性…）。在离散设置下，**不可能同时保留所有性质**——必须根据应用选择保哪一组。

### 2. 三大算子的精确性对比

| 算子 | 精确程度 | 约束 |
| :--- | :--- | :--- |
| 离散外导数 $d$ | **精确**（拓扑/Stokes 自动保证） | 仅靠网格连接性 |
| 离散积分 | **精确**（已预积分） | 与定向相容 |
| 离散 Hodge 星 $\star$ | **近似** | 依赖几何质量、网格细度 |

### 3. 工程心法

- 想要 $d^2=0$、$\partial\partial=\varnothing$、Stokes 等**拓扑/代数恒等式**？无条件保证。
- 想要数值精度高？投入到 $\star$ 的设计（外心 vs 重心 vs Galerkin）。
- 想要永远正体积？用重心对偶，但接受精度损失。

## 十三、关键术语速查

| 术语 | 含义 |
| :--- | :--- |
| 离散 $k$-形式 | 在每个 $k$-单形上存一个数；含义是连续 $k$-形式的积分 |
| $k$-单形 | 顶点（0）/边（1）/三角形（2）/四面体（3） |
| 关联矩阵 | $\{-1,0,+1\}$ 矩阵，编码定向边界关系 |
| 离散外导数 $d$ | 边界上的带符号求和；精确，源于斯托克斯定理 |
| Stokes 定理 | $\int_\sigma d\omega=\int_{\partial\sigma}\omega$；DEC 的基石 |
| $d^2 = 0$ | 离散庞加莱引理；源于"边界的边界为空" |
| 原始/对偶网格 | Poincaré 对偶；$k$-单形 ↔ $(n-k)$-胞腔 |
| 外心对偶 | 对偶顶点 = 原始单形的外心；保证正交性 |
| 重心对偶 | 对偶顶点 = 原始顶点平均；保证正体积 |
| Hodge 星 $\star$ | 原始 $k$-形式 → 对偶 $(n-k)$-形式；依赖几何 |
| 对角 Hodge 星 | $\star = \mathrm{diag}\bigl(\|\sigma^\star\|/\|\sigma\|\bigr)$，最简近似 |
| Cotan 公式 | $\frac{\|e^\star\|}{\|e\|} = \tfrac12(\cot\alpha+\cot\beta)$ |
| Whitney 形式 | 原始网格上插值的标准基函数 |
| 广义重心坐标 | 一般多边形上的插值方案；如 Wachspress |
| Galerkin 方法 | 用 Whitney 插值后积分得到的非对角 $\star$ |
| 质量集中 | 把非对角 $\star$ 压到对角，与外心对偶等价 |
| Cotan Laplace | $\Delta = \star_0^{-1} D_0^T \star_1 D_0$；DDG 最常用算子 |
| No Free Lunch | 不存在同时保所有光滑性质的离散方案 |
