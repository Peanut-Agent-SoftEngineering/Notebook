# CMU_DDG_p05

> 主题：外代数（Exterior Algebra）—— 楔积、k-向量、Hodge 星、坐标表示

## 一、为什么要学外代数

### 1. 语言即钥匙

> "如果有人用日语说'地下室有黄金'，但你不懂日语，这条信息你永远无法获取。"

- **不同任务需要不同语言**——Russell 与 Whitehead 的《数学原理》用集合论花数百页才证明 $1+1=2$；用合适的语言这本来是平凡的。
- **外代数**不是唯一正确的语言，但是**大量几何学家、物理学家用来表达深刻见解**的语言。

### 2. 核心动机：描述"带符号体积"

- 几何与物理中大量量都涉及**长度、面积、体积**，并且必须考虑**方向（朝向）**——是"向内"还是"向外"。
- 这与之前学过的**有向单纯形**直接呼应。

### 3. 应用领域

| 领域 | 用途 |
| :--- | :--- |
| **数学** | 代数几何、几何代数（Clifford 代数） |
| **物理** | 区分无质量量（速度、加速度）与有质量量（动量、力）；几何力学起点 |
| **计算机科学** | 网格上的几何计算（DEC 的代数基础） |

### 4. 课程目标：离散外微积分

- **离散外微积分**用网格上的计算替代传统矢量微积分中的纸笔公式。
- **关键洞察**：网格元素（**单纯形**）将与即将学习的 **k-形式** 形成**自然对应关系**。

### 5. 先修与路径

- **先修**：线性代数（向量加法、空间）、向量微积分（向量场如何随空间变化）。
- **学习路径**：当前处理一维向量 → 推广线性代数 → 处理具有更高维体积的对象。

## 二、从直觉到抽象：向量空间复习

### 1. 一个看似幼稚的问题

**为什么 $3 \times 4 = 4 \times 3$？**

- "3 行每行 4 个" vs "4 列每列 3 个"——它们相等是因为**计算的是同一个矩形面积**，只是计数方式不同。
- **深层意义**：物理量（如面积）是客观存在的，而我们的运算符号（如乘法）只是描述它的工具。
- **学习建议**：**不要盲信规则，永远追问"它从何而来"、"被这样定义的动机是什么"**。规则是对自然行为的数学建模。

### 2. 向量不仅是箭头

**经典视角**：向量 = 带方向的线段，可加法、可数乘。

**扩展视角**：**函数也是向量**

- $f + g$ 仍是函数（加法封闭）
- $a \cdot f$ 仍是函数（数乘封闭）
- 函数集合构成**函数空间**——一个无限维向量空间
- 当你逐条检查向量空间公理，函数和箭头**行为完全一致**

### 3. 向量空间公理

向量空间是集合 $V$ 配以两种运算（$+$ 和 $\cdot$），满足以下八条公理（$\forall \mathbf{x},\mathbf{y},\mathbf{z}\in V,\ \forall a,b\in\mathbb{R}$）：

| 类别 | 公理 | 几何直觉 |
| :--- | :--- | :--- |
| **加法交换** | $\mathbf{x} + \mathbf{y} = \mathbf{y} + \mathbf{x}$ | 先沿 $\mathbf{x}$ 再沿 $\mathbf{y}$ 与反过来到达同一点（路径无关） |
| **加法结合** | $(\mathbf{x}+\mathbf{y})+\mathbf{z} = \mathbf{x}+(\mathbf{y}+\mathbf{z})$ | — |
| **零向量** | $\exists \mathbf{0}\in V,\ \mathbf{x}+\mathbf{0}=\mathbf{x}$ | — |
| **加法逆元** | $\forall \mathbf{x},\ \exists -\mathbf{x}\ \text{s.t.}\ \mathbf{x}+(-\mathbf{x})=\mathbf{0}$ | 沿任何方向走多远，都能反方向走回 |
| **数乘相关** | 分配律、结合律等 | — |

> **注意**：
> - 当前讨论实数向量空间，可推广到复数。
> - **简单规则用几何图理解；复杂规则（如结合律）用代数符号操作**——理解符号背后的几何意义是正确操作的前提。

### 4. 维度

- **有限维（1, 2, 3 维）**：日常物体处理。
- **无限维**：处理函数时出现，因为函数本质上无法用有限点完全描述。

## 三、内积：衡量"对齐程度"

### 1. 定义与四条公理

**内积**：$\langle \cdot, \cdot \rangle: V \times V \to \mathbb{R}$，满足

| 公理 | 公式 | 直觉 |
| :--- | :--- | :--- |
| **对称性** | $\langle \mathbf{x},\mathbf{y}\rangle = \langle \mathbf{y},\mathbf{x}\rangle$ | 对齐程度与谁先谁后无关 |
| **线性性（第一参数）** | $\langle a\mathbf{x}+b\mathbf{y},\mathbf{z}\rangle = a\langle\mathbf{x},\mathbf{z}\rangle + b\langle\mathbf{y},\mathbf{z}\rangle$ | 投影是线性运算 |
| **正定性** | $\langle\mathbf{x},\mathbf{x}\rangle \ge 0$，且 $=0 \iff \mathbf{x}=0$ | 长度平方非负，零向量长度为零 |

### 2. 经典例子：欧几里得内积（点积）

$$\langle \mathbf{x},\mathbf{y}\rangle = \sum_{i=1}^{n} x_i y_i$$

- 公理来自对点积性质的抽象总结。
- 内积让我们在**任意抽象向量空间**（不限于 3D 箭头）中使用"长度"、"角度"、"垂直"。

## 四、楔积（Wedge Product）：构建有向 k 维体积

### 1. 张成 vs 楔积：不同的对象

| 概念 | 作用对象 | 产生 | 几何直觉 |
| :--- | :--- | :--- | :--- |
| **张成 $\mathrm{span}(u,v)$** | 向量集 | **线性子空间**（无限集合，无限延伸的平面） | 这些向量能到达的整个空间 |
| **楔积 $u \wedge v$** | 向量对 | **2-向量**（具体元素） | 一个**有向、有大小的平面片段**（平行四边形） |

| 特征 | 张成 | 楔积 |
| :--- | :--- | :--- |
| 对长度依赖 | ❌ 不依赖 | ✅ 依赖（长度影响大小） |
| 方向性 | 无 | ✅ 有（反对称） |
| 例 | $\mathrm{span}(u,v)$ = 整个平面 | $u \wedge v$ = 该平面上一个有向平行四边形 |

> 一句话：张成告诉你这些向量能生成什么**空间**；楔积告诉你它们构成的**有向小块**有多大、朝哪个方向。

### 2. 楔积的几何意义

- $u \wedge v$ 的**大小** = 由 $u, v$ 张成的平行四边形**有向面积**。
- **方向**由 $u \to v$ 的手性决定。
- $u, v$ 共线 $\implies u \wedge v = 0$（面积退化为零）。

### 3. 楔积的代数性质

| 性质 | 公式 | 几何直觉 |
| :--- | :--- | :--- |
| **自楔为零** | $u \wedge u = 0$ | 同向量构成退化平行四边形，面积为零 |
| **反交换性** | $u \wedge v = -\, v \wedge u$ | 交换顺序 → 反转有向面积朝向 |
| **结合律** | $(u\wedge v)\wedge w = u\wedge(v\wedge w)$ | 不同顺序构建同一平行六面体 |
| **分配律（双线性）** | $u\wedge(v_1+v_2) = u\wedge v_1 + u\wedge v_2$ | 两个平行四边形面积之和 |

**反交换性的推导**：从 $u\wedge u = 0$ 展开 $(u+v)\wedge(u+v) = 0$ 得到 $u\wedge v + v\wedge u = 0$。

### 4. 楔积 vs 叉积

| 特征 | 叉积 $u \times v$ | 楔积 $u \wedge v$ |
| :--- | :--- | :--- |
| 维度 | 仅 3D | **任意维度** |
| 结果 | 一个向量 | 一个**真正的**面积元素 |
| 度量依赖 | 依赖度量与正交基（坐标依赖） | **纯组合几何**，不依赖度量 |

> 叉积正是 3D 空间中楔积的对偶（通过 Hodge 星联系）。

## 五、k-向量

### 1. 定义

| 阶数 | 名称 | 几何意义 |
| :--- | :--- | :--- |
| 0-向量 | 标量 | 一个普通数（如 $1, \pi$） |
| 1-向量 | 普通向量 | 有向线段 |
| 2-向量 | 双向量 / bivector | **有向面积元** |
| 3-向量 | 三向量 | **有向体积元** |
| k-向量 | — | **有向 k 维体积元** |

- 所有 k-向量构成一个向量空间，记作 $\bigwedge^{\!k} V$。
- **0-向量**：楔积零个向量得到——形式上就是标量，统一了整个体系。

### 2. 关键澄清：k-向量不是具体的平行四边形

> **k-向量只编码两条信息：方向（定向）和大小（体积）**。

- 用平行四边形可视化 2-向量只是**心理模型**。
- **等价规则**：所有位于同一平面、有相同面积、相同定向的平行四边形，都表示**同一个** 2-向量。
- 类似地：3D 中可把 2-向量想象成**一小块曲面**——具有给定面积和给定法线方向。
- **结论**：k-向量是**抽象的有向体积元**，不是具体几何形状；只关心"多少体积"和"朝哪个方向"。

## 六、k-向量的基与维度

### 1. 一般规律

> **n 维向量空间中，k-向量空间的维度是 $\binom{n}{k}$。**

**推导**：
- 从 $n$ 个基 1-向量中选 $k$ 个不同的进行楔积。
- 两个排除规则使其变成"组合"而非"排列"：
  1. **反对称性**：$e_3 \wedge e_2 = -e_2 \wedge e_3$ → 相同元素不同顺序冗余
  2. **零律**：$e_4 \wedge e_4 = 0$ → 不能有重复

**构造方法**：从 $n$ 个基 1-向量中选 $k$ 个，按**严格递增顺序** $i_1 < i_2 < \dots < i_k$ 楔积。

### 2. 实例

#### $\mathbb{R}^3$ 的基

- 1-向量基：$e_1, e_2, e_3$（$\binom{3}{1}=3$）
- 2-向量基：$e_1\wedge e_2,\ e_2\wedge e_3,\ e_3\wedge e_1$（$\binom{3}{2}=3$）
- 3-向量基：$e_1\wedge e_2\wedge e_3$（$\binom{3}{3}=1$）
- 维度序列：**1, 3, 3, 1**（帕斯卡三角第 3 行）

#### $\mathbb{R}^4$ 的基

| $k$ | $\binom{4}{k}$ | 基向量 |
| :---: | :---: | :--- |
| 0 | 1 | $1$（标量，即"不做楔积"） |
| 1 | 4 | $e_1, e_2, e_3, e_4$ |
| 2 | 6 | $e_1\!\wedge\!e_2,\ e_1\!\wedge\!e_3,\ e_1\!\wedge\!e_4,\ e_2\!\wedge\!e_3,\ e_2\!\wedge\!e_4,\ e_3\!\wedge\!e_4$ |
| 3 | 4 | $e_1\!\wedge\!e_2\!\wedge\!e_3,\ e_1\!\wedge\!e_2\!\wedge\!e_4,\ e_1\!\wedge\!e_3\!\wedge\!e_4,\ e_2\!\wedge\!e_3\!\wedge\!e_4$ |
| 4 | 1 | $e_1\!\wedge\!e_2\!\wedge\!e_3\!\wedge\!e_4$ |

- 维度序列：**1, 4, 6, 4, 1**（帕斯卡三角第 4 行）

> **应用视角**：4D 中"有向面积"由 6 个独立分量构成；3D 中"有向体积"只有 1 个分量。

## 七、楔积的实操计算

### 1. 算法模板

1. **代入**：将 $\alpha, \beta, \dots$ 展开为基向量的线性组合
2. **展开**：应用分配律，像多项式乘法那样
3. **简化**：应用反对称性，消去 $e_i \wedge e_i = 0$ 的项；用 $-e_j\wedge e_i = e_i\wedge e_j$ 统一顺序
4. **合并**：合并相同基楔积的项

### 2. 2D 实例

**已知** $\alpha = 2e_1 + e_2,\ \beta = -e_1 + 2e_2$，求 $\alpha \wedge \beta$。

$$
\begin{aligned}
\alpha \wedge \beta
&= (2e_1 + e_2) \wedge (-e_1 + 2e_2) \\
&= -2\,(e_1 \wedge e_1) - (e_2 \wedge e_1) + 4\,(e_1 \wedge e_2) + 2\,(e_2 \wedge e_2) \\
&= 0 + (e_1 \wedge e_2) + 4(e_1 \wedge e_2) + 0 \\
&= 5\,(e_1 \wedge e_2)
\end{aligned}
$$

**几何解读**：结果是一个**有向面积** 5 倍单位面积元；数值 5 是面积；$e_1 \wedge e_2$ 给出方向。

### 3. 3D 高阶实例（维度检查 + 计算）

设 $\alpha$ 是 2-向量，$\beta$ 是 1-向量，$\gamma$ 是 2-向量。计算 $\star(\alpha \wedge \beta + \beta \wedge \gamma)$。

**第一步：维度检查**

- $\alpha \wedge \beta$：2 + 1 = **3-向量**
- $\beta \wedge \gamma$：1 + 2 = **3-向量**
- 和合法
- $\mathbb{R}^3$ 中 3-向量取 Hodge 星 → **0-向量（标量）**

**第二步：代入与化简**

代入定义、应用双线性与反对称，最终归约为某个标量乘以 $e_1 \wedge e_2 \wedge e_3$，再取 Hodge 星得到该标量。

> **核心洞察**：当几何直觉难以奏效（高维混合楔积），代数规则提供了**可靠且高效**的计算路径。

## 八、Hodge 星算子（$\star$）

### 1. 动机：用补集定义对象

- 有时描述一个空间**"不是什么"**比描述它**"是什么"**更高效（"除了巴伐利亚菜，我什么都喜欢"）。
- **Hodge 星**正是"补集"思想在向量空间中的数学实现：一个 k 维对象可以由其正交的 $(n-k)$ 维对象**等价地**表示，二者携带相同的大小和方向信息。

### 2. 前置概念：正交补 $U^\perp$

> 设 $U$ 是内积空间 $V$ 的线性子空间：
> $$U^\perp = \{v \in V \mid \langle u, v\rangle = 0,\ \forall u \in U\}$$

- **必须有内积**才能定义"正交"（点积是经典例子，但闵可夫斯基度量等其他内积会给出不同正交）。
- **关键**：正交补**依赖度量**，而生成空间只依赖代数结构——这是与张成空间的根本不同。

| 子空间 $U$（在 $\mathbb{R}^3$） | 正交补 $U^\perp$ |
| :--- | :--- |
| 直线（1D） | 平面（2D，所有垂直该直线的向量） |
| 平面（2D） | 直线（1D，法线方向） |
| 原点（0D） | 整个 $\mathbb{R}^3$ |

**为什么对图形学/工程师重要**：

| 应用 | 关系 |
| :--- | :--- |
| 求解线性方程组 | 矩阵 $A$ 的零空间 = 行空间的正交补 |
| 法线计算与着色 | 三角形法线 = 其平面的正交补 |
| 投影与最小二乘法 | 投影误差向量 ∈ 子空间的正交补 |
| 物理约束求解 | 允许运动空间 $U$，约束力 ∈ $U^\perp$ |

### 3. Hodge 星：比正交补多了什么

| 特征 | 正交补 | Hodge 星 |
| :--- | :--- | :--- |
| 输出 | 子空间（无方向、无大小） | **k-向量**（有大小有方向） |
| 编码 | 仅"方向" | **大小 + 方向 + 符号** |

**核心断言**：
$$\star(u \wedge v) = w$$
表示 $u \wedge v$ 与某个具体的 $(n-k)$-向量 $w$ 编码同一个几何对象。

### 4. 唯一性：方向约定

**问题**：$w$ 与 $-w$ 都正交于 $u \wedge v$ 所在平面，到底取哪个？

**约定**：对任意 k-形式 $\alpha$，
$$\alpha \wedge \star\alpha = \text{正定向单位 n-形式}\quad (\text{即}\ e_1 \wedge e_2 \wedge \cdots \wedge e_n)$$
唯一确定 $\star\alpha$ 的符号，使整个空间有统一的"右手系"。

### 5. 2D 直观：90 度旋转

- 一个向量 $u$（1-形式）的 Hodge 星 $\star u$ 是垂直于 $u$ 的向量。
- **逆时针 90 度旋转**——因为 $u \wedge \star u$ 必须填满整个 2D 平面并保持正定向。
- 例：$\star e_1 = e_2$，$\star e_2 = -e_1$。

### 6. 双重应用与循环

- $\star\star u = -u$（在合适维度/特征下）——两次 90 度旋转 = 180 度。
- 与正交补的关键差异：$(U^\perp)^\perp = U$ 但 $\star\star u = -u$ ≠ $u$，因为 Hodge 星记录了**定向**。
- 4 次循环：$u \to \star u \to -u \to -\star u \to u$。

### 7. $\mathbb{R}^3$ 中的完整 Hodge 星表

| 输入 | $\star$ 输出 |
| :--- | :--- |
| $1$ | $e_1 \wedge e_2 \wedge e_3$（标量 $\to$ 三维体积） |
| $e_1$ | $e_2 \wedge e_3$ |
| $e_2$ | $e_3 \wedge e_1$ |
| $e_3$ | $e_1 \wedge e_2$ |
| $e_1 \wedge e_2$ | $e_3$ |
| $e_2 \wedge e_3$ | $e_1$ |
| $e_3 \wedge e_1$ | $e_2$ |
| $e_1 \wedge e_2 \wedge e_3$ | $1$ |

**验证示例**：$\star e_2 = e_3 \wedge e_1$
$$e_2 \wedge \star e_2 = e_2 \wedge e_3 \wedge e_1 = e_1 \wedge e_2 \wedge e_3\quad ✓$$
（通过偶置换得到正定向单位体积元）

### 8. 关键性质

- Hodge 星将一个 **k-向量**线性映射到与其正交补对应的 **$(n-k)$-向量**。
- 通过线性性，**只需定义在基 k-形式上**——其余通过线性扩展即可。

## 九、k-形式的坐标表示

### 1. 几何独立于坐标

- **向量不是数字列表**：向量本质是几何对象，可被坐标表示，但**几何身份**与所选坐标轴无关。
- 同一向量在不同坐标系下数值不同，但仍是同一向量。
- **目标**：建立标准化、可数值化的描述方式。

### 2. 基底（Basis）

- **线性无关**：集合中没有任何元素能被其它元素的线性组合表示。
- **基底**：线性无关 + 张成全空间。
- **唯一表达**：
  $$\mathbf{v} = v^1 e_1 + v^2 e_2 + \cdots + v^n e_n$$
  系数 $(v^1, \dots, v^n)$ 是 $\mathbf{v}$ 在基底 $\{e_i\}$ 下的**坐标**。
- **维度**：基向量个数 $n$。

### 3. 几何直觉

1. **选择坐标轴**：在平面中选两个独立的轴 $e_1, e_2$。
2. **测量分量**：测 $\mathbf{v}$ 沿 $e_1$ 走多远 → $v^1$；沿 $e_2$ 走多远 → $v^2$。
3. **重构向量**：$\mathbf{v} = v^1 e_1 + v^2 e_2$。

> **基底就是测量框架，坐标就是测得的读数**。

### 4. 推广到 k-向量

- $n$ 维空间中 k-向量空间的维度为 $\binom{n}{k}$。
- 任意 k-向量都可写为 $\binom{n}{k}$ 个基 k-向量的线性组合（按递增索引排列）。
- 这构成了对 k-形式做数值计算的基础。

## 十、整体结构回顾

### 1. 外代数 vs 线性代数

| 特征 | 线性代数 | 外代数 |
| :--- | :--- | :--- |
| 处理对象 | 向量（1D 有向线段） | k-向量（多维有向体积） |
| 长度（大小） | 重要 | 重要 |
| **顺序（方向）** | 不重要 | **重要**（反对称） |
| 运算 | $+,\ \cdot,$ 内积 | $+,\ \cdot,\ \wedge,\ \star$ |

### 2. 两大核心运算

| 运算 | 类比 | 输出 |
| :--- | :--- | :--- |
| **楔积 $\wedge$** | 张成空间 | 有向、有大小的 k-向量 |
| **Hodge 星 $\star$** | 正交补 | 互补维度 $(n-k)$-向量；2D 中等同于 90° 旋转 |

### 3. 分层结构

| 层 | 对象 | 几何 |
| :---: | :--- | :--- |
| 0 | 标量 | 体积缩放因子 |
| 1 | 1-向量 | 有向线段 |
| 2 | 2-向量 | 有向面积 |
| 3 | 3-向量 | 有向体积 |
| ... | ... | ... |
| $n$ | n-向量 | 整个空间的有向体积元 |

## 十一、关键术语速查

| 术语 | 含义 |
| :--- | :--- |
| 向量空间 | 加法、数乘封闭，满足八条公理的集合 |
| 内积 | 双线性、对称、正定的二元运算；定义"长度"和"夹角" |
| 张成 $\mathrm{span}$ | 向量集合的所有线性组合（生成子空间） |
| 楔积 $u \wedge v$ | 双线性、反对称、自零、结合的运算；产生有向 k 维体积 |
| k-向量 | 有向 k 维体积元，构成 $\bigwedge^k V$，维度 $\binom{n}{k}$ |
| 0-向量 | 标量 |
| 反对称性 | $u \wedge v = -v \wedge u$ |
| 自楔零律 | $u \wedge u = 0$ |
| 帕斯卡三角 | k-向量空间维度的对称序列 |
| 正交补 $U^\perp$ | 与 $U$ 正交的所有向量；依赖内积 |
| Hodge 星 $\star$ | k-向量 $\to (n-k)$-向量；保留大小和定向 |
| 方向约定 | $\alpha \wedge \star\alpha$ = 正定向单位 n-形式 |
| 基 k-向量 | $e_{i_1} \wedge \cdots \wedge e_{i_k}$（$i_1 < \cdots < i_k$） |
| 基底 / 坐标 | 唯一线性组合表示 → 坐标系下的"读数" |
| k-形式 | 用于测量 k-向量的工具（下一讲主题） |
